## §3 10 key concepts extracted (P-01..P-10)

| ID | Concept | Implication |
|---|---|---|
| **P-01** | **Preparation = first stage of work** (не pre-work, а часть work) | Работа/результат начинается там, где начинается подготовка |
| **P-02** | **Метод-метода = понимание что перед задачей есть подготовка** | Первичная проверка мета-метода: «нужна подготовка или сразу действие?» |
| **P-03** | **Preparation calibration** — сколько / насколько детально / когда / зачем / куда | Качество подготовки = match с важностью задачи |
| **P-04** | **Speed-vs-depth judgment** — «надкибись и не важно» vs «остановиться разобраться» | Mastery-навык: триаж задач по требуемой подготовке |
| **P-05** | **Sequence: prep → action** | Всегда: понять что есть prep → выбрать метод prep → пройти prep → выбрать метод action → выполнить |
| **P-06** | **Trick of unique-method creation** — сразу делать = берём из existing; делаем prep → появляется картина → создаём unique custom метод | Preparation = creative ground; уникальные методы рождаются из подготовки |
| **P-07** | **Calibration questions** — сколько / когда / зачем / куда / насколько | Операционный чеклист этапа подготовки |
| **P-08** | **Mastery visible at transitions** — prep→action / study→action / action→feedback→improvement | Мастерство = качество переходов, не просто «делание» |
| **P-09** | **Loop: action → feedback → improvement → next prep** | Непрерывное улучшение встроено в мета-метод |
| **P-10** | **Skill/ability lives at the transition layer** — навык и способность делать = mastery at transitions | Мастерство измеримо = качество петель prep→action→feedback |
