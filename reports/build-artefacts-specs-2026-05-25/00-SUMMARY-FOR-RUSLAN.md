# 📋 Build Artefacts Specs — кратко (≤5 мин)

**Что это.** Каждому из 12 Build-артефактов (видео A/B/C, Notion Personal/Team OS, Charter,
лендинг+FAQ, discovery-звонок, юр+счёт, supporting+brand) теперь есть **спека — чертёж**: цель,
аудитория, структура, hook, CTA, откуда substrate, R12-чек, критерии готовности, анти-паттерны,
варианты. 12 артефактов = 10 spec-файлов (юр+счёт и supporting+brand объединены). Это **не сами
артефакты** — сам текст видео / поля Notion / Charter пишешь ты по спеке.

**Зачем.** Закрывает кашу «что говорить в видео? какие базы? что в Charter?». Было «надо что-то
делать» → стало «делаю X по спеке Y». И каждая спека написана так, чтобы умный партнёр потом
тоже мог по ней работать.

**На кого всё заточено.** Один тип человека — **умный партнёр** (T1 методолог + R12-мост + smart
T3 тестер): открыл, за 2-5 мин поймал substance, решил тестировать или нет — без ощущения
обработки. Масса (платящие когорты, блогеры) — secondary, зовём после.

**Что блокер.** Видео A — единственный артефакт без зависимостей; от него зависит почти всё.
Пока его нет — Wave 1 сжигает контакты, лендинг пустой, звонок не на что опереть.

**Где растёт опасность.** R12-риск не равномерен. **Высшая поверхность: Charter + Видео C + Team
OS** — там соблазн mass-movement / lock-in / fudge долей максимален, и там автоматически
включаются R12-эксперты. Charter — самый critical (там «не доим / не запираем» либо имеет зубы:
Mondragón 5:1 с числовым примером, fork-and-leave 30-day первым пунктом, 4 детектора жадности).

**Критический путь к выходу из Build.** Видео A → (Personal OS 5 баз ‖ Charter текст) → Charter
R12-ревью (Прапион-tier) → Discovery отрепетирован ≥5 раз → ≥1 T1 confirmed + ≥3 T3 активны + юр
начато → Build exit ≥80%. Видео B/C, Team OS demo, лендинг, supporting — желательны, но не строго
блокируют выход.

**Что решаешь ты (13 решений в §16 main + по 5 вариантов на каждый артефакт в spec-файлах):**
приоритет critical-набора · когда Notion · порядок видео · timing Team OS · смарт-контракты в
видео C · формат подписи Charter · ratio 5:1 · юр-форма · какие 5 баз · лендинг язык/формат ·
discovery длина · темп Build · 13-й артефакт (новый обязательный НЕ всплыл).

**Что дальше.** Читаешь Main (20-30 мин) → picks решения → Week 1: видео A запись, Notion 5 баз
(под Дмитрий trial), Steuerberater email, CRM-разметка. **Не запускается авто** — каждый
launch'аешь сам.
