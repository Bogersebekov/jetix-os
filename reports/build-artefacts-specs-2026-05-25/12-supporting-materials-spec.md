# 📚📊 Phase 11 — Спека Supporting materials (course skeleton + Telegram + sales-min + brand-min)

> **Что это.** Чертёж 4 вспомогательных под-артефактов, которые строят доверие поверх видео +
> лендинга. **НЕ финальный контент** — а minimum credible baseline: «здесь serious work, эти
> ребята настоящие, есть substance». Если в спеке появится реальный контент курса/постов — это
> уже артефакт.
>
> **R12 AUTO-FIRE nlp** на Telegram positioning + sales-minimum — phrasing проверен; one-pager
> НЕ marketing fluff; pitch deck без vanity metrics. [src: prompt Phase 11 R12 — F2-F3, R-med]

---

## 1. Назначение (one-liner)

Минимальный credible baseline из 4 под-артефактов (скелет курса + Telegram-позиционирование +
sales-минимум + brand-минимум), который при касании любым из них даёт умному партнёру
consistent picture «эти ребята настоящие, есть substance» — без претензии на финальность.

## 2. Цель (поведенческая)

Не «полноценные финальные артефакты», а **smart T3 после consuming любого из 4 получает
consistent picture, не противоречащий видео A/B/C / лендингу / Charter**. Целевое поведение —
доверие («тут думали, не на коленке»), а не конверсия. [src: prompt Phase 11 acceptance —
F2, R-high]

## 3. Primary audience (умный партнёр)

**T1 + smart T3** (trust-building secondary surface beyond видео + лендинг).
- **Bloom-ступень:** 2-4 (Понимает → Анализирует) — углубление доверия после первого касания.
- **Знает:** как отличить «serious» от «инфоцыган»; ценит skeleton без воды.
- **НЕ знает:** наш полный курс (его ещё нет — это Run); нашу визуальную систему.
- **Хочет:** убедиться, что за видео есть глубина; visual consistency как сигнал серьёзности.
- **Отпугнёт:** premature brand-investment (полный logo redesign), fake metrics, pitch deck >10
  slides, Telegram cadence-pressure до готовности substance.
- *Примеры роли (IP-1):* методолог Maxim/Oleg-tier (смотрит skeleton курса); тестер Дмитрий
  (читает Telegram).

## 4. Secondary audience (mention only)

Visual-партнёры (для brand-minimum). Будущая когорта (курс skeleton прорастёт в Run, когда T1
со-создаёт). Массовый подписчик Telegram — зацепит позже, но Build = baseline.

## 5. Ключевые сообщения (что supporting доносит)

1. **За видео есть глубина** — skeleton курса показывает структуру (Bloom 1-7 + 7 принципов).
2. **Эти ребята настоящие** — visual consistency + serious one-pager.
3. **Substance, не маркетинг** — Telegram-посты про суть, не про продажи.
4. **Это начало, не финал** — честно: курс end-to-end появится в Run с партнёром.

## 6. Структура — 4 под-артефакта (спека каждого)

### 6.1 Course skeleton (table of contents, НЕ контент)
- sections + Bloom 1-7 mapping + пересечение с 7 принципами;
- **только skeleton** — реальный контент появится в Run, когда T1-партнёр со-создаёт;
- глубина: sections only vs sections + lesson titles (R1 §15).
[src: Consolidated HL §4 ступени + Outreach Content §4 Bloom + §3 принципы — F2-F3, R-med]

### 6.2 Telegram positioning
- channel description (что это, для кого, чего НЕ ждать);
- первые 5-7 постов outline (что публикуем, чтобы умный новый подписчик понял substance);
- **НЕ content calendar на месяцы** — только стартовый каркас;
- язык: только русский vs + английский (R1 §15).
[src: Outreach Content §7 narrative arc Stage 1-2 + P0-P2 artifacts — F2-F3, R-med]

### 6.3 Sales-minimum (one-pager + pitch deck-light)
- **one-pager** — 1 страница «вот что мы делаем» в ответ на «расскажи» (execution-plan: почти
  готов, доработка);
- **pitch deck-light** — 5-7 slides для face-to-face;
- НЕ marketing fluff, без vanity metrics.
[src: Execution Plan §3 «1-pager почти готов» + PARTNER-OFFERING (контент one-pager) — F2-F3, R-med]

### 6.4 Brand-minimum
- visual identity: logo если есть / typography / 2-3 colors / минимум template для consistency;
- НЕ полный redesign — минимум для узнаваемости;
- DIY Canva vs professional designer (R1 §15).
[src: PARTNER-OFFERING HL (стиль) + Jetix vision — F2-F3, R-med]

## 7. Hook начала

Для каждого под-артефакта hook = **«serious без пафоса»**. Course skeleton открывается структурой
(не «лучший курс в мире»); Telegram — постом про суть (не «подпишись, не пропусти»); one-pager —
ясной 1-строкой; brand — чистой consistency. Hook supporting = сигнал глубины, не громкости.

## 8. CTA конца

Trust-building, low-friction. Course skeleton → «следи, курс растёт в Run». Telegram → CTA-02
(subscribe). One-pager → «напиши, если откликнулось». **Запрещено:** «купи курс сейчас» (курса
ещё нет), cadence-pressure в Telegram. [src: Outreach Content §5.2 + §5.4 — F2-F3, R-med]

## 9. Формат / длина

- **6.1** course skeleton: ToC (sections / + lesson titles).
- **6.2** Telegram: description + 5-7 постов outline.
- **6.3** one-pager: 1 страница; pitch deck: **≤10 slides** (потолок).
- **6.4** brand: logo + typography + 2-3 colors + template.

## 10. Зависимости

- **Pre-reqs:** видео A/B/C (для consistency); лендинг (one-pager = его сжатая версия); Charter
  (для consistency сообщений).
- **Блокирует:** ничего critical — supporting = nice-to-have для Build exit (R1-A §Phase 1).
- **Параллельно:** всё остальное (supporting не блокер).
- **Анти-паттерн зависимости:** НЕ делать полный brand redesign до Build exit. [src: prompt
  Phase 11 анти-паттерн — F2, R-high]

## 13. Acceptance criteria

- ✅ smart T3 после consuming **любого** из 4 под-артефактов получает **consistent picture**.
- ✅ Не противоречит видео A/B/C / лендингу / Charter (consistency-чек).
- ✅ Course skeleton = только структура (не контент); честно «растёт в Run».
- ✅ Telegram = 5-7 постов outline (не calendar на месяцы).
- ✅ Pitch deck ≤10 slides; без vanity metrics.
- ✅ Brand-minimum = минимум для consistency (не полный redesign).
- ✅ nlp-ревью: нет fluff/vanity/cadence-pressure.

## 14. Анти-паттерны

- ❌ **Premature brand-investment** — полный logo redesign / website rebuild до Build exit.
- ❌ **Fake metrics** — «1000 учеников», которых нет.
- ❌ **Pitch deck >10 slides** — раздутая презентация.
- ❌ **Telegram cadence-pressure** — «постим каждый день» до готовности substance.
- ❌ **Course skeleton как готовый курс** — обещание того, чего нет.
- ❌ **One-pager как marketing fluff** — прилагательные вместо сути.

## 15. Варианты / R1 decisions (surface)

- **R1-S1 Course skeleton глубина** — sections only vs sections + lesson titles.
- **R1-S2 Telegram язык** — только русский vs + английский.
- **R1-S3 Pitch deck design** — DIY Canva (быстро, дёшево) vs professional designer (дороже,
  солиднее). *Факт:* premature investment = анти-паттерн; DIY достаточно для Build.
- **R1-S4** Brand-minimum scope — logo + 2 colors (минимум) vs полная mini-система?
- **R1-S5** Telegram запускать в Build или ждать Run? *Факт:* cadence-pressure до готовности
  substance = анти-паттерн; вариант — канал-заглушка с 5-7 постами без обещания регулярности.
