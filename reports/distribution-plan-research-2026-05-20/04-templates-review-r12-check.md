## §1 R12 paired-frame discipline — canonical definition

Per Pillar C Tier 2 rule 12 (additive 2026-05-12) + R12 Charter ack 2026-05-12 + H7 People-NS LOCKED 2026-05-12 commit `93b796d`:

> **No extraction beyond agreed share.** AI / substrate cannot extract value from members beyond agreed share; members can fork-and-leave without penalty.

**Operationalisation в outreach templates:**

1. **Explicit offer** — concrete value Jetix даёт target (substrate / artifact / intro / opportunity / methodology / cohort)
2. **Explicit ask** — concrete request (call / feedback / commitment / introduction / time-allocation)
3. **Proportionality** — ask ≤ offer in scope; not extraction-only
4. **Reversibility** — target can decline / fork-and-leave без penalty; no «sunk cost» pressure tactics
5. **Authentic substrate** — offer = real-deliverable backed by Jetix substrate, не vapor

**Failure modes (template flag):**
- Missing offer line → REVISE (extraction-only ask)
- Missing ask line → REVISE (vague; target doesn't know what's wanted)
- Ask > offer scope → REVISE (e.g., asking 6-month commitment с offer of 1 article)
- Pressure tactics («last chance» / «running out of time» if false) → REVISE
- Vapor offer (deck doesn't exist; substrate not built) → REVISE OR build substrate FIRST
