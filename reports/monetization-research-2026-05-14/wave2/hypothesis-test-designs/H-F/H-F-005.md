## H-F-005 — Shared infrastructure (KB / tooling / templates)

**Hypothesis statement:** Realm-wide shared infrastructure: KB / tooling / templates / methodology library (Foundation Part 3 LOCKED). Network effects: more Clans → richer shared KB → more value per Clan.
**Why test:** Network effect on shared KB; reduces per-Clan duplication.
**Minimum viable test:** Foundation Part 3 KB deployed; 1-2 Clans contribute + consume; track KB growth + cross-Clan citation rate.
**Success metric:** KB grows ≥20% Year 1; cross-Clan citations ≥30% of all citations; member NPS on KB positive.
**Failure metric:** KB stagnates; siloed Clan KBs; access tier complexity rejected.
**Confounders:** Foundation Part 3 quality; access tier UX.
**Precedent test:** Mondragón shared services (60-year sustained); Wikipedia commons (20+ years, 60M+ articles); Linux kernel shared codebase; Stack Overflow.
**Phase fit:** Phase 1+ (Foundation Part 3 LOCKED).
**Required L1 fit:** Any.
**R12 alignment:** ✓ KB contributions credited per H-C-012; member-portable on fork.
**Cross-pollination:** Combines with H-C-012 (contribution rewards); combines with H-M-012 (white-label SaaS for KB); amplifies H-A-013 (Realm goods).
**Bayesian prior:** **High** — Mondragón / Wikipedia / Linux proven commons model.
