## H-C-018 — Life-improvement metrics — TRM 6 resources tracked at member level

**Hypothesis statement:** Member's own TRM (Capital / Time / Audience / Knowledge / Compute / Team / +Body/Charisma if accepted) tracked; growth = life improvement; visible to member (personal dashboard).
**Why test:** L4 intrinsic incentive; aligns Realm engagement with member's own life goals.
**Minimum viable test:** Deploy personal TRM dashboard for 30 members; 1-Season baseline + tracking.
**Success metric:** ≥80% members engage with dashboard monthly; ≥50% report measurable improvement in ≥2 resources; NPS positive.
**Failure metric:** Dashboard ignored; tracking burden too high; member sees no improvement.
**Confounders:** Self-report variance; resource definition ambiguity.
**Precedent test:** Strava annual stats (90M users); LinkedIn skill assessments; Stoic philosophy "examined life" (millennia); CrossFit benchmark workouts (Fran / Grace / Helen — track personal progress); RescueTime / Beeminder personal tracking.
**Phase fit:** Phase 1+.
**Required L1 fit:** Any.
**R12 retest:** Verify dashboard data private to member by default; sharable per opt-in; data portable; no aggregation without consent (per H-M-011 retest).
**Cross-pollination:** Combines with H-C-015 (identity); combines with H-A-003 (TRM operator earns from own resources); amplifies L4.
**Bayesian prior:** **Medium-High** — Strava / habit tracker proven для personal motivation.
