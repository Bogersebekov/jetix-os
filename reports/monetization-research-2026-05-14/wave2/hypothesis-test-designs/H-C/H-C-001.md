## H-C-001 — Realm rank visibility — every action affects visible rank

**Hypothesis statement:** Each member action (Quest / mentor / Clan / defection) → E1 Persona stats → visible to all Realm. Layer 2 (M-B) cooperation amplifier.
**Why test:** Visible rank drives cooperation vs defection by making free-riding visible; rank algorithm transparency vs gaming.
**Minimum viable test:** Deploy rank visibility (basic schema); track defection events over 1 Season; cohort 20-50 members.
**Success metric:** Defection events <5%; member NPS on rank visibility ≥+5; rank gaming complaints <2.
**Failure metric:** Rank gaming becomes optimization target; defection rate increases (rank-as-status backlash); rank algorithm complaints.
**Confounders:** Cold-start cohort selection bias; algorithm choice dominates outcomes.
**Precedent test:** GitHub contributions graph (visible since 2008; sustained); Stack Overflow reputation; LeetCode rank; LoL MMR; LinkedIn All-Star.
**Phase fit:** Phase 1+ (foundational).
**Required L1 fit:** Any.
**R12 retest:** Verify member can opt-out of public visibility (private rank still tracked for member personal dashboard); rank decay rules transparent (per H-C-013); appeal mechanism for incorrect rank.
**Cross-pollination:** Combines with H-C-002 (Season retro — rank update boundary); combines with H-C-004 (Clan rank — aggregation); combines with H-C-013 (rank decay); amplifies H-A-016 (external rate uplift).
**Bayesian prior:** **High** — GitHub / Stack Overflow proven.
