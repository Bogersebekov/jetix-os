# Srinivasan — *The Network State* (2022) — Deep Extract

## §1 Author + context

Balaji Srinivasan — former CTO Coinbase, former a16z general partner, Stanford PhD. *The Network State* (2022) — self-published manifesto on decentralized state formation via online community → physical archipelago → diplomatic recognition.

## §2 Seven-step framework (canonical)

1. **Founder + One Commandment** — single founder с clear, narrow thesis
2. **Online community** — recruit first 1K members на shared values
3. **Network union** — collective action capability (e.g., group purchasing)
4. **In-person events** — periodic gatherings → real bonds
5. **Crowdfunded archipelago** — physical sites across world, network-connected
6. **Census + economy** — internal accounting; recognized internal economy
7. **Diplomatic recognition** — treaty с existing sovereign states

## §5 Five Jetix-relevant concepts beyond Steps

### Concept 1 — Cloud-first identity, land-second
Balaji's claim: digital identity precedes physical territory. Estonia e-residency precedent (100K residents from 170+ countries) validates. **Jetix Phase 1-3 = pure digital; Phase 4+ if physical layer pursued, treaty-protected only.**

### Concept 2 — Founder Commandment
Single, narrow thesis distinguishes Network State from generic community. Examples: «no carbs» (keto state), «1 hour exercise daily» (fitness state), «universal basic compute». Jetix candidates per HQ-WO-004 (A..F).

### Concept 3 — Network State vs Nation State
NS is opt-in, distributed, identity-portable; Nation State is coercive, territorial, identity-fixed. **Jetix favors NS over NS-style coercion** (R12 fork-and-leave; constitutional opt-out).

### Concept 4 — Census as Legitimacy
Internal accounting + economy size = legitimacy precursor. Realm member count + Realm GDP (Castronova framing) is census-equivalent. Phase 3-4 publishing census data → external recognition path.

### Concept 5 — Recognition Cascade
First sovereign state recognition triggers cascade. Balaji's example: small nation A recognizes Jetix → larger nation B follows. Phase 5 strategy: identify «friendly first recognizer» (Estonia possible candidate per existing e-residency adjacency).
