# Phase 6 — Bucket Cross-Mapping (ALL 80 books → 4 research buckets)

## §6.1 Method

Per book: assign **primary bucket** + **secondary bucket(s)** + **relevance score per bucket** (0-3 scale).

**Buckets:**
- **P** = propaganda-recruitment (persuasion / influence / group psychology / cult mechanics / marketing-sales)
- **S** = sota (philosophy-of-science / ML-AI state / SE state)
- **M** = methodology (research methods / problem-solving / systems thinking / facilitation / SE method)
- **N** = nlp (Neuro-Linguistic Programming corpus + critiques)

**Scale:** 3 = core/canonical for bucket • 2 = strong relevance • 1 = adjacent/touches • 0 = unrelated.

**Critical:** Single book often maps to **multiple** buckets. Pre-existing folder placement = strong signal for primary bucket but NOT exclusive.

## §6.4 Notable cross-bucket findings

1. **Polanyi books** (Personal Knowledge + Tacit Dimension) span **M + S** (both methodology of research + philosophy of science). Should be referenced from BOTH research prompts.
2. **Kahneman** spans **P + S + M** strongly — System 1/2 framework is foundational across all three.
3. **NLP corpus + Bolton + Robbins** = direct **N + P** dual bucket (NLP techniques ARE persuasion techniques).
4. **Schelling — Strategy of Conflict** (gamification folder) actually belongs to **S** (game theory canon) + **M** (strategic-interaction methodology) + adjacent **P** (recruitment via commitment).
5. **Levenchuk corpus** (5 books) — all M=3 (core methodology); 04-intellekt-stek = also S=3 (intelligence-stack as state-of-art lens).
6. **17.pdf = Shchedrovitsky** = Russian methodology canon (MMK lineage Levenchuk inherits from). Should be primary M reference for Russian-language research methodology questions.
7. **cialdini-influence-ru-1984** = Russian translation of cialdini-influence-1984 (English in propaganda-recruitment/). **DUPLICATE content — choose one for processing in downstream prompts**.
8. **Gamification corpus** (13 books) — mostly P=2-3 + M=2 dual (game design IS persuasion engineering methodology). Should be added as **5th adjacent bucket** OR distributed P+M.
9. **arxiv MAS papers (3)** — pure S; useful for sota prompt on multi-agent scaling.
10. **Anthropic safety (Askell HHH, Bai CAI)** — S=3 + M=2 + P=2 (constitutional AI as methodology to encode persuasion-resistant values).
