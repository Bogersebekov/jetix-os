# 🎨 Каталог схем — Execution Plan Fixation

Шесть mermaid-схем к плану исполнения. Все встроены в `06-mermaid-schemes.md` и в главный
документ `decisions/strategic/EXECUTION-PLAN-FIXATION-2026-05-24.md`. Мягкие цвета, без
жаргона, читаются без расширений (8-20 узлов).

| # | Название | Что показывает | Тип | Узлов |
|---|---|---|---|---|
| **EX-1** | Вся картина исполнения | baseline → фиксация → Руслан сам → 2 направления → маховик | flowchart LR | ~14 |
| **EX-2** | Что Руслан делает сам | 3 видео + 5 шаблонов + остальное + чего НЕ делает | mind-map LR | ~18 |
| **EX-3** | Два направления | Дмитрий-сторона × партнёр-сторона + перекрёстная обратная связь + слияние | two-track TB | ~12 |
| **EX-4** | 4 типа партнёров | T1/T2/T3/T4 × что хотим × что даём × R12-риск × результат | lattice LR | ~17 |
| **EX-5** | Дерево решений | порядок видео A→B→C → Wave 1 → ветки результата | decision tree TD | ~13 |
| **EX-6** | Таймлайн 3 недель | по дням и секциям (видео / платформа / партнёры / свои силы) | gantt | ~16 |

## Как читать

- **EX-1** — начни отсюда, общая картина.
- **EX-2** — детализация «что руками».
- **EX-3** — почему два направления параллельно.
- **EX-4** — главная схема про партнёров (4 разных сделки).
- **EX-5** — что делать когда (логика «готово/нет»).
- **EX-6** — календарь на 3 недели.

## Стиль (mandate)

- Фон `#fafafa`, тёмный текст, мягкие заливки (#ffd6d6 / #d6f0d6 / #cce6ff / #fff4e6 / #ffe0a0 / #eeeeee).
- Без жаргона в подписях — человеческий язык.
- Пунктир (stroke-dasharray) — для «уйти / НЕ делаем / опционально».
- Продолжает стиль HL-1..HL-5 из consolidated-hl.
