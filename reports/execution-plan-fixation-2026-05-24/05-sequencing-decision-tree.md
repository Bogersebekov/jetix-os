# 📅 Порядок: что делаем когда

> **Главная мысль части:** план без порядка — это список желаний. Здесь — конкретная
> последовательность: что **обязательно на этой неделе**, что на следующей, и что
> запускается только **по триггеру** (когда случилось определённое событие).
>
> Один принцип над всем: **всё держится на видео A.** Пока его нет — остальное буксует.

## §3 Что по триггеру (запускается, когда случилось событие)

Это не расписание — это «если X, то Y». Не делаем заранее.

| Триггер (если случилось) | Действие (то делаем) |
|---|---|
| **1+ партнёр T1 откликнулся с интересом** | Разбор-звонок → R12 8-вопросов → предложение методологического пилота |

## §4 🛑 Где притормозить (anti-rush — защита от спешки)

Спешка тут вреднее медлительности. Намеренные тормоза:

- **НЕ рассылать Wave 1 без готовых видео.** Outreach без материала = низкое качество,
  сжигаешь контакты.
- **НЕ застревать в перфекционизме на видео C.** Потолок — 14 дней. Лучше выпустить
  «достаточно хорошее», чем доводить до блеска месяц.
- **НЕ обещать партнёрам золотые горы по ROI.** Это нарушение «не доим» — честные цифры,
  не завышенные.
- **НЕ пропускать R12 8-вопросов** перед каждым касанием партнёра. Ни разу.
- **НЕ откладывать сессию с Дмитрием в ожидании «идеального шаблона».** Лучше итерировать
  на черновом, чем ждать совершенства.

## §5 🌳 Дерево решений (текстом; mermaid — в части 6, схема EX-5)

```
СТАРТ: Видео A готово?
  ├─ НЕТ → Запись видео A (наивысший приоритет, всё ждёт его)
  └─ ДА  → Видео B готово?
            ├─ НЕТ → Параллельно: запись видео B + Notion-шаблон Дмитрия
            └─ ДА  → Видео C готово?
                      ├─ НЕТ → Параллельно: видео C + подготовка Wave 1 + trial Дмитрия
                      └─ ДА  → Wave 1 отправлен + обратная связь Дмитрия собрана?
                                ├─ Wave 1: 0 ответов → пересмотреть качество outreach, поправить видео
                                ├─ Wave 1: 1+ ответ  → разбор-звонки + итерация платформы (направление A)
                                └─ Дмитрий дал инсайты → правим шаблон + заводим Севу
```

**Как читать дерево:** идёшь сверху вниз. На каждой развилке — простой вопрос «готово/нет».
Не перепрыгивай: пока видео A нет, нет смысла думать про Wave 1.

## §6 ⏱️ Сколько всего времени (оценка по неделям)

| Неделя | Что | Время |
|---|---|---|
| **Неделя 1** | Видео A+B + Notion + Дмитрий + подготовка Wave 1 | ~25-35 ч |
| **Неделя 2** | Видео C + отправка outreach + первые звонки + правка Дмитрия + Сева | ~20-30 ч |
| **Неделя 3** | Обработка ответов + итерация + первый методологический пилот | ~15-25 ч |
| **Итого** | | **~60-90 ч за 3 недели** |

При ~3-4 часах глубокой работы в день — реалистично укладывается в 3 недели.

## §7 📌 Зависимости одной картинкой (что от чего)

Чтобы было видно, что нельзя делать вперёд чего:

- **Видео A** → ничего не зависит, делается первым.
- **Видео B** ← нужно видео A (последовательность смысла).
- **Видео C** ← нужны A+B.
- **Outreach партнёрам (Wave 1 send)** ← нужны все 3 видео + список целей.
- **Сессия с Дмитрием** ← нужен Notion-шаблон + trial 5-7 дней.
- **Онбординг Севы** ← нужна правка шаблона после Дмитрия.
- **FAQ** ← нужны первые реальные разговоры.
- **Левенчук/Цэрэн/Прапион** ← после реакции Maxim/Oleg.

**Узкое горлышко:** видео A. Всё остальное выстроено за ним. Поэтому пункт 1 недели 1 —
действительно наивысший приоритет.
