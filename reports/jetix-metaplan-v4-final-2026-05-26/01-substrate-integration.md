   ⚠️ **R12 STRICT highest surface** — gamification = max manipulation risk (addictive loops / variable

   progression only / opt-out always / NO dark patterns.

2. **Хакатоны / Соревнования / Экспедиции (Direction 16):** events = основной заработок + community engine.
   «Люди сами будут создавать хакатоны, искать исследования, решать задачи, ездить на экскурсии — всё через
   Jetix». Очень обширный слой.

3. **Кланы (EXPANSION в Network #14):** «кланы = основная часть. Кланы будут между собой и сотрудничать и
   соперничать — дух соревнования, но уважение друг к другу. Кланы позволят разные направления выбирать
   пробовать. Внутри клана можно делать почти всё что хочешь, придерживаясь только базовых ценностей».
   → Jetix = «качалка / склад» — НЕ диктует методы/темы внутри клана; даёт infrastructure + Charter floor

## §3 Кланы lifecycle — что разворачивает V4 (overview, full в Phase 15)

Кланы упоминались в V3 Foundation (Network mesh cells = прото-кланы), но V4 раскрывает их **operationally
как living entities** — sub-section §K в Network direction:

- **Formation** — founding members / Charter signing / initial Mondragón allocation / first project.
- **Governance** — internal (Steward + consensus/voting/RACI per клан Charter).
- **Внутри-клана свобода** — methods/topics/management style = inner-clan; ТОЛЬКО R12 + ценностной floor
  enforced platform-wide («тяга к жизни / уважение к соревнующимся»).
- **Соревнование между кланами** — healthy (соревнования/хакатоны inter-clan/mastery tournaments); НЕ
  destructive (R12 STRICT: no member poaching / no resource sabotage / no anti-clan extraction).
- **Сотрудничество** — collaborations on big projects / cross-clan expeditions / shared research / talent exchange.
- **Fork-and-spawn** — клан split / spawn новый (Mondragón: cooperative spawning cooperatives).
- **Dissolution** — graceful unwind (asset distribution per Charter / member migration).
- **Inter-clan governance** — Stewards across clans (peer-check ротация) + Foundation-level dispute resolution.
- **Platform role:** Jetix = качалка/склад (infrastructure + Charter floor + events) — НЕ контролёр.
