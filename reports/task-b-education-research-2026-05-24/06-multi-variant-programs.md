# Phase 6 — Multi-variant educational program design

> **Per Task B prompt §2.E.** Multi-variant catalog: 2h overview → multi-year sustained. Each variant designed using **unified ID model from Phase 3 §6.1**: backward-design Stage 1 (outcomes) → Stage 2 (evidence) → Stage 3 (learning); diagnosed Dirksen gap; Merrill 5 principles; Gagné 9 events; Bloom level targets; ZPD scaffolding; Hattie mindframes; Make-It-Stick spacing.
>
> **Per Ruslan voice 24.05** — «через обучающую линзу всё прогнать + multi-variant programs».

---

## §0 Shared design constants (applies all variants)

### §0.1 Audience archetype map (Welcome-frame stages O-144)

| Stage | Profile | Typical entry-time available | Needs |
|---|---|---|---|
| **Stage 1: Stranger** | Heard of Jetix once; curious | 15 min - 2h | Welcome + thesis frame |
| **Stage 2: Browser** | Read 1-2 posts; understands premise | 2-8h | Method V2 §A-§F surface |
| **Stage 3: Engaged** | Active reader; wants try Method | 8-40h | Full Method V2 + first practice |
| **Stage 4: Practitioner** | Practicing for weeks | 40-100h | Cohort + first hypothesis test |
| **Stage 5: Cohort member** | Active cohort participation | 100-500h | Workshop residency + co-design |
| **Stage 6: Reformer-aspirant** | Co-creating Jetix substrate | 500-2000h | Multi-year mastery + clan-Charter |
| **Stage 7: Advocate** | Running own clan / forking | 2000+ h | Multi-year co-creation |

### §0.2 Three orthogonal axes per variant

1. **Time budget** — how long learner commits (real-world constraint)
2. **Audience stage** — current Welcome-frame stage (1-7)
3. **Outcome target** — Bloom level / McTighe facet / МИМ qualification rung

### §0.3 Shared enduring understandings (UbD Stage 1 per Phase 3 §1.3)

Per Phase 7 Jetix Lens — these are constant across ALL variants:

1. **Жить чтобы жить + не умереть + развиваться** (O-138 foundational values) → not extraction-game
2. **Method-of-method composition** (O-107, O-121) → методология первична, не функции
3. **Hypothesis-driven learning + life** (O-179) → не накапливать «у меня есть», а тестировать «if X then Y»
4. **AI-augmented intellect stratification** (O-180 + O-182) → adequate intellect chooses developing methods
5. **R12 anti-extraction discipline** → fork-and-leave at every level; no entrapment

What varies: depth, evidence required, scaffold density, delivery medium.

### §0.4 Anti-pattern flags

- **Edu-tech extraction:** variant must not extract data / lock-in / sell ascension. R12 paired-frame applied per variant.
- **Empty cheerleading:** pseudo-Freirean / pseudo-Dweck without epistemological content (cross-cite Phase 1 §4.5 + Phase 2 §3.3) — every variant must have falsifiable content + verifiable performance.
- **Caplan signalling trap:** variant must produce verifiable outputs, not «certificates» as social signal.

## §1 Variant 1 — 2h overview («Welcome to Jetix»)

### §1.1 Spec

| Dimension | Value |
|---|---|
| Time | 2h |
| Audience | Stage 1 → Stage 2 (Stranger → Browser) |
| Format | Video + 1-pager + 1 reflective prompt |
| Outcome target | Bloom 2 Comprehension + Bloom 6 Evaluation (decide if Jetix interest serious) |
| Delivery | Async (no live component) |
| Cost (learner) | Free |
| Cost (Jetix to produce) | One-time recording 4-8h + 1-pager 2h |

### §1.2 Content (Phase 3 ID model)

**Stage 1 (outcomes) UbD:**
- BIG IDEA: «Education + intellect can be augmented by AI in a way that develops rather than diminishes the learner — if you choose the right methods.»
- ENDURING UNDERSTANDING: «The choice of methods is itself a method. Choose ones that develop you, not shortcuts that hollow you out.» (O-180)
- ESSENTIAL QUESTION: «Что мне сейчас изучать (и каким образом) чтобы через 5-10 лет я стал кем хочу стать?»

**Stage 2 (evidence):**
- Self-rating on 3 prompts at end (no submit; private)
- Click-through to «Browser» track Variant 2 (signal of interest)

**Stage 3 (learning experiences):**

| Min | Event (Gagné) | Content |
|---|---|---|
| 0-5 | Gain attention | Hook: «Если AI делает то, что ты собирался выучить за 5 лет, за 30 секунд за электричество — что ты выбираешь делать вместо этого?» |
| 5-10 | Inform objective | This 2h gives you 3 frames: paradigm shift / Method V2 thesis / next step |
| 10-30 | Stimulate recall + present stimulus 1 | Educational paradigm shift (O-176/177) — verbatim of Ruslan voice notes |
| 30-60 | Present stimulus 2 + guidance | Method V2 thesis (one diagram, no full content) |
| 60-80 | Present stimulus 3 + guidance | Hypothesis-driven life (O-179, O-185) |
| 80-100 | Elicit performance | Reflective prompt: «What is your current hypothesis about how YOUR life should develop?» |
| 100-110 | Feedback (general) | Possible-answer frames (not «correct» — they're learner-specific) |
| 110-120 | Enhance transfer | What to do next: 1) Variant 2 (1-day) if you want more / 2) Continue reading at own pace |

### §1.3 Pricing + R12 discipline

- **Free.** Period. No upsell email-capture before delivery; opt-in for next-variant invite after content delivered.
- R12: Free at this stage = mass-paradigm-shift O-184 + welcome-frame O-144 + anti-extraction.

### §1.4 Required artefacts

- Video (Khan-style; voiceover + simple slides; 15-30 min)
- 1-pager PDF (Russian primary; English secondary) — extracted from Method V2 + Ruslan Notes
- Notion or simple webpage hosting (env-gap mitigation per Dirksen)

## §2 Variant 2 — 1-day intensive (8h, Workshop format)

### §2.1 Spec

| Dimension | Value |
|---|---|
| Time | 8h (one day; can split 2×4h weekends) |
| Audience | Stage 2 → Stage 3 (Browser → Engaged) |
| Format | Workshop (live or synchronous online) |
| Outcome target | Bloom 3 Application — apply Method V2 §A-§F to learner's own situation |
| Delivery | Live + recording for repeat |
| Cost (learner) | €100-300 (early-cohort) → €500-800 (steady-state); free for hardship cases (R12) |
| Cost (Jetix to produce) | Workshop facilitator-day + prep 4h |

### §2.2 Content

**Stage 1 outcomes (UbD):**
- Big idea: «Method V2 is a method-of-methods you can apply to your own life-development today, not a credential to acquire over years.»
- Essential question: «Какая у тебя текущая методология выбора методов? Адекватная или unadequate?»

**Stage 2 evidence:**
- One filled-out Method V2 §A welcome frame for the learner's own context
- One articulated hypothesis (Hypothesis Architecture layer 1-3) on something the learner wants to test next
- Workshop output portfolio (1 page, peer-reviewed)

**Stage 3 learning experiences (Mazur PI + ОДИ flavor):**

| Hours | Module |
|---|---|
| 0-1 | Welcome + paradigm shift (Variant 1 condensed + Q&A) |
| 1-2 | Method V2 §A welcome frame demo + work in pairs |
| 2-3 | Method V2 §F drill exercises + Mazur PI ConcepTest cycle (2-3 ConcepTests on §F) |
| 3-4 | LUNCH + diffuse-mode walk (Oakley) |
| 4-5 | Method V2 §J meta-method composition + СМД-style schematization |
| 5-6 | Hypothesis architecture — learner writes own first hypothesis |
| 6-7 | Peer review in groups of 3 (Mazur PI extended; Hattie mf 7) |
| 7-8 | Reflection + commitment + Workshop output |

### §2.3 R12 paired-frame

- Workshop must declare upfront: «You can leave at any time. No deposit forfeit. The Method is open-source after this Workshop regardless.» (anti-fork-prevention)
- Pricing tier: hardship cases free (mass paradigm O-184)
- No upsell during; opt-in next-variant after

### §2.4 Required artefacts

- Workshop facilitator guide (3-5h prep)
- Workbook (5-10 pages; Russian)
- Recording archive for non-live access

## §3 Variant 3 — 1-week deep dive (30-40h)

### §3.1 Spec

| Dimension | Value |
|---|---|
| Time | 30-40h over 1 week (5-8h/day for 5 days OR async over 2-3 weeks) |
| Audience | Stage 3 → Stage 4 (Engaged → Practitioner) |
| Format | Hybrid: async Khan-style modules + 2 live Workshop sessions + cohort forum |
| Outcome target | Bloom 4 Analysis + Bloom 5 partial Synthesis — start crafting own meta-method |
| Delivery | Async self-paced + 2 live + asynchronous cohort interaction |
| Cost (learner) | €500-1500 |
| Cost (Jetix to produce) | 30h content production + 2 live Workshop × 4h + cohort moderation 10h |

### §3.2 Content

**Stage 1:**
- Big idea: «Method V2 is composable — you can take the 8 meta-method components (O-121) and recombine them for your specific life-development trajectory.»
- Essential question: «Какие 3 component'а O-121 у тебя сейчас самые слабые, и какой план их укрепить?»

**Stage 2 evidence:**
- Composed personal meta-method (3-5 pages)
- 3 hypotheses ready for week-2+ testing
- Peer-reviewed compositions × 2 (give + receive)

**Stage 3 sequencing (Bruner spiral + Make It Stick spacing):**

| Day | Module | Format |
|---|---|---|
| 1 | Method V2 §A-§F core + first hypothesis | Async video + 2h live |
| 2 | Method V2 §G-§L applied | Async video + ConcepTests + cohort Q&A |
| 3 | Hypothesis architecture 7-layer deep dive | Async + practice + cohort review |
| 4 | Meta-method 8-component composition O-121 | Async + 2h live Workshop (ОДИ-style) |
| 5 | Personal meta-method draft + peer review | Cohort exchange |
| (6-7 buffer) | Refinement + spaced retrieval re-pass on days 1-3 content | Async |

### §3.3 R12 + Caplan

- Verifiable output (composed meta-method + reviewed hypotheses) = substance lane vs Caplan-signal
- Cohort opt-out at any point with proportional refund (anti-fork-prevention R12)

## §4 Variant 4 — 1-month program (80-120h)

### §4.1 Spec

| Dimension | Value |
|---|---|
| Time | 80-120h over 4-5 weeks |
| Audience | Stage 4 (Practitioner) |
| Format | Hybrid live + async + cohort sprint |
| Outcome target | Bloom 5 Synthesis — design + start running personal hypothesis-test |
| Delivery | Khan modules + weekly Workshop live + cohort + 1:1 mentor 2-3 sessions |
| Cost (learner) | €2000-5000 |
| Cost (Jetix to produce) | 80h content production + 4 Workshop × 4h + 4×3h mentor = ~60-80h per cohort of 10-20 |

### §4.2 Content additions vs Variant 3

- **First-hypothesis-test sprint:** learner picks one falsifiable hypothesis and runs it during the month
- **Mid-month mentor 1:1** (2-3 sessions over month)
- **Cohort sprint output:** group challenge (peer instruction + ОДИ + clan formation seed)
- **Strategic Plan introduction** — connect personal hypothesis to wider context

### §4.3 Multi-variant pricing tier discussion

R12 paired-frame ⚠️ — pricing at €2000-5000/month requires explicit anti-extraction guarantees:
- Mondragón-ratio-equivalent: facilitator/owner wage ratio capped per FUNDAMENTAL §6.1 R12 + RUSLAN-LAYER 4 action classes
- QF revenue distribution to cohort participants who contribute substrate
- Fork-and-leave: any participant can take all materials + cohort connections to start own group
- Hardship cases free (≥10% of seats)

## §9 Cross-cutting design principles (applies all variants)

### §9.1 Pedagogy checklist per variant

Each variant must demonstrate (refute if not):

- [ ] Stage 1 Big Idea + Essential Question declared
- [ ] Stage 2 Evidence acceptable defined before content
- [ ] Bloom level target tagged
- [ ] Gagné 9 events covered (or explicit which events ceded; e.g., self-pace defers some)
- [ ] Merrill 5 principles invoked
- [ ] ZPD scaffolding sequenced
- [ ] Hattie mindframes 1-6 audit pass
- [ ] Make-It-Stick spacing in evaluations
- [ ] Sweller cognitive load budgeted (≤4 novel chunks per session)
- [ ] R12 paired-frame audit pass (no fork-prevention; no wage-ratio violation; no non-consensual distribution; no extraction-beyond-share)
- [ ] Hardship-case access (mass paradigm O-184)

### §9.2 Differentiation discipline per Tomlinson DI

Each variant should offer:
- Content depth options (e.g., optional advanced reading per module)
- Process options (solo vs cohort vs mentor-led)
- Product options (essay / model / video / code / Workshop output)
- Affect-environment options (online async / online live / in-person)

### §9.3 Anti-pattern refusal explicit

| Anti-pattern | Refusal mechanism |
|---|---|
| Empty cheerleading (pseudo-Dweck) | Specific strategy + falsifiable next step in every module |
| Banking model (pseudo-Freire) | Co-construction + dialogue protocol enforced |
| Coverage trap (UbD twin sin) | Outcome-first design enforced per checklist §9.1 |
| Style-matching waste (Willingham) | Match content to task, not «learning style» |
| Caplan signal trap | Verifiable portfolio output, not certificate |
| Cult recruitment (СМД abuse) | Charter + fork-and-leave + transparency |
