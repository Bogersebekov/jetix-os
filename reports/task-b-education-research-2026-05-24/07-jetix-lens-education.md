**Course-specific additions vs current Method V2:**
- ConcepTests per chapter (Bloom 1-4)
- Application exercises (Bloom 3-5) per learner's own context
- Cumulative project: composed personal meta-method (Bloom 5-6)
- Spaced retrieval cadence built-in (Make It Stick)
- Cohort forum + 2-4 live Workshop sessions (Mazur PI + ОДИ flavor)

#### §A.3.2 Path 2 — **Book / textbook**

Method V2 as **textbook** requires:
- Front matter: course overview, suggested study sequence, prerequisites
- Per chapter: Big Idea + Essential Question (UbD upfront)
- Embedded exercises (Bloom 1-5) at end of each chapter
- Margins / sidebars: glossary, cross-references (Левенчук style)
- Reference appendix: 47 sources organized by topic
- Index: full concept index
- 7 didactic principles compliance check per chapter (Russian didactics §4 Phase 5)

**Page estimate:** Current 43K words ≈ 200-250 textbook pages. Add 30-40% for exercises + apparatus → ~280-320 pages.

**Critical re-write needed:** TL;DR-style prose must become **scaffolded exposition** (Bruner spiral). Currently it telegraphs results; textbook requires conceptual buildup.

**Production estimate:** 200-300h editorial work for 1 person.
