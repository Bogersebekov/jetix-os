# 🗂️ Personal OS в Notion — за 3-4 минуты

**Что строим.** Облегчённую Notion-версию Jetix для ежедневной жизни. Файлы на диске =
правда, Notion = красивая витрина + голосовой ввод + подключённый Claude Code. Главное:
любой человек копирует к себе за час и настраивает под свою нишу.

## 🗂️ Базы (на выбор: 8 / 11 / 14)

| База | Зачем |
|---|---|
| 📅 Daily Log | дневник, главный голосовой ввод |
| 🚀 Projects | 4 типа + 3 чекпоинта + детект застрявших |
| ❓ Hypotheses | «что хочу узнать» — сердце познания |
| 🤝 CRM | люди + орги (наследует crm/) |
| 🔄 Reviews | неделя/месяц/квартал/год |
| 🧭 Strategic | POINT A → POINT B + решения |
| ❤️ Life Pulse | энергия/сон/отношения/деньги |
| 📚 Knowledge | источники/факты/понятия |

Плюс Command Center — хаб, всё видно с одной страницы.

## 🎤 Голосовой ввод

- Диктуешь → Claude Code раскладывает по базам как **черновики**.
- Утром/вечером смотришь → подтверждаешь вручную.
- **Никогда не перезаписывает боевые записи** — только черновик или дозапись в дневник.

## 🔄 Шаблоны анализа (8 готовых)

- День (5 мин) → Неделя (20 мин) → Месяц (45 мин) → Квартал (1.5ч) → Год (полдня).
- Плюс: ревью проекта, дизайн гипотезы, подготовка к звонку (с этическим чеком).
- Claude Code предзаполняет цифры, ты отвечаешь на «почему».

## 🍴 Fork-friendly

- **Два слоя:** универсальный фундамент (не трогают) + надстройка под нишу (под себя).
- **5 примеров:** инженер / исследователь / предприниматель / методолог / гуманитарий.
- **Этика встроена:** твои данные — твои, уйти можно в любой момент, никаких замков.

## 🚀 Порядок (4 недели)

1. **Неделя 1:** 5 баз + Command Center + первые записи (POINT A, 3 проекта, 5 контактов).
2. **Неделя 2:** вторичные базы + первое ревью + Claude Code bootstrap.
3. **Неделя 3:** голосовой ввод + шаблоны анализа.
4. **Неделя 4:** первый форк (Дмитрий?) + итерация.

## ✅ 8 решений за тобой (R1)

1. Сколько баз — 8 / 11 / 14?
2. Голос — авто-черновики на всё или ручная очередь?
3. Сколько ниш в раздачу — 5 / 1-2 / только Layer 1?
4. Старт — базы или хаб сначала?
5. Автоматизация — ручной / скрипты / гибрид?
6. Апдейты Layer 1 — вручную или авто-pull?
7. Канал раздачи — публичная ссылка / инвайты / git?
8. Strategic — простой POINT A/B или полный аналог шаблонов?

партнёрам → возвращаешься к execution-plan (видео / Wave 1).
