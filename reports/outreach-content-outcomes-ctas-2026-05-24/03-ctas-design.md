### §0.2 R12 4 RUSLAN-LAYER extraction action classes (Default-Deny)

1. **extraction_beyond_share** — value extraction beyond explicitly agreed share
2. **wage_ratio_violation** — Mondragón 5:1 ratio cap violated (per Clan Charter)
3. **non_consensual_distribution** — distributing value without recipient consent
4. **fork_prevention_attempt** — preventing learner / partner from leaving without penalty

### §0.3 R12 8-item checklist (per CTA mandatory)

Per prompt §4.C:
1. **Offer explicit** — what value learner gets
2. **Ask explicit** — what we ask in return
3. **Voluntary opt-in** — learner can decline без penalty
4. **Fork-and-leave preserved** — learner can leave любой времени
5. **Cooperative share framing** — NOT extraction; cooperative value exchange
6. **Mondragón 5:1 cap mentioned where applicable** — Stage 6+ economic mechanics surface
7. **No manipulation** — no fake scarcity / fake social proof / fake urgency / FOMO
8. **Specific contact + timeline** — clear path forward
