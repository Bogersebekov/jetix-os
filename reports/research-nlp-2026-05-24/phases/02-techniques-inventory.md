# Phase 2 — Core NLP Techniques Inventory

> **R12 STRICT pre-screen per technique**: каждая technique surfaced carries (a) brief mechanism, (b) R12-compatibility check (transparent + covert), (c) empirical-status flag (pending Phase 6), (d) Phase 7 IMPORT/SKIP disposition.

> **Method**: O'Connor+Seymour *Introducing NLP* (1990) used as canonical layout (~50 techniques organized as standard practitioner reference); cross-referenced с Bandler+Grinder foundational corpus + Dilts Encyclopedia 2000.

---

## §2.0 NLP «Four Pillars» (frame)

O'Connor+Seymour articulate four foundational «pillars» of NLP practice:

1. **Outcome** — knowing what you want (well-formed outcomes)
2. **Sensory acuity** — noticing what you're getting (calibration)
3. **Behavioral flexibility** — varying what you do until you get what you want
4. **Rapport** — relationship of trust + responsiveness

[src: oconnor-seymour-introducing-nlp-1990 — three-minute-seminar p ~760-815]

**R12 pre-screen of frame itself**: «Behavioral flexibility until you get what you want» — **R12-sensitive framing**. If «what you want» = persuasion outcome over another person без consent, this becomes **persistence-until-compliance** = R12 violation. Founders' framing assumes own-outcome only; downstream popularization extended to influence-over-others. R12 STRICT pre-screen: pillar 3 applies к own behavior toward own outcome; **never** к behavior aimed at extracting other-person compliance без disclosure.

## §2.1 Outcome Specification — POSERS / Well-formed outcomes

### Mechanism

«POSERS» mnemonic from O'Connor:
- **P**ositive — что хочешь, не от чего бежишь
- **O**wn part — в твоей контроле
- **S**pecific — кто/где/когда/что/как
- **E**vidence — sensory-based evidence that lets you know you have it
- **R**esources — есть ли нужные
- **S**ize — chunk up/down to motivating + actionable

[src: oconnor-seymour-introducing-nlp-1990 p ~932-983]

Plus **Ecology check** — undesirable byproducts, who else affected, «would you take it now?» self-test.

### R12 + Empirical + Disposition

| Aspect | Note |
|---|---|
| Self-applied | ✅ R12 OK |
| Coach-assisted | ✅ R12 OK if transparent |
| Sales «outcome elicitation» | ⚠️ Borderline — depends on whose outcome is being elicited |
| Empirical | Aligns w/ Locke & Latham goal-setting research (robust RCT) + OKR practice |
| Phase 7 | **STRONG IMPORT** — maps to Method V2 §J component 2 (outcome specification with evidence procedure) + DR-38 + FPF F-G-R |

## §2.2 Sensory Acuity / Calibration

### Mechanism

Calibration = practitioner's discipline to notice **micro-behavioral changes** in client (breathing rate, skin tone, pupil dilation, muscle tonus, voice qualities) and **correlate them с reported internal states**. Idea: if Mary's breathing-rate-12 + slight-cheek-flush correlates с her saying «I felt confident in that moment», that calibration becomes signature for «Mary's confidence state».

> «To calibrate is to learn to read another person's responses precisely in an ongoing relationship.» [src: oconnor-seymour-introducing-nlp-1990 p ~2240-2289]

### R12 + Empirical + Disposition

| Aspect | Note |
|---|---|
| Self-applied (notice own states) | ✅ R12 OK |
| Coach-assisted clinical | ✅ R12 OK |
| Covert calibration for sales/influence | ❌ R12 violation (extracting cognitive-state information без disclosure for compliance leverage) |
| Empirical | Micro-expression literature (Ekman) supports detection of affective states; NLP-specific «state calibration via idiosyncratic micro-signs» = weaker but plausible; **systematic compliance-leverage application** unsupported |
| Phase 7 | **IMPORT for self-awareness** (own bodily state tracking); SKIP covert-on-other version |

## §2.3 Representational Systems (VAK) — fuller treatment

### Mechanism

Three primary internal representation channels:
- **V (Visual)** — pictures, brightness, distance, color, location
- **A (Auditory)** — sounds, tone, volume, tempo, internal/external
- **K (Kinesthetic)** — feelings, pressure, temperature, location, intensity
- (Plus **Olfactory** + **Gustatory** as auxiliary)

> «Some people will be primarily visual, internally generating images. Some people will be auditory, generating sounds or words internally. Some people will be kinesthetic, generating feelings or sensations.» [src: bandler-grinder-frogs-into-princes-1979 + oconnor p ~1304-1424]

### Sub-distinctions

- **Preferred Representational System (PRS)** — the system someone «leads with» / spends most internal time in
- **Lead System** — system used to ACCESS information (you might lead с visual but represent in kinesthetic)
- **Reference System** — system used to evaluate/confirm

[src: oconnor-seymour-introducing-nlp-1990 p ~1424-1646]

### Predicates by system

| System | Predicate examples |
|---|---|
| V | see, picture, look, focus, clear, perspective, dim, bright, illustrate |
| A | hear, sound, listen, tune, harmony, click, ring, loud, quiet |
| K | feel, grasp, touch, pressure, weight, solid, smooth, rough, warm, cold |

[src: oconnor-seymour-introducing-nlp-1990 p ~2042-2125]

### R12 + Empirical + Disposition

| Aspect | Note |
|---|---|
| Weak claim: notice predicates, adapt language → improved understanding | ✅ R12 OK |
| Strong claim: PRS as fixed cognitive type | ❌ **EMPIRICALLY FALSIFIED** (Sharpley 1984/1987; Heap 1988; subsequent reviews — Phase 6) |
| Pseudoscientific authority extraction («you're an auditory person») | ❌ R6 + indirect R12 |
| Empirical (weak claim) | Generic communication-matching aligns with active listening + accommodation theory; not specifically NLP |
| Empirical (strong claim) | **NOT supported** — no replicable evidence for stable PRS trait |
| Phase 7 | SKIP strong claim. Weak claim covered better by Bolton People Skills (Phase 5) without VAK pseudoscience baggage. **DEFAULT SKIP** VAK framework. |

## §2.4 Eye Accessing Cues

### Mechanism (claimed)

Looking direction indicates which internal sensory system being accessed:
- **Up-Left** — visual remembered (Vr)
- **Up-Right** — visual constructed (Vc)
- **Side-Left** — auditory remembered (Ar)
- **Side-Right** — auditory constructed (Ac)
- **Down-Left** — auditory internal dialogue (Ad)
- **Down-Right** — kinesthetic (K)

(Reversed for left-handed people, per founders.)

[src: oconnor-seymour-introducing-nlp-1990 p ~1646-1783; bandler-grinder-frogs-into-princes-1979 p ~804-830]

### R12 + Empirical + Disposition

| Aspect | Note |
|---|---|
| Empirical status | ❌ **EMPIRICALLY FALSIFIED** — Sharpley 1984/1987; Wiseman et al. 2012 *PLOS ONE* meta-test of eye-accessing-cues directly disconfirmed |
| R12 risk | High — promotes false expert authority over reader of cues |
| Phase 7 | **HARD SKIP** — do not propagate, do not «but it's useful as heuristic» — falsified claim cannot be soft-promoted |

## §2.5 Submodalities

### Mechanism

Within each rep system, finer distinctions modulating affective response:

**Visual sub:** bright/dim, near/far, large/small, framed/panoramic, color/B&W, focused/blurred, location-in-space, still/moving, associated (looking through own eyes) / dissociated (seeing self)

**Auditory sub:** volume, tone, tempo, location in space, mono/stereo, distance, internal/external

**Kinesthetic sub:** pressure, temperature, location, size of area, movement, duration, intensity

[src: oconnor-seymour-introducing-nlp-1990 p ~1856-1937]

### Submodality interventions

- **Swish pattern** — replace unwanted internal image with desired image using submodality contrast
- **Phobia cure (fast/V-K dissociation)** — visualize phobic memory while dissociated (seeing self from observer position) + B&W + small/distant → reduces fear charge
- **Belief change via submodality shift** — move belief from «certainty» submodality structure к «doubt» submodality structure

### R12 + Empirical + Disposition

| Aspect | Note |
|---|---|
| Self-applied imagery work | ✅ R12 OK |
| Clinical with informed consent | ✅ R12 OK |
| Empirical (generic imagery rescripting) | ✅ Strong RCT support (CBT, EMDR-adjacent) |
| Empirical (NLP single-session phobia cure as marketed) | ❌ No replicated RCT support |
| Phase 7 | OPTIONAL IMPORT — but CBT imagery rescripting literature provides better-validated equivalents |

## §2.6 Anchoring — fuller variants

Beyond foundational anchoring (Phase 1 §1.4):

### §2.6.1 Resource Anchoring (canonical)

1. Identify needed state (e.g. «calm-confidence»)
2. Client recalls specific past moment of that state, vividly
3. At peak of state recall, anchor applied (unique touch / unique tonal cue / unique image)
4. Test by re-firing anchor — does state return?
5. Future-pace — imagine upcoming situation; fire anchor; state available?

[src: oconnor-seymour-introducing-nlp-1990 p ~2393-2556]

### §2.6.2 Stacking Anchors

Same anchor applied across multiple resourceful-state recalls (calm + confident + competent + curious) → compound «super-resource» anchor.

### §2.6.3 Chaining Anchors

Sequence anchors A → B → C → D, designed to lead client from problem state (e.g. stuck) → bridge state (curious) → resource state (energized) → outcome state (committed).

[src: oconnor-seymour-introducing-nlp-1990 p ~2556-2594]

### §2.6.4 Collapsing Anchors

Negative state anchor + positive state anchor fired simultaneously → neural state interference → negative state diminishes.

[src: oconnor-seymour-introducing-nlp-1990 p ~2594-2662]

### R12 + Empirical + Disposition (anchoring family)

| Variant | R12 (self) | R12 (other-applied disclosed) | R12 (covert) | Empirical |
|---|---|---|---|---|
| Resource | ✅ | ✅ | ❌ | Conditioning robust; durability claim weaker |
| Stacking | ✅ | ✅ | ❌ | Generic compound conditioning |
| Chaining | ✅ | ✅ | ❌ | No RCT evidence specifically |
| Collapsing | ⚠️ unclear self-version | ⚠️ requires informed consent | ❌ | No replicated evidence; theory disputed |

**Phase 7**: SELF-anchoring = potential personal-productivity import (parallel to Atomic Habits cue-routine-reward). Other-applied = clinical only. Covert = HARD SKIP.

## §2.9 Reframing — complete catalog

### §2.9.1 Content reframing

- **Meaning reframe** — «Behavior B means X (negative)» → «Behavior B can also mean Y (positive)»
- **Context reframe** — «Behavior B is problematic in context C1» → «Behavior B is useful in context C2»

[src: bandler-grinder-reframing-1982 + oconnor]

### §2.9.2 Six-step reframing (process)

(Already covered Phase 1 §1.5.) Communication w/ part → positive intention → creative-part alternatives → ecology check → future-pace.

### §2.9.3 Six-step variants (Reframing 1982)

- **Negotiation between parts** — for internal conflict (e.g. «part wants security» vs «part wants adventure»)
- **Creating a new part** — when no existing part has desired capability
- **Visual squash** — parts integration via visualized merging of two parts

[src: bandler-grinder-reframing-1982 chapters III + IV]

### R12 + Empirical + Disposition

(Already in Phase 1.) ✅ Cognitive reframing — STRONG IMPORT (CBT-compatible, maps to R12 paired-frame). 6-step parts protocol — OPTIONAL (IFS better validated).

**Caveat**: Sleight of Mouth (Dilts Phase 3) = adversarial application of reframing patterns; SKIP for outreach use per parent prompt §2.7. (Reserved for internal R12 self-audit.)

## §2.10 Future Pacing

### Mechanism

After installing new behavior/anchor/belief, client mentally rehearses upcoming situation, imagining the new resource active. Bridges training-context → real-context.

[src: oconnor-seymour-introducing-nlp-1990 p ~2727-2771]

### R12 + Empirical + Disposition

✅ Self-applied mental rehearsal = robust empirical phenomenon (mental rehearsal literature, sports psychology). Aligns с implementation intentions (Gollwitzer). IMPORT.

### Mechanism

Client revisits painful past memory; therapist applies resource-state anchor at peak of memory; client re-experiences memory while resource-anchored → memory acquires «as if I had resource then» quality.

[src: oconnor-seymour-introducing-nlp-1990 p ~2662-2727]

| Self-applied | ⚠️ Caution — re-traumatization risk без support |
| Empirical | Imagery rescripting (Arntz et al.) = strong RCT for PTSD/trauma; NLP-specific «change personal history» = no separate RCT |
| Phase 7 | NOT RELEVANT TO JETIX — not a recruitment/communication technique; clinical intervention. SKIP from Jetix import list. |

## §2.12 New Behavior Generator

### Mechanism

1. Identify desired behavior
2. Visualize self performing it (dissociated — see self from outside)
3. Step inside the image (associate) — feel doing the behavior
4. Future-pace: imagine real upcoming context, run new behavior

[src: oconnor-seymour-introducing-nlp-1990 p ~2771-2801]

### R12 + Empirical + Disposition

✅ Self-applied mental simulation — aligns with mental rehearsal + behavioral activation literature. IMPORT as «personal practice» element.

## §2.13 Logical Levels / Neurological Levels (Dilts)

### Mechanism

Dilts' hierarchy of change levels (introduced briefly here, fuller in Phase 3):

1. **Environment** — when/where
2. **Behavior** — what
3. **Capability** — how
4. **Beliefs/Values** — why
5. **Identity** — who
6. **Purpose/Spiritual** — for whom/what greater

Claim: intervention at higher level cascades to lower levels; intervention at lower level alone often fails because higher-level structure constrains it.

[src: oconnor-seymour-introducing-nlp-1990 p ~3018-3101; dilts-delozier-encyclopedia 2000 throughout]

### R12 + Empirical + Disposition

| Aspect | Note |
|---|---|
| As framework | ✅ Heuristic OK — similar to other multi-level models (Maslow, Spiral Dynamics) |
| Empirical | No RCT specifically validating «neurological levels» as causal hierarchy; aligns roughly with self-determination theory but no specific empirical claim that level-5 intervention dominates level-2 |
| R12 risk | Identity-level intervention without consent (e.g. «you ARE a successful person») can be coercive identity-binding; SKIP that mode |
| Phase 7 | OPTIONAL — framework as map can structure questioning (similar to Jetix multi-tier structure); SKIP identity-binding application |

## §2.14 Belief Change Techniques

### Mechanism

Multiple protocols including:
- **Belief submodality shift** — find submodality structure of «belief I'm sure of» vs «doubt»; shift problem belief into doubt structure, install new belief in certainty structure
- **Museum of personal history** — visualize old/limiting beliefs as museum exhibits (dissociated)
- **Belief audit** — meta-model questioning of beliefs to surface assumptions/empirical basis

[src: oconnor-seymour-introducing-nlp-1990 p ~3324-3477]

### R12 + Empirical + Disposition

| Aspect | Note |
|---|---|
| Self-applied (belief audit) | ✅ R12 OK — Socratic discipline |
| Other-applied submodality belief shift | ⚠️ Powerful intervention; requires informed consent; potential R12 violation if used to install belief in service of practitioner's outcome |
| Empirical | Belief change literature (CBT) supports cognitive restructuring; NLP-specific submodality protocols = no separate RCT |
| Phase 7 | IMPORT belief-audit (self-applied / meta-model based); SKIP submodality belief installation by another |

## §2.15 Values and Criteria Elicitation

### Mechanism

> «Make a list of the 10 or so most important values in your life. ... Elicit your answers by asking such questions as: What's important to me? What truly motivates me? What has to be true for me?» [src: oconnor-seymour-introducing-nlp-1990 p ~5444-5485]

**Hierarchy of criteria** — relative importance ordering via stepping-up («what would have to be true for you to override criterion X?») yields **Criterion Equivalence** (which criterion is dominant when in conflict).

### R12 + Empirical + Disposition

✅ Self-applied OR transparent coaching = R12 OK. Aligns with values-clarification literature, ACT (Acceptance Commitment Therapy) values work.

**Phase 7**: IMPORT — maps to Jetix Cohort target ontology O-161/O-162 (understanding partner motivation via values discovery, voluntary + transparent).

⚠️ **Sales abuse-mode**: «discover client's values, then craft pitch to hit those values» = R12 violation (extraction_beyond_share via values-leverage без disclosure). NLP-sales training widely teaches this. SKIP that mode.

## §2.17 Strategies — TOTE + Elicitation

### Mechanism

Strategy = specific **sequence of representational system states** producing a behavioral outcome.

Notation: V → A → K → (test) → V (loop or exit)

Example elicitation of spelling strategy:
- Visualize word (Vc) — see it
- Check feeling (K) — does it «look right»?
- If yes: exit (write) | If no: revise visualization

[src: oconnor-seymour-introducing-nlp-1990 p ~6891-7396 strategies chapter]

### Elicitation protocol

1. Ask about specific behavior («когда вы делаете X хорошо, как именно это происходит?»)
2. Notice eye accessing cues + predicates indicating rep-system sequence
3. Test by interrupting sequence — does behavior break?
4. Install in someone else via leading them through same sequence

### R12 + Empirical + Disposition

| Aspect | Note |
|---|---|
| As skill-modeling discipline | ✅ Aligns with cognitive task analysis + apprenticeship literature |
| Single-session «install» claim | ❌ Unsupported — deliberate practice literature (Ericsson) shows skill transfer slower |
| Empirical | Eye-cue-based strategy elicitation = ❌ falsified (eye cues themselves falsified). Direct skill-modeling discipline (interview + observation + replication) = compatible with cognitive task analysis literature. |
| Phase 7 | IMPORT modeling discipline (without eye-cue reliance) → parallel to methodology-engineer ROY + DR-38 8-component method + Method V2 §J |

## §2.18 Timeline / Time Therapy

### Mechanism

Client visualizes own past-present-future as spatial line (often subjectively located: «in front of me» for future, «behind» for past, or various other configurations).

Interventions:
- **«In time» vs «through time»** — subjective experience type (immersed vs sequential observer)
- **Past-resource installation** — visit past moment of strength via timeline
- **Future-pacing on timeline** — drop new behavior into specific future date
- **Limiting decision identification** — walk timeline backward to find moment limiting belief installed; reframe at that point

[src: oconnor-seymour-introducing-nlp-1990 + James+Woodsmall *Time Line Therapy* 1988 (not in primary corpus but standard reference)]

### R12 + Empirical + Disposition

| Aspect | Note |
|---|---|
| Self-applied imagery on timeline | ⚠️ Compatible with imagery rescripting; trauma-competent practitioner needed if past-pain work |
| Empirical | Imagery rescripting (Arntz, Smucker) RCT-supported; NLP «timeline therapy» branded version = no separate RCT |
| R12 | Heavy clinical territory; not communication/recruitment technique |
| Phase 7 | NOT RELEVANT TO JETIX — clinical only. SKIP. |

## §2.19 Parts Integration / Visual Squash

### Mechanism

For internal conflict between two parts (e.g. «adventurer» vs «security-keeper»):
1. Identify both parts; place each on a hand (visualization aid)
2. Negotiate — what does each part want? (positive intention)
3. Find higher-level shared intention
4. Bring hands together — integrate symbolically; experience merged state
5. Future-pace integrated resource

[src: bandler-grinder-reframing-1982 + dilts-delozier-encyclopedia parts work]

### R12 + Empirical + Disposition

| Aspect | Note |
|---|---|
| Self-applied | ✅ R12 OK if voluntary |
| Therapeutic | ✅ Compatible with IFS (Schwartz) + Voice Dialogue (Stone) traditions |
| Empirical | IFS = emerging RCT base; NLP «visual squash» specifically = no separate RCT, but parts-work general framework empirically supportable |
| Phase 7 | OPTIONAL — clinical territory; not core to Jetix recruitment |

## §2.21 Selling / Negotiation / Meetings (NLP applied)

O'Connor dedicates a chapter to applying NLP in sales/business contexts [src: p ~5919-6332].

Pattern: rapport-build → calibrate values + metaprograms + rep-system → frame offer in matching language → pacing-leading к close.

## §2.22 Phobia Cure / Fast Phobia Cure

### Mechanism

V-K dissociation protocol:
1. Client visualizes self in cinema seat (dissociated position #1)
2. Sees self watching screen (dissociated position #2 — double dissociation)
3. On screen: B&W film of phobic memory, played start-to-finish at distance
4. Step into film at end (associate); run film backward in color very fast (≤2 sec); step out
5. Test — recall phobic stimulus; reduced response

## §2.23 Swish Pattern

### Mechanism

Submodality-based behavior change protocol:
1. Identify trigger (cue that precedes unwanted behavior — e.g. cigarette pack)
2. Visualize unwanted behavior outcome (e.g. self smoking)
3. Visualize desired outcome (self as non-smoker — fit, energized)
4. «Swish» — unwanted image shrinks to small dot in corner, replaced rapidly by desired image at large/bright
5. Break state; repeat 5-7 times
6. Test trigger; should auto-trigger desired image

[src: oconnor-seymour-introducing-nlp-1990 p ~6504-6582]

### R12 + Empirical + Disposition

| Aspect | Note |
|---|---|
| Self-applied | ✅ R12 OK |
| Empirical | Generic behavioral cue-replacement aligns with habit-replacement literature; NLP «swish» specifically — no separate RCT |
| Phase 7 | OPTIONAL — better-validated equivalent в Atomic Habits cue replacement + implementation intentions |

## §2.24 Frames

«Frame» = perspective / context for interpretation.

Standard NLP frames [src: oconnor p ~5969-6040]:
- **Outcome frame** — orient discussion toward outcome
- **Ecology frame** — consider broader consequences
- **Backtrack frame** — review what's been agreed
- **«As if» frame** — assume condition is true; what follows?
- **Contrast frame** — compare current to desired
- **Agreement frame** — find shared ground
- **Relevancy challenge** — «how does this relate to outcome?»

## §2.25 Cross-corpus reference: O'Connor's overall organization

O'Connor+Seymour *Introducing NLP* 1990 — the canonical textbook — is structured as:

1. Outcomes
2. Communication + Rapport
3. The doors of perception (rep systems + eye cues + submodalities)
4. Physiological states + emotional freedom (anchoring + states)
5. Loops + systems + levels
6. Talking sense (meta-model)
7. Uptime + Milton model
8. Metaphor + reframing
9. Values + criteria + metaprograms
10. NLP in practice (selling/meetings/negotiation/therapy)
11. Modeling

This ordering reveals NLP's **canonical taxonomy** at the time of textbook codification (1990). 16 years after the founding, NLP had stabilized into this ~30-technique core inventory.
