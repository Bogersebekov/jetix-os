# Phase 3 — Dilts + Applications

> **Pre-screen:** Dilts = «second generation» NLP — extended core 1970s patterns into **modeling discipline applied to historical geniuses** (Strategies of Genius series 1994-95) + **language patterns for belief change** (Sleight of Mouth 1999) + **encyclopedia/textbook reference** (Encyclopedia of Systemic NLP 2000). Of these, **Sleight of Mouth carries the highest R12 risk** for Jetix outreach context — adversarial reframing applied to others' objections = covert influence pattern. Per parent prompt §2: SOM = SKIP for outreach use; OK for internal R12 self-audit only.

## §3.0 Dilts' positioning

Robert Dilts = early 1970s student of Bandler+Grinder at UC Santa Cruz. Differs from founders in:
- Less emphasis on pure modeling of therapists; more on **modeling historical figures** (Aristotle, Socrates, Mozart, Disney, Tesla, Einstein, etc.)
- Introduced **Neurological Levels** (extended Bateson's Logical Levels)
- Codified **Sleight of Mouth** 14 patterns (collected from observing Bandler in seminars)
- Wrote the **NLP Encyclopedia** (2000, 1700+ pages with Judith DeLozier) — terminology reference

> «Sleight of Mouth patterns come from the study of how language has been used to elicit, identify, and shift core beliefs. ... Sleight of Mouth patterns have proved to be one of the most powerful sets of NLP techniques for verbal belief change.» [src: dilts-sleight-of-mouth-1999 p ~240+330+726]

**R12 epistemic flag**: Dilts' modeling claims about historical figures (Aristotle, Tesla, etc.) = **retrospective inference from text** — no direct observation, no validation against the modeled person's actual cognitive process. Methodologically weak. **Functions as authority-borrowing** for NLP framework — «NLP can model Aristotle's thinking» = unfalsifiable claim that lends prestige.

## §3.1 Sleight of Mouth — 14 patterns

### §3.1.1 The pattern set

| # | Pattern | Function |
|---|---|---|
| 1 | **Intention** | Direct attention to purpose/motive behind statement |
| 2 | **Redefining** | Substitute one of the words in statement for similar-meaning word with different connotation |
| 3 | **Consequences** | Surface future consequence (positive or negative) of holding belief |
| 4 | **Chunk Down** | Break generalization into specific component parts |
| 5 | **Chunk Up** | Generalize specific into larger category, often shifting connotation |
| 6 | **Analogy** | Make analogous comparison that recasts belief |
| 7 | **Change Frame Size** | Re-scale time/people/perspective frame |
| 8 | **Another Outcome** | Shift focus to different goal than the one motivating belief |
| 9 | **Model of the World** | Reframe belief as one perspective among many |
| 10 | **Reality Strategy** | Question the cognitive process used to construct/confirm belief |
| 11 | **Counter Example** | Provide specific instance that contradicts belief generalization |
| 12 | **Hierarchy of Criteria** | Surface higher-level criterion that overrides the criterion in belief |
| 13 | **Apply to Self** | Apply the belief recursively to the speaker stating it |
| 14 | **Meta Frame** | Comment on the belief from outside its frame (meta-level) |

[src: dilts-sleight-of-mouth-1999 throughout — patterns introduced chapters 2-7]

### §3.1.2 Worked example — Dilts' canonical case

Belief: «I'm afraid of failing.»

Each pattern produces a different verbal response:

| Pattern | Response |
|---|---|
| Intention | «What's important about not failing for you?» |
| Redefining | «You're not afraid of failing; you're being cautious about taking premature action.» |
| Consequences | «Letting fear stop you guarantees you won't try, which guarantees the failure you fear.» |
| Chunk Down | «Specifically which part of failing are you afraid of?» |
| Chunk Up | «Trying anything carries risk; you're really afraid of being human.» |
| Analogy | «Babies fall many times learning to walk; fear of falling would prevent walking.» |
| Change Frame Size | «In 5 years, how big will this fear seem?» |
| Another Outcome | «The real question isn't fear of failing; it's what you want to create.» |
| Model of the World | «Some people see attempts that don't succeed as data, not failure.» |
| Reality Strategy | «How specifically do you know you'd fail before trying?» |
| Counter Example | «You've succeeded before — what about those times?» |
| Hierarchy of Criteria | «Isn't growth more important than avoiding failure?» |
| Apply to Self | «You're so afraid of failing that you fail to act — proving the fear.» |
| Meta Frame | «That belief itself is producing the result you fear.» |

[src: dilts-sleight-of-mouth-1999 — multiple examples chs 2-7]

### §3.1.3 R12 STRICT pre-screen — **Sleight of Mouth is the most R12-sensitive Dilts contribution**

**Application contexts and R12 status:**

| Context | R12 status | Reasoning |
|---|---|---|
| **Self-applied** (audit your own beliefs by running 14 patterns on them) | ✅ R12 OK | Internal Socratic discipline |
| **Therapeutic with informed consent** (client knows: «we're going to challenge belief») | ⚠️ Borderline — depends on consent depth | Cognitive-therapy parallel; OK clinically |
| **Public debate / disclosed argument** («I'm going to argue against X») | ⚠️ Adversarial framing — OK if disclosed as argument | Rhetoric; not covert |
| **Outreach / sales — covertly applied to overcome client objection** | ❌ **HARD R12 VIOLATION** | Pattern's defining feature is to **dissolve other's belief** without their meta-awareness; extracts compliance |
| **Internal team self-audit** (audit own arguments via 14 patterns for weak points) | ✅ R12 OK | Same as self-applied — directed at own beliefs |

### §3.1.4 Why Sleight of Mouth in outreach = R12 violation specifically

The structural critique:

1. **Asymmetric power** — practitioner has 14-pattern arsenal; respondent doesn't know which pattern is being deployed. Asymmetric awareness = covert manipulation if respondent doesn't know game.
2. **Extraction over share** — pattern's purpose = move respondent from «I won't buy» to «I will buy» WITHOUT respondent acquiring new factual information. Extraction of decision change purely via verbal pattern = R12 extraction_beyond_share if conversion creates value flow to practitioner.
3. **Fork-prevention risk** — strongest patterns (Consequences + Apply to Self + Meta Frame) make exit feel cognitively unwise («letting fear stop you guarantees failure» — applies pressure to act). Erodes fork-and-leave option.
4. **Empirical status** — NLP claim that SOM produces durable belief change в other person без their cooperation = unsupported by RCT. Even when «works» as discussion technique, persistence of changed-belief weak.

> **SKIP-list entry (LARGE-15)**: Sleight of Mouth deployed adversarially in **outreach / sales / recruitment** = HARD SKIP. Maintained ONLY for self-applied internal audit (parent prompt §2.7 explicit allowance).

### §3.1.5 What R12-compatible alternative looks like

Per Welcome-frame O-144 + R12 paired-frame discipline:
- Instead of «dissolve their objection via SOM», **surface the objection explicitly**: «That sounds like a real concern. What specifically would have to be true to make this work for you?»
- Instead of «Apply-to-Self attack», **paired-frame disclosure**: «Here's the offer + here's its limitation; if either is wrong for you, walking away is the right move.»
- Instead of «Counter-Example pressure», **fork-and-leave reinforcement**: «If after seeing X you're still not aligned, the right call is don't engage.»

This is the **structural inversion**: SOM resolves objection by reframe; R12 paired-frame resolves objection by validating the partner's right to NOT engage.

[src: dilts-delozier-encyclopedia-of-systemic-nlp-2000 — alphabetical entries throughout]

### §3.2.1 Notable Dilts-specific contributions surfaced in Encyclopedia

**SCORE Model** (Symptom, Cause, Outcome, Resource, Effect):
- **Symptom** — presenting problem
- **Cause** — underlying causes
- **Outcome** — desired result
- **Resource** — what's needed
- **Effect** — long-term consequences of achieving outcome

Used as diagnostic frame in coaching. R12 OK self-applied; potential coaching tool.

**Neurological Levels (Dilts hierarchy)** [reprise from Phase 2 §2.13]:
1. Environment
2. Behavior
3. Capabilities
4. Beliefs & Values
5. Identity
6. Spiritual / Purpose

[src: dilts-delozier-encyclopedia-of-systemic-nlp-2000 — Neuro-Logical Levels entry]

**Perceptual Positions** (extending Bateson triple description):
- **First Position** — self
- **Second Position** — other person
- **Third Position** — neutral observer
- **Fourth Position** — system / collective

This is one of the more durable NLP contributions; perspective-taking discipline.

**Logical Levels of Change** (Dilts coaching application):
- Generative coaching = work at level above presenting problem
- E.g. behavior problem? Look at beliefs. Identity crisis? Look at purpose.

**Imprint** (early experience installing belief/strategy):
- Erik Erikson + developmental psychology adjacency
- Dilts protocol: «Re-imprint» — return to imprint moment with resource (similar to imagery rescripting)

**Spiritual / Generative NLP** (later Dilts work):
- Field-based work; beyond individual cognition
- More speculative; less empirical grounding

### §3.2.2 R12 + Empirical status — Encyclopedia overall

The Encyclopedia is **terminological reference**, not RCT-validated knowledge. Its function = canonicalize vocabulary. Per Phase 6 critical review, the Encyclopedia entries inherit empirical status of underlying claims (most: unvalidated by independent research).

**Phase 7 disposition for Encyclopedia content:** Use as terminology reference; do not promote any entry to Tier 2 / Foundation; specific high-value imports (Perceptual Positions, SCORE) noted separately.

## §3.3 Strategies of Genius — Dilts Modeling Discipline

Dilts' **Strategies of Genius** series (3 volumes, 1994-95):
- Vol I: Aristotle, Sherlock Holmes, Disney, Mozart
- Vol II: Einstein
- Vol III: Tesla, Freud, Leonardo da Vinci

Method claim: Dilts «modeled» each figure's cognitive strategy from their writings/biographies, extracted strategy sequence (TOTE structure), and offered as installable in modern practitioner.

### §3.3.1 The «Disney Strategy» as illustrative example

Most-cited Dilts modeling result. Disney supposedly used three distinct «rooms» / positions:
- **Dreamer** — ideation, no constraint
- **Realist** — implementation specifics
- **Critic** — find weakness, constructive critique

Move through three positions sequentially when developing project. Each position has different physiology (where to sit, what to ask).

### §3.3.2 R12 + Empirical status

| Aspect | Note |
|---|---|
| As **brainstorming protocol** (move through three positions) | ✅ R12 OK + useful regardless of empirical basis |
| As **claim that Disney actually used this strategy** | ❌ Methodologically unfounded — retrospective inference from text |
| As **«install Disney genius»** marketing | ❌ Pseudoscientific overclaim |
| Empirical | Multi-position thinking (perspective shifts) supported by creative-cognition literature; specific «Disney Strategy» branding ≠ historical accuracy |
| Phase 7 | IMPORT the **3-position protocol** (Dreamer/Realist/Critic) as **brainstorming heuristic** — discard claim about Disney specifically. Equivalent to «yes-and / red-team / steelman» discipline in modern engineering. |

### §3.3.3 General methodological critique of Dilts modeling

Founders' modeling (Bandler+Grinder) = **direct observation** of living therapists (Perls, Satir, Erickson) — methodology stronger.

Dilts modeling of historical figures = **textual hermeneutics dressed as cognitive modeling** — methodology weaker. The figures are dead; we have only writings + biographies, filtered through Dilts' own NLP-prior framework. Outputs reveal more about Dilts' framework than about modeled figure.

**R12 implication**: Selling «Tesla's strategy installable in you» / «Einstein's thinking process modeled» = R12-sensitive overclaim if priced as transformative. Falls under empirical_falsified_overclaim class.

> **SKIP-list entry (16)**: Dilts «modeling historical genius» marketing claims as installable strategies — surface as **brainstorming heuristics** stripped of identity-binding marketing.

## §3.4 Belief Change Cycle (Dilts)

Six-stage belief change model in Sleight of Mouth book [src: dilts-sleight-of-mouth-1999 chs 12-13]:

1. **Wanting to believe** — desire for new belief
2. **Open to believing** — willingness to test
3. **Currently believe** — present belief state
4. **Open to doubting** — willingness to question
5. **Used to believe** — past tense; transitioned
6. **Museum of personal history** — old beliefs as historical artifacts

Pattern: move limiting belief from #3 («I currently believe I'm not good enough») through #4 + #5 + #6, while parallel-installing new belief at #1 → #2 → #3.

### §3.4.1 R12 + Empirical

| Aspect | Note |
|---|---|
| Self-applied | ✅ R12 OK |
| Coach-assisted disclosed | ⚠️ Powerful; informed consent essential |
| Empirical | Cognitive restructuring (CBT) base supports general framework; specific Dilts Belief Change Cycle protocol = no separate RCT |
| Phase 7 | OPTIONAL — better-validated equivalent in CBT cognitive restructuring + ACT defusion |

## §3.5 Modeling Discipline — Dilts' methodological proposal

### Dilts' modeling protocol

1. **Identify exemplar** of skill/state
2. **Observe** behavior + ask about cognition (predicates + eye cues + reported strategies)
3. **Identify TOTE structure** (Test-Operate-Test-Exit)
4. **Identify supporting beliefs + values + identity-level structure** (Neurological Levels)
5. **Subtract** exemplar's idiosyncratic features
6. **Test** by installing in novice; does it produce same result?

[src: dilts-delozier-encyclopedia-of-systemic-nlp-2000 + dilts modeling literature]

### §3.5.1 R12 + Empirical + Disposition

| Aspect | Note |
|---|---|
| As cognitive task analysis (CTA) | ✅ Compatible with extensive CTA + apprenticeship literature |
| Eye-cue reliance | ❌ Eye cues falsified; modify protocol to remove this step |
| Specific TOTE format | ✅ Cybernetics base solid |
| Identity-level installation | ⚠️ R12-sensitive; install own identity? OK; have practitioner install on you? Coercive |
| Empirical | Modeling/CTA general framework OK; NLP-specific «install in single session» claim unsupported |
| Phase 7 | **STRONG IMPORT** modeling-as-discipline (stripped of eye-cue + single-session claims) — parallel to methodology-engineer ROY + DR-38 8-component meta-method + Method V2 §J |

## §3.6 Pattern: «Coach the coach» / Logical Level alignment (Dilts coaching)

Dilts' practitioner-level coaching framework:

Levels alignment exercise:
1. Have client articulate at each Neurological Level: «In context C, who am I (Identity)? What do I believe (Belief)? What can I do (Capability)? What am I doing (Behavior)? Where + when (Environment)?»
2. Notice **incongruence between levels** (e.g. Belief level says «I value health» but Behavior level shows opposite)
3. Work on **alignment** — adjust higher levels first

[src: dilts-delozier-encyclopedia-of-systemic-nlp-2000 — Coaching + Neuro-Logical Levels entries]

| Empirical | No specific RCT validating «Neurological Levels» causally; framework parallels Self-Determination Theory + value-congruence literature loosely |
| Phase 7 | OPTIONAL framework — useful map but not unique to NLP |
