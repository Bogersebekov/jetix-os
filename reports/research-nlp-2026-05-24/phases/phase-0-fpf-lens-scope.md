## §0.2 Why NLP needs DIFFERENT discipline than propaganda research

Propaganda research (parallel bucket finished 2026-05-24) imported 65 substrate items + skipped 70 — но **propaganda corpus = mostly empirically validated psychology + sociology** (Cialdini, Kahneman = peer-reviewed; Bernays + Lippmann = historical record).

NLP corpus is **structurally different:**

1. **Mixed empirical status** — некоторые claims (rapport, modeling, well-formed outcomes) overlap с standard counseling psychology; другие (eye accessing cues, VAK preferred representation system, fast phobia cure single-session) **empirically falsified** by Sharpley 1984/87 + Heap 1988 + Witkowski 2010 35-year review.
2. **Commercially monetized** — Robbins seminars, Bandler workshops, NLP-certified-coach industry created strong incentives для overclaim.
3. **Origin story controversy** — Bandler+Grinder 1970s Santa Cruz; Grinder later distanced («New Code NLP»); legal disputes about IP/certification authority.
4. **Manipulation toolkit risk** — Milton model (covert hypnosis), Sleight of Mouth (objection-attack patterns), anchoring без consent — direct R12 violation если deployed in sales/recruitment без disclosure.

**Therefore:** Phase 6 critical review is NOT optional polish — it's the discipline that decides which techniques pass R12 filter into Phase 7 Jetix Lens import vs. SKIP-list.

## §0.7 R12 pre-screen taxonomy (inherited from propaganda bucket)

From RUSLAN-LAYER 2026-05-18 `.claude/config/default-deny-table.yaml` action classes — same 4 classes used in propaganda bucket apply here:

1. **extraction_beyond_share** — technique that lets organizer extract value beyond agreed share (e.g. covert anchoring, hidden Milton inductions, NLP-sales scripts that elicit purchases без informed consent)
2. **wage_ratio_violation** — technique creating dependency where members work for sub-share return (e.g. NLP-cert pyramids that bind trainees к further unpaid trainings; Robbins-style scarcity-induced upsells)
3. **non_consensual_distribution** — technique distributing member's content/identity без consent (e.g. testimonial extraction via altered state, covert recording for «modeling»)
4. **fork_prevention_attempt** — technique creating exit-friction or punishment for leaving (e.g. self-anchoring to leader; identity-level reframes binding self-worth к group)

**Plus** NLP-specific addition for Phase 6:

5. **empirically_falsified_overclaim** — claim falsified by 35-year empirical review yet still marketed as cognitive truth (e.g. eye accessing cues, VAK as fixed types, single-session phobia cure). Surfacing such техник without empirical-status flag = R6 violation + indirect R12 (extracts trust from learner via pseudoscientific authority).
