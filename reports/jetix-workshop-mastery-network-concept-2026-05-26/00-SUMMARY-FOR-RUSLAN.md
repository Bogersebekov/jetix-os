# 📋 SUMMARY для Руслана — Workshop + Mastery + Network

> 10 минут. Полный main = 45-60 мин. Самое живое — §7 Worked examples.

## Что это в одной фразе

Твой voice 26.05 («мега-мастерская») развёрнут в **тело Vision'а**. До сих пор Jetix описывался
абстрактно (система / платформа / кооператив). Теперь у него есть **место, люди, прогрессия**:
мастерская. Добавлено **3 новых направления** поверх 11 метаплана-v2 → **14 total**, + Vision
переписан с мастерской как центральной метафорой.

## 3 новых направления (3 грани одного объекта)

- **#12 🏛️ Мастерская** = *где*. Пространство из 8+ зон: стена инструментов (улучшаешь / ставишь
  новый станок), исследовательский центр, зона мастерства, зона встреч/клуб, тренировочный +
  спортзал, медитация, отдых. Не курс, не коворкинг, не bootcamp. Tacit knowledge (Полани) — уже в
  Method V2: «Workshop = hands-on, не лекции».
- **#13 🎯 Мастерство** = *что прокачивают*. Накопление методов + **выбор нужного в нужный момент**
  (твой voice). Meta-method уровень 3 (дизайн стратегии выбора). AI делает рутину, человек идёт на
  фронтир (O-182). Вечно тренируешь. Масса, не элита (O-184). Ядро = твои O-176..O-185.
- **#14 🌍 Сеть** = *как распределено*. Mesh из локальных cells (не star — это R12-требование).
  Ресурсы в общий пул (знакомства/склады/хаты/машины/идеи/деньги/инфо — всё opt-in + consent).
  Передвижение людей. Одна сеть с лучшими наработками. **Primary R12 surface.**

## Почему это critical

1. **Vision получает тело** — был абстрактным, стал местом, куда приходишь утром и выходишь немного
   бо́льшим мастером.
2. **Workshop = primary frame наружу** — маршрут партнёра теперь начинается с «вот мастерская», а
   не с абстрактного «видения». Снимает риск «звучит эзотерично».
3. **14 направлений, но не каша** — 3 новых = тело; 11 старых встраиваются как детали (Метод =
   педагогика, Платформа = станки, Заработок = экономика Сети) + дуга (Видение/Master Plan).
4. **3 хаба** (было 2): #1 Метод · #8 R12 · **#12 Мастерская (новый)**.

## Online → Offline (4 фазы по «кто крутит маховик»)

Year 1 Build (виртуальная, основатель руками) → Year 2 Run (hybrid, первое **арендованное** место
Берлин, +партнёры+петля) → Year 3-5 Scale (3-10 мастерских, mesh, founder = 1 из многих) → Year 5+
Mature (30-100+, member-owned, сеть сама). **«Быстро оффлайн» (твой voice) уточнил:** первое место
= арендованное, только при подтверждённой Run-петле (anti-WeWork). Цифры — сценарные, не обещания.

## R12 (Сеть = primary surface)

Каждый механизм роста идёт **парой**: «как растит» + «как НЕ становится сектой/extraction».
Анти-секта чек-лист: mesh не star · founder уход не ломает · split публичен 5:1 · fork-and-leave
забираешь долю · вход открыт 1-4 free · локальная автономия · метрика рост не время · pooling
opt-in+consent · **ВЫХОД ЗВУЧИТ ПЕРВЫМ**. Закон: защита растёт **быстрее** системы.

Тонкая зона (твоё решение, §11 п.5): «голодные/натренированные/дисциплинированные» как культурный
вектор — **не enforced**. Нельзя выгнать за «недостаточную дисциплину». Граница между культурой
развития и сектой держится явно.
