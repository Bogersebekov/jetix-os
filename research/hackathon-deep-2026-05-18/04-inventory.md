## §1 11 major platforms catalogued

### §1.1 General hackathon platforms

#### 1. MLH (Major League Hacking)
- **URL:** https://www.mlh.com/
- **Founded:** 2015 by Mike Swift + Jonathan Gottfried (see profile 05)
- **Scale:** 1M+ developers; 150K hackers in 2025 Season; 184 hackathons; 12,500 projects
- **2026 Season:** July 2025 - June 2026; 200+ events
- **2026-02-18 acquisition:** DEV community platform (3M developers)
- **Format:** Student-friendly + beginner-friendly gold standard; virtual every weekend
- **Sponsorship model:** Tiered (gold / silver / bronze + API sponsor)
- **Info-flow profile:** HIGH community / MEDIUM depth / STRONG mentor pipeline
- **Jetix engagement candidate:** A1+A2 strong fit; sponsor tier А3 candidate if Year-2+

#### 2. Devpost
- **URL:** https://devpost.com/
- **Founded:** 2009 (originally Challenge Post)
- **Scale:** Hosts most hackathons (MLH events + corporate); 1M+ users
- **Model:** Hosting platform (not organizer typically)
- **Format:** Project submission + judging infrastructure
- **Info-flow profile:** Aggregator pattern; passive info-flow
- **Jetix engagement candidate:** Use as platform для Hypothesis C events

#### 3. Devfolio (India + global)
- **URL:** https://devfolio.co/
- **Founded:** Jan 2019
- **Scale:** 35+ hackathons since launch; India focus + global Web3
- **Notable events:** ETHIndia (annual); HackThisFall (month-long virtual + 5 build stations); Warpspeed (Bengaluru AI agents); Build India Hackathon (with Anthropic + Replit + Lightspeed 2026-02-15)
- **Info-flow profile:** STRONG Web3 / Indian ecosystem; depth events (ETHIndia Villa = «more intentional, smaller circle»)
- **Jetix engagement candidate:** A1+A2 fit для Asia outreach; observation Phase 1

#### 4. HackerEarth
- **URL:** https://www.hackerearth.com/
- **Founded:** 2012 (India origin)
- **Scale:** **10M+ developers** community
- **Model:** Hackathon hosting + technical recruitment + assessments
- **Format:** Coding + challenge hosting + scoring infrastructure
- **Info-flow profile:** MEDIUM-HIGH community / strong recruitment focus / less open-source emphasis
- **Jetix engagement candidate:** Lower priority — recruitment-focused model less aligned

#### 5. AngelHack
- **URL:** https://angelhack.com/
- **Founded:** 2011
- **Scale:** 300,000+ developer network; StackUp platform 200K+ active
- **Format:** Global cities series; in-person + remote; bootcamps + follow-on funding для standout projects
- **Info-flow profile:** HIGH international reach / MEDIUM depth / STRONG mentorship
- **Jetix engagement candidate:** A1+A2 candidate especially для emerging markets

#### 6. Hackathon.com
- **URL:** https://corporate.hackathon.com/
- **Founded:** Aggregator + corporate hackathon services
- **Model:** Listing + corporate hackathon production services
- **Info-flow profile:** Aggregator pattern; corporate focus
- **Jetix engagement candidate:** Discovery only; not direct engagement

### §1.2 Web3-specific platforms

#### 7. ETHGlobal
- **URL:** https://ethglobal.com/
- **Founded:** 2017 (originally ETHWaterloo)
- **Scale:** Major events 1,000-3,000 participants; annual ETHOnline 5,000+
- **2026 events:** Cannes (Apr 3-5) · New York (Jun 12-14) · Lisbon (Jul 24-26) · Tokyo (Sep 25-27) · Mumbai Q4
- **Themes 2026:** Zero-Knowledge Proofs / privacy / AI x Crypto / autonomous agents
- **Prize model:** Sponsor bounties (~$50K each × 10-20 sponsors)
- **IP handling:** Open-source mandatory; participant retains
- **Info-flow profile:** HIGH-HIGH (depth + breadth); STRONG ecosystem
- **Jetix engagement candidate:** A1+A2+A3 strong fit; aligns Ethereum architecture decision (text_007)

#### 8. ETHDenver
- **URL:** https://ethdenver.com/
- **Founded:** 2018
- **Scale:** **25,000+ from 125 countries** (largest blockchain hackathon)
- **2025 prize pool:** $1,029,833 USD + up to $10M in grants/investments
- **Sponsors 2025:** EigenLayer, Flow, Hedera, Base, BNB Chain (each $50K+), Optimism, Internet Computer, Gravity Galxe, Wormhole, zkSync
- **Format:** Week-long BUIDLathon festival
- **Tracks:** Infrastructure+Scalability / Identity+Privacy+Security / DeFi+NFTs+Gaming / DAOs+Communities / Impact+Public Good
- **Info-flow profile:** MAX scale / EXTREMELY HIGH depth / STRONG VC + investor pipeline
- **Jetix engagement candidate:** A1+A2+A3 high priority; aligns Ethereum architecture; potential C event venue

### §1.3 AI-focused platforms

#### 9. Lablab.ai
- **URL:** https://lablab.ai/
- **Scale:** 100,000+ innovators worldwide
- **Format:** 7-day virtual hackathons; ~3,000 spots per event
- **Sponsors:** Anthropic (Claude) · OpenAI · Mistral · Meta · TogetherAI (rotating)
- **Themes:** AI agent building / specialized verticals (Legal AI / Enterprise / Customer Service)
- **IP handling:** Participant retains; open-source encouraged
- **Info-flow profile:** STRONG AI community / continuous cadence / MEDIUM depth (7-day vs week+ at ETHGlobal)
- **Jetix engagement candidate:** A1+A2 strong fit; aligns Anthropic ecosystem

#### 10. AI Grant (Nat Friedman + Daniel Gross / NFDG)
- **URL:** https://aigrant.org/
- **Founded:** 2017
- **Scale:** ~60 companies funded by 2024; Batch 4 2026
- **Model:** Accelerator (NOT typical hackathon); $250K SAFE per startup + $350K Azure + $250K partner credits
- **NFDG fund:** $1.1B committed capital (2023)
- **Portfolio notables:** Pika, Browserbase, Cursor, Granola ($43M Series B at $250M valuation), Perplexity
- **Info-flow profile:** ULTRA-HIGH depth / SMALL cohort / EXTREMELY HIGH downstream impact
- **Jetix engagement candidate:** Different format (accelerator не hackathon); potential application для Workshop / FPF substrate fund
