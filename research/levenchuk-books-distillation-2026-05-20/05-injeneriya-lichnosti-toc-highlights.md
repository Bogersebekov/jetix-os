## §3 Key Chapter Highlights

### §3.1 Глава 1 — Инженерия личности и системная инженерия (стр. 9, body line 233) ⭐⭐

**Самая важная глава.** Структурный тезис: «инженерия личности = обучение». Точно так же, как «менеджмент» = инженерия организации, «инженерия личности» = инженерия процесса обучения. Целевая система — мастерство студента (то, чему учат). Создатели мастерства — учительская команда (методолог, методист, преподаватель, культуртрегер, администратор обучения, архитектор личности). У целевого мастерства есть **описание** (digital twin) — формальная спецификация того, что человек умеет делать после обучения. Адаптация системной схемы проекта (из «Системного мышления 2024») к проекту учебного курса. Внешние роли проекта обучения — студенты, заказчики, регуляторы, рынок труда. Учительская команда — это её отдельная орг-модель.

**Ключевые термы:** инженерия личности / мастерство / описание мастерства / цифровой двойник мастерства / учительская команда / методолог / методист / преподаватель / культуртрегер / архитектор личности / администратор обучения / провайдер обучения / коммерческая возможность обучения / LXP.

### §3.2 Глава 2 — Практика культуртрегерства (стр. 56, body line 2427)

Культуртрегер — роль агента, который приводит студента к точке, где обучение становится интересным. Его задача — пробудить желание учиться. «Алгоритм культуртрегерства: его нет!» (это импровизационная роль). Левенчук подчёркивает — коммерческая возможность обучения возникает только после культуртрегерской работы (студент должен захотеть платить). Это прямой anchor к Outreach concept doc F2: outreach = culturologist-as-marketing.

**Ключевые термы:** культуртрегер / усиление интеллекта / алгоритм культуртрегерства / коммерческая возможность обучения / alpha коммерческой возможности.

**Predicted overlap:** Outreach concept doc F2 (outreach = культуртрегерство-как-маркетинг); KA-01 Левенчук pitch (Левенчук как культуртрегер, привлекающий студентов в ШСМ); sales-outreach agent.

### §3.3 Глава 3 + Глава 4 — Методологическая + Методическая работа (стр. 67, 91; body lines 2877, 3812) ⭐

Двойная глава: что учить (методология) + как учить (методика). Методолог отвечает за **alpha описания мастерства** — формальный документ, описывающий что студент должен уметь после обучения (это «образовательный стандарт» в терминах Левенчука). Кругозорное обучение (general education / понимание) — глубже, дольше, не для прикладной работы. Профессиональное обучение — навыки для конкретной работы.

Методист отвечает за **alpha метода обучения** — как именно учить (mastery learning / перевёрнутый класс / blended / collaborative). Когнитивистика и нейронаука как наследники психологии. **Конструктивизм** (студент строит знания сам) vs. **конструкционизм** (студент строит знания через создание артефактов). Навыки учим вдвое быстрее, чем понимание. LXP — Learning Experience Platform — инструмент методиста.

**Ключевые термы:** методология обучения / методика обучения / alpha описания мастерства / alpha метода обучения / образовательный стандарт / кругозорное vs. профессиональное обучение / конструктивизм / конструкционизм / mastery learning / перевёрнутый класс / collaborative interactive learning / LXP / blended learning / transfer of learning.

**Predicted overlap:** Education Layer concept doc + ВУЗ Jetix / Jetix Workshops (alpha описания мастерства = образовательный стандарт Jetix Курсу X); mastery-formula wiki; persistence wiki batch-7 (transfer of learning).

### §3.4 Глава 8 — Практика администрирования обучения (стр. 168, body line 7257) ⭐

### §3.5 Глава 10 — Примеры инженерии личности (стр. 183, body line 7954)

Финальная глава с тремя примерами: **психотерапия** = обучение мастерству «не бояться пауков»; **лидерство** = обучение мастерству «добросовестно работать в нашем проекте»; **продвижение** = обучение мастерству «покупки наших сервисов». Это reframing: всё то, что мы обычно называем «лечение / лидерство / маркетинг», — на самом деле проекты обучения. Это методологически очень важно для Jetix outreach: каждый pitch = обучающий проект, не «продажа».

**Ключевые термы:** психотерапия как обучение / лидерство как обучение / продвижение как обучение / маркетинг как обучение мастерству покупки.

**Predicted overlap:** Outreach concept doc (продвижение = обучение); Hackathon concept doc (хакатон = обучающий формат); KA-01 Левенчук pitch (любой pitch = обучающее воздействие).
