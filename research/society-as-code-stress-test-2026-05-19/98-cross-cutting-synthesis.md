## §1 12 patterns

### Pattern 1 — Toffler-Castells-Lessig lineage = 1980-2006 info-society-as-code progression

Information-society precedent traceable as **continuous progression**: Toffler 1980 «information» (passive) → Castells 1996 «protocols + flows» (sociological) → Lessig 1999 «code» (executable + political). Each stage gains semantic depth toward Jetix metaphor.

**Implication:** Society-as-code = **mature framing** with 40-year predecessor lineage; not novel-by-itself. Jetix differentiation = synthesis + operationalisation, NOT primary metaphor invention.

[src: `02-toffler` + `03-castells` + `04-lessig`]

### Pattern 2 — Lessig «code is law» = closest direct precedent

Lessig 1999/2006 = closest single-thinker precedent (executable code semantic + 4-modality framework). **Lessig 2011 «Republic, Lost»** extends scope cyber → democratic institutions — partial precedent для Jetix extension к ALL society. **DAO hack 2016** = empirical refutation case for strict «code is law».

**Implication:** Jetix must acknowledge Lessig debt + DAO refutation lesson — strict «code = law» empirically false; **«code as one regulator among 4»** (Lessig framework) is the responsible position.

[src: `04-lessig` §1.1-1.4 + §4.3]
