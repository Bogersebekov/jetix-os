## §3 Top 5 key findings

### §3.1 Hybrid FPF-natural language is the recommended default (Phase 3 §5)

Pure FPF — high cognitive overhead, narrow audience reach. Pure natural language — low epistemic discipline, confidence inflation risk. **Hybrid layered** (natural language surface + FPF annotations selective + F-grade explicit for contested claims + R receipts for forecasts) = best-of-both per Phase 3 quadrant analysis. Audience-adaptive per Phase 3 §5.2 + Phase 4 audience styling map.

### §3.2 Cialdini × R12 paired-frame audit = constitutional discipline (Phase 2 §6.4)

Of Cialdini's 7 influence principles, **Scarcity 🔴 + Commitment 🟡** require explicit R12 anti-extraction audit pre-send. Specifically: «only 3 spots left» / FOMO manipulation = R12 violation; «first-cohort 5-15 founding engineers Q3 2026» = real capacity scarcity (OK). Pre-send checklist needs expansion from current ONE-PAGER §6.4 8-item to 15-item incorporating Cialdini-per-principle audit (D-COMM-2 decision).

### §3.3 ≥3 Левенчук cross-cites = Schramm shared-field + Heath Credible operationalized (Phase 1 §3.4 + Phase 4 §5.3)

Левенчук substrate plays triple role: Schramm shared experience field (RU systems audience trust) + Heath Credible (anti-authority via external corroboration) + Cialdini Social Proof (verifiable via line-offset citations). **Discipline:** ≥3 specific cross-cites per outreach material (book + chapter + line offset) — already in DISTRIBUTION-PLAN §1.1 acceptance criteria; reaffirmed.

### §3.4 R-batch-9-N3 timing-hubris paraphrase = L3 institutional + humanitarian survival kit (Phase 4 §3.6 + §4.6)

«сейчас уже должны заниматься такие системы» (audio_709 verbatim) = institutional disqualification risk. Paraphrase «one of the systems contributing to convergence» = HR-4 mitigation. Mandatory для L3 + humanitarian materials. Phase 7 R3 recommendation makes pre-send linter / discipline explicit.

### §3.5 25-cell time-budget × audience matrix = anti-information-overload discipline (Phase 6 §6.2)

Match content density к recipient time-budget. 30-sec elevator ≠ 1-day immersion. Anderson TED Goldilocks logic applied per (audience × budget) cell. Operational handle: pre-send identify cell; deliver content per Phase 6 §2 matrix. CRM `outreach_log.jsonl` schema augment proposed (D-COMM-3 decision).
