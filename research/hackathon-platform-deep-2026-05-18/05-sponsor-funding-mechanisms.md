## §1 Mechanism 1 — Quadratic Funding (QF)

### §1.1 Description
Matching pool + crowd-quadratic contribution: matching ∝ (√(individual contributions))². Many small contributors > one large. Gitcoin canonical implementer (>$60M distributed since 2018). [src: Gitcoin docs, public:retrieved_pre-2026 + Phase 1 §2.5]

### §1.2 R12 fit (anti-extraction)
**Very high.** QF inherently anti-plutocratic: $100 from 100 individuals > $10K from 1 individual after matching. Aligns с R12 Tier 2 rule 12 (anti-extraction beyond agreed share). [src: Pillar C §4.1 + r12-programmable-ethereum packet]

### §1.3 Pros / cons
- **+ R12-native:** distributes power to many contributors.
- **+ Trust-substrate:** transparent on-chain.
- **+ Sponsor-friendly long-term:** sponsor contributes to matching pool; many contribute small.
- **− Setup complexity:** Ethereum / Optimism / Polygon infra required.
- **− Tax / regulatory uncertainty** в EU.
- **− Sybil-resistance** needs Passport / verified identity (Gitcoin Passport).

### §1.4 Sponsor profile fit
- **Best fit:** Crypto-native sponsors (Ethereum Foundation, Optimism, Polygon) OR sophisticated corporate (Anthropic).
- **Med fit:** EU AI corporates (Bosch, SAP).
- **Low fit:** Traditional RU corporate (Sber/Tinkoff/Yandex) — unfamiliarity barrier.

### §1.5 Jetix applicability
- **Year-1:** **matching pool mode** — sponsor contributes €5K-50K, matched by crowd contributions (small amounts).
- **Year-2+:** **full QF governance** — DAO-level allocation per Ethereum arch.
- Combine с participation Passport (Gitcoin) for Sybil-resistance.

### §1.6 First-event recommendation
**Hybrid mode:** sponsor cash + light QF matching pool. Sponsor commits €5K → Jetix sets aside €1.5K QF matching pool → community contributions matched. €15K total prize pool achievable from €5K sponsor cash.

## §2 Mechanism 2 — Grants (Gitcoin / Optimism RetroPGF / AI Grant)

### §2.1 Description
- **Gitcoin grants** — quarterly QF rounds; ~$1M-5M per round; open-source focus.
- **Optimism RetroPGF** — retroactive Public Goods Funding; ~$30M-100M per round; impact-based.
- **Protocol Guild** — long-term protocol developer funding.
- **AI Grant (Gross + Friedman)** — $250K SAFE + $350K Azure + $250K partner credits for AI startups; biannual cycles. [src: `research/hackathon-deep-2026-05-18/06-ai-grant-nfdg-deep-profile.md`]
- **EU Horizon** — large-scale research grants.

### §2.2 R12 fit
**High** (Gitcoin / Optimism RetroPGF). **Medium-high** AI Grant (SAFE = capital — minor R12 friction; но open-source mandate).

### §2.3 Pros / cons
- **+ Sustained funding** не tied to single event.
- **+ Validation signal** (grants vetted by community / experts).
- **+ Open-source-aligned** (RFC OSS mandate).
- **− Application time** (2-4 weeks per cycle).
- **− Competitive bandwidth** (acceptance rate <20%).
- **− Lag** (apply Q3 → funds Q4-Q1).

### §2.4 Sponsor profile fit
- **Grants = substitute for sponsor cohort partially.** Grant body acts as substrate sponsor.
- AI Grant target: «AI-native Workshop demo + hackathon platform» — fits Jetix narrative.

### §2.5 Jetix applicability
- **Q3 2026:** AI Grant Batch 5 application (per concept doc D); Workshop demo prerequisite.
- **Q4 2026:** Gitcoin grants round (Hackathon platform as public good).
- **2027:** Optimism RetroPGF (after impact track-record).

### §2.6 First-event recommendation
Apply **AI Grant Batch 5 in parallel** с first event execution. Если accepted Q3-Q4: funds Q1 2027 calendar.

## §3 Mechanism 3 — Corporate sponsorship

### §3.1 Description
Per Phase 1 §1.6, §3.1-3.5: corporate sponsor commits cash + API credits + employee mentor time для brand exposure + recruitment + IP access.

### §3.2 Variants
- **API credits sponsorship** — Anthropic / OpenAI / Cohere — €5K-50K equivalent + brand.
- **Cloud compute sponsorship** — AWS / GCP / Azure — €5K-100K equivalent.
- **Cash + recruitment** — RU corporate (Sber / Yandex / Tinkoff) — €10K-100K + hire pipeline.
- **Cash + brand** — EU corporate (Bosch / SAP / Siemens) — €20K-200K.
- **Multi-event annual sponsor tier** — €100K-500K annual commitment.

### §3.3 R12 fit
**Medium.** Corporate sponsors traditionally extract value via recruitment / IP / brand. R12 mitigation: contractual clauses (IP retention, salary cap on recruited talent, transparency of sponsor influence). Per concept doc A §10 R12 preservation.

### §3.4 Pros / cons
- **+ Familiar mechanism** (sponsor sells, sales template exists).
- **+ Quick funding** (B2B sales 2-8 weeks).
- **+ Strategic alignment** (recruitment pipeline).
- **− Sponsor extraction risk** (IP / talent claim) — R12 friction.
- **− Sponsor dependency** (single sponsor = single point of failure).
- **− Brand-coupling friction** (sponsor brand on event).

### §3.5 Sponsor profile fit
- **Tier 1 (LLM vendors):** Anthropic primary (FPF aligned + Workshop alignment); OpenAI secondary.
- **Tier 2 (Cloud):** AWS / GCP secondary.
- **Tier 3 (RU corporate):** Sber AI Journey / Tinkoff — high CAC but recruitment-driven; alignment uncertain.
- **Tier 4 (EU corporate):** Berlin tech stack ecosystems.
- **Tier 5 (Crypto):** Ethereum Foundation; Optimism.

### §3.6 Jetix applicability
- **Year-1:** primary funding mechanism (€5K-50K per event).
- **Year-2+:** secondary (QF + grants dominant); corporate retained for specific events.

### §3.7 First-event recommendation
**Anthropic** primary target — API credits + brand + cash. Quick-money P1 existing relationships если any. Berlin tech network secondary.

## §4 Mechanism 4 — Crowdfunding

### §4.1 Description
Indiegogo / Kickstarter / Manifund for hackathon prize pool. Hackathon participants OR sponsors OR audience contribute to specific event prize pool.

### §4.2 R12 fit
**High** (no extraction; participants contribute voluntarily).

### §4.3 Pros / cons
- **+ Community-aligned** (small contributions from many).
- **+ Pre-event marketing built-in** (crowdfunding campaign = pre-event funnel).
- **+ Validates demand** (если no funding → no demand signal).
- **− Limited scale** (rare crowdfunding > €50K for non-product events).
- **− Production overhead** (campaign video, rewards, etc.).
- **− Platform fees** (5-10%).

### §4.4 Sponsor profile fit
- Community-driven events.
- Mission-aligned audiences (open-source, civic-tech).
- Niche AI/crypto events where audience cares.

### §4.5 Jetix applicability
- **Year-1:** experimental (small €5K test campaign as side-channel).
- **Year-2+:** combined с QF (Manifund's QF-aligned crowdfunding).

### §4.6 First-event recommendation
**Skip Year-1 first event.** Add Year-2 if community size sufficient. Manifund (https://manifund.org) as future direction.

## §5 Mechanism 5 — Token launch (deferred Phase 2+)

### §5.1 Description
Project-native token (Jetix DAO governance token) sold via ICO / fair launch / liquidity mining to fund operations + reward early participants.

### §5.2 R12 fit
**Medium-high IF executed per Mondragón / QF principles** (per `swarm/awaiting-approval/r12-programmable-ethereum-2026-05-18.md` Option D Hybrid acked); **Low if pure VC-influenced ICO**.

### §5.3 Pros / cons
- **+ Largest funding potential** ($1M-100M+ historically).
- **+ Long-term aligned** (token holders = stakeholders).
- **+ Direct R12 enforcement programmable.**
- **− Regulatory risk** (US securities / EU MiCA framework).
- **− R12 violation risk** if executed naively.
- **− Cohort distortion** (token speculators ≠ value-creating cohort).

### §5.4 Sponsor profile fit

- Aligned with Ethereum arch acked.

### §5.5 Jetix applicability

### §5.6 First-event recommendation

## §6 Mechanism 6 — Hybrid models (best-of-mechanisms)

### §6.1 Description
Combine 2-4 mechanisms per event to optimise tradeoffs.

### §6.2 Canonical hybrid for Jetix Year-1 first event

- Sponsor (Anthropic) commits €5K credits + €5K cash → €10K base.
- Jetix QF matching pool €2K → ~€6K matched contributions (1.5x typical multiplier).
- Total prize pool: €16K from €5K cash.
- IP: open-source mandatory; participants retain rights.
- R12 check: ✓ (sponsor cannot claim participant IP; QF distributes equitably).

### §6.3 Hybrid for Year-2 events
**Grant + corporate annual sponsor + QF + crowdfunding:**
- AI Grant SAFE $250K (Year-1 acceptance).
- 3 corporate annual sponsors × €50K = €150K.
- QF matching pool €50K × 4 events = €200K matched.
- Crowdfunding pre-event campaigns €5K × 4 = €20K.
- Total Year-2 funding: ~€420K + grant equity.

### §6.4 Hybrid for Year-3+
**Token + grants + QF dominant; corporate secondary; crowdfunding niche.**
