# Skill: /search-kb

## Описание
Поиск информации в Knowledge Base.

## Триггер
`/search-kb {query}` — поисковый запрос.

## Алгоритм

### 1. Проверить MOC (быстрый путь)
Прочитать `knowledge-base/_index.md` → найти релевантный кластер → прочитать `_moc.md` кластера.
Если в MOC есть подходящая статья — прочитать её и ответить.

### 2. Grep по frontmatter tags
```bash
grep -rl "tags:.*{keyword}" knowledge-base/
```
Прочитать найденные файлы.

### 3. Full-text grep
```bash
grep -rl "{query}" knowledge-base/ raw/
```
Если в KB нет — искать в raw/ (может быть ещё не скомпилировано).

### 4. Ответить
- Показать найденную информацию с указанием источника
- Если ничего не найдено — честно сказать "Не найдено в KB"
- Если найдено только в raw — предложить `/compile` для создания KB-статьи

## Правила
- Порядок поиска строгий: MOC → tags → full-text
- НЕ выдумывать информацию, если не найдена
- Указывать из какого файла информация

## Оценка токенов
Чтение: ~5,000 | Генерация: ~1,000 | Итого: ~6,000 (0.6% от 1M)
