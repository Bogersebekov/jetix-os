# /close-day — Вечернее закрытие дня

## Шаг 1: Прочитать сегодняшний daily-log
Найди файл daily-log/YYYY-MM-DD.md (сегодняшняя дата). Если нет — сообщи.

## Шаг 2: Спросить итоги
Спроси у меня:
1. Что сделал сегодня? (кратко)
2. Какие инсайты/решения?
3. Уровень энергии (1-5)?
4. Что на завтра?

## Шаг 3: Обновить daily-log
Заполни секции в сегодняшнем daily-log:
- "Что сделано" — из ответов
- "Инсайты" — если есть
- "На завтра" — план
- "Энергия" — уровень

## Шаг 4: Обновить проекты
Для каждого проекта, который был затронут сегодня:
1. Добавь запись в projects/{project}/log.md: `[YYYY-MM-DD] — Что сделано → Где остановились → Что дальше`
2. Обнови next_action в projects/{project}/overview.md frontmatter

## Шаг 5: Обновить Notion (если нужно)
Если были существенные результаты — обнови лог работы на соответствующей странице проекта в Notion через MCP.

## Шаг 6: Git commit + push
```bash
git add -A
git commit -m "daily: YYYY-MM-DD — [краткое описание что сделано]"
git push
```

## Правила
- Не коммить без подтверждения итогов от пользователя
- Будь кратким в записях логов
- Если проект не затронут — не трогай его лог
