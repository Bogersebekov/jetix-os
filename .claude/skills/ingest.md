# Skill: /ingest

## Описание
Извлечь ключевые факты из raw-источника. Обновить frontmatter, добавить секцию Extracted Facts.

## Триггер
`/ingest {filepath}` — путь к файлу в `raw/`.

## Алгоритм

### 1. Прочитать raw-файл
Открыть указанный файл. Проверить что `pipeline: raw` или `pipeline: ingested` (для повторного инжеста).

### 2. Извлечь факты
Прочитать оригинальный текст и извлечь:
- Конкретные данные (цифры, даты, имена)
- Ключевые утверждения
- Цитаты (с указанием автора)
- Противоречия с существующими KB-статьями (если замечены)

### 3. Обновить файл
**НЕ менять оригинальный текст!** Только:

a) Обновить frontmatter:
```yaml
pipeline: ingested
processed: true
```

b) Добавить в конец файла (после оригинального текста):
```markdown
---
## Extracted Facts
<!-- Добавлено /ingest YYYY-MM-DD -->

### Данные
- Факт 1
- Факт 2

### Ключевые утверждения
- Утверждение 1

### Противоречия
<!-- [ПРОТИВОРЕЧИЕ] с KB-файлом, если обнаружено -->

### Теги для компиляции
- topic/subtopic
```

### 4. Подтвердить
Показать Руслану извлечённые факты. После подтверждения — сохранить.

## Правила
- Оригинальный текст НЕПРИКОСНОВЕНЕН
- Extracted Facts добавляются ТОЛЬКО в конец, после разделителя `---`
- Если файл уже ingested — перезаписать секцию Extracted Facts

## Оценка токенов
Чтение: ~3,500 | Генерация: ~1,500 | Итого: ~5,000 (0.5% от 1M)
