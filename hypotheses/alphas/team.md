# Alpha: Team

## Purpose
Group performing работу — the people/agents executing the hypothesis test.

## State graph
```
seeded → formed → collaborating → performing → adjourned
```

## State definitions

| State | Meaning |
|---|---|
| **seeded** | Initial members identified; not yet organized |
| **formed** | Members onboarded; roles clear |
| **collaborating** | Working together; communication established |
| **performing** | Optimal performance; team effective |
| **adjourned** | Team disbanded или transitions к new work |

## Jetix application

Для hypothesis substrate, team includes:
- **Ruslan (solo)** — many hypotheses are personal; team = {ruslan}
- **Ruslan + brigadier** — autonomous scaffolding с ack gates
- **Ruslan + ROY agents** — domain expert dispatch (mgmt-expert / systems-expert / etc.)
- **Ruslan + partner** — collaborative hypothesis (Левенчук cross-pollination)
- **Founding Circle** — future multi-party hypotheses
