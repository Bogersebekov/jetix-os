# Рабочие процессы Jetix OS

## Утренний ритуал (перед рабочими сессиями)

1. **Открыть Claude Code** в jetix-os
2. **Прочитать вчерашний лог** — что было сделано, что осталось
3. **Определить фокус дня** — максимум 3 задачи, привязанные к проектам
4. **Создать daily log** — зафиксировать план дня
5. **Начать первую рабочую сессию**

## Рабочая сессия (по проекту)

1. Прочитать `projects/XX/CONTEXT.md` — вспомнить контекст
2. Работать над задачей
3. Обновить CONTEXT.md — статус, следующие шаги, решения
4. Записать результат в daily log

## Вечерний ритуал (закрытие дня)

1. **Обновить daily log** — что сделано, что не сделано и почему
2. **Обновить CONTEXT.md** затронутых проектов
3. **Зафиксировать решения** в decisions.md (если были стратегические)
4. **Определить приоритет на завтра** — записать в лог
5. **Коммит и пуш** изменений в git

## Недельный обзор (воскресенье вечер)

1. Прочитать логи за неделю
2. Оценить прогресс по каждому проекту
3. Определить фокус следующей недели
4. Обновить CONTEXT.md всех проектов
5. Записать weekly review
