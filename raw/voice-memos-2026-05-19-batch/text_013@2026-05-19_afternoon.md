## §2 Core additions surface'нутые

### Дополнения к K-6 components (поверх text_012)

11. **System self-knowledge** — должна знать и понимать себя хорошо / как работает самостоятельно
12. **Awareness of influences** — что хорошо влияет / что плохо влияет (positive / negative discrimination)
13. **Active selection** — должна выбирать что положительно влияет
14. **Constant improvement drive** — хочется развиваться → больше / крупнее / стабильней / проворней
15. **Meta + concrete mixed** — meta-level описать + конкретные примеры на понятном уровне
16. **Gradient view** — никакая система 100% perfect нигде; **части можно rassматривать и улучшать**
17. **Break into parts** — система потребления информации разбивается на части → улучшать parts → composite improvement общей system
18. **Recursive lifecycle** (8-step):
    1. Создали (описали) метод
    2. По методу описали план улучшить себя
    3. По методу описали план улучшить planet
    4. Воплотили план в жизнь
    5. Новые questions / hypotheses появляются
    6. По методу обработали new hypotheses
    7. Continue cycle
    8. System развивается в хорошем ключе
