## §2 NEW components surface'нутые (поверх text_012 + text_013)

### Information consumption focus

19. **System consumes info about** (3 categories):
    - How own system works
    - How to interact with other systems
    - Rules of how other systems work

20. **Rule-knowledge dependency** — survival/success зависит from how well system knows rules of other systems + how to interact

21. **Intellect quality → rule knowledge quality** — better intellect (= method of info processing + computational capability) → better rule knowledge → better development/life

### 3-stage throughput model

22. **Stage 1 — Input throughput** — количество информации которое может попасть внутрь
23. **Stage 2 — Processing** — метод + качество + скорость + вычислительные способности
24. **Stage 3 — Output use** — качество использования; влияет **memory** (насколько система может помнить уже занесённую info + использовать её)

### Cyclic / process flow

25. **Post-output cycle** — система обучается / тренируется / работает / улучшается → возвращается в **«разведку»** (reconnaissance)
26. **Reconnaissance phase** — смотрит новую info + определяет какую надо потребить → consume → cycle
27. **NOT just circle — PROCESS** — continuous, infinite («это как будто даже не круг это вот процесс»)
28. **Per-iteration selection** — выбираются:
    - Some values
    - Some method of info processing
    - Specific information

### Jetix Workshop = Exokortex (КРИТИЧНО)

29. **⭐ Jetix Workshop = exokortex (экзокортекс)** — embodied method of info processing
30. **Exokortex enables** (4 capabilities):
    - AI amplification + acceleration self
    - Same-language interaction (FPF)
    - Higher capability interaction между bigger systems
    - Quality / speed / efficiency improvement

### System production output

31. **System produces** (3 types):
    - New methods of info processing
    - Improves existing methods
    - Creates / transforms new methods

32. **Full circle:** consume info + methods → output info + methods (continuous transformation)
