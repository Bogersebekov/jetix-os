1. **Society-as-code metaphor**: общество = программный код, разработанный многими разрабами через миллионы commits — миллионы фич + миллионы багов
2. **«Долго писали, охуенные фичи добавили»** — обществу присуща ценность накопленного opus magnum (NOT anti-civilization frame)
3. **«Миллионы багов»** — критическая дисфункция концептуально равная features (без anti-civilization);
   - Государства не могут сотрудничать
   - Растрата ресурсов на «залупу» (misalignment vs alignment)
   - Города/люди internal misalignment
   - Фрирайдеры / читеры
4. **Scale-of-alignment hierarchy**: 10 → 100 → 1000 → state-myth-scale (80M Germany через мифы)

6. **Jetix mission positioning**: «мы идем исправлять накопившиеся баги — самые critical для системы»
7. **«Время для чистки/революции/перестройки»**: tools available + political tension high = window открыт
8. **Intellect-as-debugger** философское positioning: «интеллект помогает дебажить жёбаную жизнь»
9. **Mass-call action**: Jetix приглашает других людей тоже debugging life заниматься («мы идем за вами, гниды, баги порешаем»)
10. **Бытовые баги** (личный уровень): «тикток зависимые, смысла не видят» = тоже bugs для исправления
11. **Cascade smartness pattern**: умнейшие → менее умнейшие поумнели → все человечество прокачали
