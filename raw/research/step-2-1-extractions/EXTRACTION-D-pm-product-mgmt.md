# 1. Framework distillations

## Cagan / SVPG — Empowered Product Teams
- **Def:** Outcome-owning teams; continuous discovery before delivery; **four risks** (value, usability, feasibility, viability) mitigated via prototypes + assumption tests, not PRDs.
- **Source:** RESULT-06 §A (*Inspired* 2008/2017, *Empowered* 2020, *Transformed* 2024).
- **Swarm:** Four-risks = default critic rubric. Cagan 2025 SVPG (teams 8→3, delivery near-instant, discovery dominates) validates 5×4 matrix: exec cost→0 → judgment density binds.

## Torres — Continuous Discovery / OST
- **Def:** Weekly interviews → **OST** (outcome → opportunity → ≥3 solutions → assumption tests). Importance×evidence 2×2. Story-based backward interviews.
- **Source:** RESULT-06 §B (*Continuous Discovery Habits* 2021).
- **Swarm:** OST = canonical goal-hierarchy artifact. Compare-3-solutions = parallel-hypothesis discipline. Torres 2025–26 warning: AI-OST from synthetic personas = "hypothesis generation masquerading as validation" → critic grounds on real traces.

## JTBD (Christensen / Ulwick / Moesta / Klement)
- **Def:** Customers "hire" products for functional+emotional+social progress. Ulwick **ODI**: outcome = direction+metric+object; opportunity algorithm = `Importance + max(Importance − Satisfaction, 0)`. Moesta **Switch Interview four forces**: Push / Pull / Anxiety / Habit.
- **Source:** RESULT-06 §C (Christensen HBR 2005/2016; Ulwick Strategyn Dec 2025).
- **Swarm:** Framework-agnostic lens. Four-forces = adoption-resistance diagnostic (Anxiety=hallucination fear, Habit=workflow inertia). Ulwick Dec 2025: outcomes *before* AI application.

## Lean Startup (Ries / Blank / Osterwalder)
- **Def:** BML; MVP as learning experiment; innovation accounting (baseline→tune→pivot/persevere); 10-pivot taxonomy; Blank 4-step Customer Development; Osterwalder BMC.
- **Source:** RESULT-06 §D (Ries 2011; Blank 2005; Osterwalder 2010). **No 2025–26 Ries primary — gap.**
- **Swarm:** Validated-learning core survives; "minimum" of MVP obsolete at 2hr prototype. BML = Review-phase frame.

## OKRs / North Star (Grove / Doerr / Wodtke / Ellis / Amplitude)
- **Def:** Grove Intel 1975 MBO; Wodtke *Radical Focus*: single-O per quarter + Monday/Friday ritual; Amplitude NSM tree.
- **Source:** RESULT-06 §E; Linear zero-OKR counter (Saarinen Lenny Sep 2023).
- **Swarm:** NSM compatible; Wodtke single-O = only classical OKR surviving swarm cadence. Linear zero-OKR ($1.25B @ 100 ppl) = live alternative.

## Shape Up (37signals / Singer / DHH)
- **Def:** Fixed time, variable scope. 6-wk build + 2-wk cooldown. **Appetite** replaces estimates. Betting table replaces backlog. Shaping (fat-marker sketches, rabbit-holes). Hill charts. Circuit breaker at cycle end.
- **Source:** RESULT-05 §A (Singer 2019; **DHH Pragmatic Engineer Apr 2026 — 6-wk cycle "needs rewriting"**; Fried Jan 2025 — 2nd ed unpublished).
- **Swarm:** Swarm outer cycle IS structurally Shape Up. Hill chart = novelty axis (LLMs hallucinate uphill / execute downhill). AI-compressed: 2–4 wk at Jetix scale (no primary specifies).

## Scrum / Kanban / XP
- **Source:** RESULT-05 §B (Scrum Guide Expansion Pack Jun 2025 "AI as team member" rejected; Beck Pragmatic Engineer Jun 2025; Anderson LinkedIn Mar 2026).
- **Swarm:** Scrum *principles* survive; story points/velocity/ceremonies break under AI compression. Kanban flow + **WIP-per-agent** (Anderson Mar 2026) = scheduling primitive. **Beck: "TDD is a superpower with agents" + "agents delete tests to make them pass"** → TDD = mandatory guardrail.

## TOC / Lean Software (Goldratt / Poppendieck / Kim / Kersten)
- **Def:** Goldratt 5 focusing steps; Poppendieck 7 principles; Kim Three Ways; Kersten Flow Framework + *Output to Outcome* 2026: "cost of outputs collapses to zero; operating model becomes the bottleneck."
- **Source:** RESULT-05 §D; DORA 2024/2025.
- **Swarm:** Most analytically powerful lens for swarm pathologies. **Constraint migrated from execution to critical systemic judgment.** DORA 2025 #1 amplifier: small batches.

## Grove — High Output Management / Only the Paranoid Survive
- **Def:** `manager output = org output + neighboring-org output under influence`. **TRM**: task-specific, 3 postures (structured low / supportive med / objective-only high). **1-on-1 as subordinate's meeting.** Meetings taxonomy: process-oriented default vs. mission-oriented exception. **Paired indicators.** **SIP**: 10× force. Constructive confrontation.
- **Source:** RESULT-07 §A (Grove 1983/1995/1996).
- **Swarm:** TRM = primary brigadier↔expert delegation primitive. Output equation: `direct + network-under-influence + agent-under-direction`. 1-on-1 → brigadier↔expert async check-in. SIP = watchtower. Paired indicators = self-check against reward-hacking.

## Drucker — Effective Executive / Managing Oneself
- **Def:** Effectiveness > efficiency. "What needs to be done?" **Feedback analysis**: write expected outcome → compare at 9–12 mo (sole reliable strengths-discovery). Build on strengths. Five Drucker questions.
- **Source:** RESULT-07 §E; Drucker Forum 2024 "The Next Knowledge Work."
- **Swarm:** Feedback analysis = swarm self-audit. Every decision logs expected outcome → compound ledger. Strengths-based role-mode: critic doesn't optimize; optimizer doesn't discover.

## Watkins — First 90 Days / STARS / Six Disciplines
- **Def:** Break-even point. **STARS**: Startup/Turnaround/Accelerated-growth/Realignment/Sustaining-success — prescriptions differ. First 30 days = listen. Six Disciplines (2024): Pattern Recognition / Systems / Mental Agility / Structured Problem-Solving / Visioning / Political Savvy. RPM cycle.
- **Source:** RESULT-07 §F (2003/2013/2024; Big Think Apr 2025).
- **Swarm:** STARS recursive per new task-class: novel domain = startup-STARS → structured prompt + monitoring (low-TRM). **3 AI failure modes (hallucination / programmed personality / optimistic bias)** = mandatory critic checks.

## Horowitz — Hard Thing / What You Do Is Who You Are
- **Def:** **Wartime vs. peacetime CEO** — canon is peacetime-calibrated. **Culture is behaviors, not beliefs** ("When people say they have 20 values, I ask how many behaviors. If none, culture is hypocrisy"). Shocking rules. "Don't eat shit nibbling." **Decision debt** (Sequoia Feb 2026): "worst debt, paralyzes a company." CEO designs communication architecture.
- **Source:** RESULT-07 §G.
- **Swarm:** SIPs trigger wartime mode. Culture lives in prompts/rubrics/incentive contracts. Decision-debt = swarm KPI.

## Laloux — Teal / Reinventing Organizations
- **Def:** **Self-management** (roles hold authority; Morning Star **CLOU** = peer-written contract replacing job descriptions; **advice process** = decision-holder consults affected + expert). **Buurtzorg**: 15K nurses, teams of 12, 50-person HQ, 30% higher patient satisfaction.
- **Source:** RESULT-07 §H (Laloux 2014/2018; HBR Bernstein & Bunch 2016; Quartz Zappos retreat).
- **Swarm:** Matchmaker + agent swarm IS structurally teal. **Advice process** = swarm decision primitive for intermediate authority. CLOU = expert-contract schema. Buurtzorg 50-HQ-for-15K = Jetix topology. **Caveat:** collapses without high-agency individuals + real purpose.

## Netflix — Hastings / McCord
- **Def:** (1) **Talent density**; (2) **Keeper Test** + generous severance; (3) **Context not Control** / informed captain; (4) freedom + responsibility. Highly aligned, loosely coupled. **Farming for dissent**. Top-of-personal-market pay.
- **Source:** RESULT-07 §C (culture memo 2009/2024/2025; *No Rules Rules* 2020).
- **Swarm:** Context-not-control = brigadier↔expert primitive. Farming for dissent = explicit critic invocation. Keeper test at partnership level (annual "rebuild-around?"), not termination.

## 37signals / DHH (beyond Shape Up)
- **Source:** RESULT-07 §B. Calm company; async-default; agent-first CLI (2026). **2021 Basecamp controversy** (Platformer) = canonical values-on-walls anti-pattern.
- **Swarm:** Async-default = comm mode.

## Stripe — Collisons
- **Def:** 6 Operating Principles (Users first / Craft & beauty / Urgency & focus / Egoless collab / Obsess over talent / Stay curious). **Writing as default** (footnoted CEO emails, no slide decks, pre-meeting memos, RFCs). "Disagree and commit." March 2026: Claude Opus 4.5 at 92% on Stripe integration benchmark.
- **Source:** RESULT-07 §D.
- **Swarm:** Writing-as-craft = coordination substrate. Benchmark validates: good docs → good agent output.

## Holdco Doctrine (Buffett / Munger / Leonard / Wilkinson)
- **Def:** **Decentralized ops + centralized capital allocation**. Permanent capital. **Tiered hurdle rates** (Constellation: 30% IRR <$1M). Operator quality = binding constraint. Annual letter. **Two-year letter 4 rules:** report bad news early / guard reputation / confer on large moves / recommend successor.
- **Source:** RESULT-07 §J.
- **Swarm:** Brigadier = capital allocator (cycle budget per expert); experts = operators (autonomous). **Two-year letter 4 rules = role-mode charter default. Hurdle rate = confidence threshold below which swarm refuses.**

## Solopreneur / Leverage (Ravikant / Graham / Sivers)
- **Def:** Ravikant leverage: labor / capital / **code / media** (permissionless). Graham *Founder Mode* (2024). Graham *Writes and Write-Nots* (2024): "writing IS thinking." Sivers *Hell Yeah or No*.
- **Source:** RESULT-07 §I.
- **Swarm:** Agents = fifth permissionless leverage. Wiki = where thinking happens.

## Escalation / HITL triggers

(a) SIP-class signal (Grove 10×) → wartime escalation; (b) Cagan value/viability risk above critic confidence-bound; (c) Leonard hurdle-rate miss → refuse, don't escalate; (d) Horowitz decision-debt alarm → consolidated escalation; (e) Watkins 3 AI failure signals detected → escalate with traces; (f) Grove paired-indicator divergence → human review. ## Discovery vs. Delivery

- Cagan empowered-team ↔ role-mode autonomy (problems not roadmaps)
- Torres continuous discovery ↔ critic + integrator modes

# 4. Cycle timing / escalation / delegation primitives

**Shape Up outer × CE inner (RESULT-05 D2 synthesis):** CE = inner loop inside Shape Up build-phase. Shape Up = 2–4 wk outer (AI-compressed from 6 wk per DHH Apr 2026); CE = per-feature Plan→Work→Review→Compound. Cooldown ≈ human episodic aggregation of per-loop compound.

**1-on-1 cadence (Grove):** solo → monthly partner 1-on-1s, partner-owned agenda (RESULT-07 D3). Swarm: brigadier↔expert check-ins at compound-cycle boundaries, expert agenda.

**TRM thresholds:** low = structured + decomposition + monitoring; med = objective + constraints + examples; high = objective + success criteria. **Per-task-class.** Literature thin (Software Philosopher Dec 2025 earliest) → **open contribution** per RESULT-07 D7 §3.

**OKR hierarchy:** Jetix scale = Wodtke single-O/quarter; NSM replaces cascading KRs; compound ledger surfaces "what moved NSM this cycle" (Drucker feedback-analysis at org scale).

**Decision authority by hurdle rate (Leonard):** autonomous above X, escalate between X–Y, refuse below Y. Avoids ask-the-human-every-time loop.

**Two-year letter (Buffett) 4 rules = full HITL contract:** report bad news immediately / guard reputation / confer before major moves / recommend successor. Per role-mode charter.
