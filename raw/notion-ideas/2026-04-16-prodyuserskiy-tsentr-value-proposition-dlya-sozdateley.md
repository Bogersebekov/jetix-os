# Продюсерский центр — value proposition для создателей контента

## Идея (title-поле Notion)

Продюсерский центр — value proposition для создателей контента

## Суть (content_preview Notion)

Конкретное ценностное предложение продюсерского центра для контент-мейкеров: больше внимания и энергии на творчество, аутсорс рутины (ведение соцсетей, монтаж, монетизация, продажи), доступ к коллаборациям с другими создателями, больше времени на ключевое действие — создание контента. По сути, центр забирает всё кроме творчества.
