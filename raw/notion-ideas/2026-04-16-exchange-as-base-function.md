# Обмен как базовая функция: взаимодействие → видение → условия → сделка

## Суть
Обмен — базовая функция всех взаимодействий. Чтобы обмен произошёл: 1) взаимодействие/видение друг друга, 2) понимание условий. Можно менять видение на продукт (стартапер показывает презентацию — получает деньги). Обсудить глубже с экономистами и философами.

## Действие
Обсудить с экономистом Владиславом. Связать с установками про ценность и деньги.
