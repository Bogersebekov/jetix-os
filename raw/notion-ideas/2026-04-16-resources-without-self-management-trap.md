# Ресурсы без управления собой = ловушка

## Суть
Если есть ресурсы, но не можешь управлять собой (телом, вниманием, отношениями) — ресурсы бесполезны. Сам себя в клетку сажаешь. Идея для бизнеса: помогать людям с деньгами, которые не могут управлять вниманием, телом, отношениями. Чтобы быть хорошим управленцем — сначала управляй собой.

## Действие
Использовать для позиционирования: я помогаю людям с ресурсами использовать их на максимум. Связать с формулой звериного состояния.

## Идея (полный текст)
Ресурсы без управления собой = ловушка. Клиенты с деньгами и ментальными клетками
