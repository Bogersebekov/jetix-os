# Инструкция-маршрутизатор для состояний

## Идея (title-поле Notion)

Инструкция-маршрутизатор для состояний

## Суть (content_preview Notion)

Создать набор инструкций вида «если попадаю в такое-то состояние / обстановку — делаю то-то, не делаю то-то». Это снимает нагрузку с принятия решений в моменте: вместо того чтобы «обезьяна» выбирала из миллиона вариантов, она следует обдуманному маршруту. Вероятность успешного дня резко растёт.

## Действие

Описать 5–7 типовых состояний/обстановок и для каждого — конкретные действия и запреты.
