# «Бить твёрдым по пустому» — pre-interaction intelligence doctrine

## Одна строка

Перед любым стратегическим взаимодействием — максимальная разведка, просчёт исходов, выбор момента. Идти туда где пусто, не в лоб. Универсально, повышает вероятность успеха ~100%.

## Контекст

- Surface'нуто из voice batch 2026-05-13/14 (audio_653@13-05_20-15-04, §1.2-1.5 Принципы + Инсайты + Стратегические гипотезы).
- Stage 5: NO Tier A match → NOVEL principle candidate.
- Application: продажи / найм / партнёрство / убеждение / outreach / negotiations.
- Ruslan ack 2026-05-14: «тоже блять это фиксируем» — pool item для Tier 1 / Tier 2 RUSLAN-LAYER consideration.

## Mechanics

1. **Phase 1 — Intelligence:** собрать всё про человека/ситуацию (public + network + voice + behavioural)
2. **Phase 2 — Simulation:** просчёт ответов / реакций / альтернативных исходов
3. **Phase 3 — Key actions only:** выбор минимального числа действий с максимальным leverage
4. **Phase 4 — Timing:** идти когда «пусто» (нет конкуренции внимания / нет защитной позы у counterpart)

## Где использовать

- **Tier 1 RUSLAN-LAYER manager principle** (self-discipline): применять к каждой L1 outreach (Цэрэн, Левенчук, ...).
- **Tier 2 RUSLAN-LAYER instance** (если для agents в swarm): autoresearch-brigadier + sales-lead default doctrine.
- **Video Цэрэну §3:30-5:00** (Наработки): мягкая formulation — «один из принципов на которых строится — глубокая предварительная подготовка, просчёт исходов, выбор момента. Это видео — пример этого же принципа.»

## Framing risk

- Прямая phrase «бить твёрдым по пустому» = Жуков / Сунь Цзы — звучит militant; для Цэрэна (методолог-инженер) smooth into «strategic reconnaissance» / «engineering posture» / «не атаковать сильное».
- В RUSLAN-LAYER можно сохранять direct phrasing (private context).
- В outreach — soften to «pre-interaction intelligence» / «глубокая подготовка перед встречей».
