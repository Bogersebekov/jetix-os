# Dota Skill Ceiling → Realm Class Mastery

## Суть
Lift Dota's perpetual mastery dimensionality (1000s of hours, never «done») на Realm E2 class progression — anti-plateau design.

## Mapping table
| Dota 2 | Realm E2 Class |
|---|---|
| 130 heroes (deep specialization options) | Sub-class trees (deep depth) |
| MMR + visible rank | Class tier + visible metric |
| Skill ceiling 10K+ hours | Mastery space measured years not months |
| Replays + analysis tools | Quest retrospectives + peer review |
| Esports legends (Faker, Miracle-) | Realm Legends (E1 Great People system) |

## Применение
- Each class designed для multi-year mastery curve
- Multiple skill dimensions per class (technical + strategic + interpersonal)
- Visible mastery metric (separate from level)
- Tools for self-improvement (retro + peer review)
- Visible legends per class

## Что подтверждает
- Dota retention 13+ years primarily through skill ceiling
- Stark contrast с casual games (Candy Crush plateau ~Month 3)
