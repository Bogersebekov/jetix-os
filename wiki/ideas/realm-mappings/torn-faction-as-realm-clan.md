# Torn Faction → Realm Clan (applied pattern)

## Суть
Direct lift Torn Faction mechanics (Respect / Armoury / Organized Crimes / Wars / Tree) на Realm E3 Clan structure — substrate validated by 20-year operating game с 50K stable community.

## Mapping table
| Torn | Realm E3 |
|---|---|
| Faction Respect | Командная Reputation |
| Faction Armoury | Общий пул (AI tools, templates, contacts) |
| Organized Crimes (80-95% split) | Organized Projects (revenue share 80-90%) |
| Faction Wars | Соревновательные тендеры |
| Faction Tree | Дерево прокачки команды |
| Crime Skill | Sub-class specialization |

## Применение
1. Команда формируется (3-10 чел) — Torn Faction size band identical
2. Pre-defined roles за Organized Project (driver/hacker/etc → researcher/closer/designer)
3. Reputation accumulates коллективно
4. Wars = competitive deal bidding между двумя кланами
5. Tree = долгосрочное развитие команды

## Что подтверждает
- Strategic Insight §4.2 explicit precedent (Ruslan's voice item audio_631)
- Torn 20-year operating без серьёзных balance crashes
- Community-sourced stability (no whaling, no pay-to-win primary)
