# EVE Economy → Realm Resources (applied pattern)

## Суть
Lift EVE's player-driven economy + MER transparency + PLEX bridge pattern as Realm E5 (Resources) substrate.

## Mapping table
| EVE Online | Realm E5 |
|---|---|
| ISK currency | Knowledge tokens |
| PLEX bridge ($→ISK) | Premium credits (real $ → Knowledge) |
| Monthly Economic Report | Quarterly Realm MER (transparency) |
| Mining → refining → manufacturing chains | Multi-stage quest decomposition |
| Player-set market prices | Clan-internal marketplace |
| Sovereignty / corp space | Niche specialization ownership |

## Применение
- E5 Knowledge tokens + AI credits + Focus tokens + Audience reach (4 ресурсов)
- Quarterly transparency report (Realm MER)
- Player-driven marketplace pricing
- Premium-to-earned bridge (no whaling)

## Что подтверждает
- 20+ year EVE economy stable
- Real economist (Guðmundsson) validated approach
- Lehdonvirta-Castronova Virtual Economies (T1 book) — EVE как case study
