# Декомпозиция жизни на подпроекты

## Одна строка

Жизнь — один большой проект, но менеджерить его целиком невозможно, как невозможно изучать все науки одновременно.

## Содержание (из Notion)

Жизнь — один большой проект, но менеджерить его целиком невозможно, как невозможно изучать все науки одновременно. Решение — разбить на подпроекты (спорт, учёба, бизнес и т.д.), для каждого описать цели и задачи. Тогда можно оценивать и улучшать каждое направление по отдельности, а не тонуть в общем ощущении хаоса.

## Действие

Создать документ «Подпроекты жизни»: список направлений, цели, задачи, критерии для каждого.
