# Цепочка «результат → метод → аудитория» как бизнес-модель

## Одна строка

Сначала добиться значимых результатов с помощью системы управления ресурсами (дневники, отчёты, фокус), а затем описать сам метод — и на это.

## Содержание (из Notion)

Сначала добиться значимых результатов с помощью системы управления ресурсами (дневники, отчёты, фокус), а затем описать сам метод — и на этом построить контент, аудиторию и новые проекты. Метод становится продуктом, а результаты — доказательством его работоспособности.

## Действие

Документировать свой метод параллельно с его применением
