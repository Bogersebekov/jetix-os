# Jetix as Standards Body / «фюрершайн» — H8 candidate pool

## Одна строка

Jetix эволюционирует Workshop (substrate) → People-NS (coordination) → **Standards Body** (governance protocol) для управления сложными AI/data/decision системами. Как ПДД / ISO / ICAO — единые правила поверх национальных границ, но substrate = mastership reputation + Realm progression, не bureaucratic accreditation.

## Draft thesis (3 lines, не LOCKED)

1. Jetix evolves: Workshop (Phase 1, substrate) → People-NS (Phase 2-3, coordination protocol) → **Standards Body** (Phase 4-6, governance protocol).
2. Precedent: ISO / ICAO / FAA / ПДД — координирующие protocols над national bureaucracies; Jetix differs в том что substrate = reputation+Realm progression, не paper accreditation.
3. L4-L6 trajectory: после L3 federation (≥1K members) Jetix publishes **Jetix Standards** — open-source frameworks для управления AI/data/decision systems. L0-L6 certification становится recognized credential поверх corporations.

## R12 anti-extraction check (NEEDED before promotion)

- **Question:** что Jetix takes от cert holders? Per-certification fee? Realm percentage? Equity?
- **Risk:** если model = «pay Jetix to be certified» → extraction → нарушает Tier 2 R12 anti-extraction (no extraction beyond agreed share).
- **Anti-pattern:** ICANN-like rent-extraction; bureaucratic gatekeeper.
- **Pro-pattern:** sertifikat как side-effect of L-progression; revenue source = L0-L6 progression services (deepening / mentoring / Realm participation), не cert itself.
