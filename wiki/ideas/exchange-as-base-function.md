# Обмен как базовая функция: взаимодействие → видение → условия → сделка

## Одна строка

Обмен = базовая функция всех взаимодействий: 1) увидеть друг друга, 2) понять условия, 3) совершить сделку.

## Что можно обменивать

- **Видение на продукт** (стартапер показывает презентацию → получает инвестиции).
- **Время на деньги** (классический работник).
- **Внимание на знания** (студент → университет).
- **Доступ на лояльность** (community membership).
- **Репутацию на доверие** (бренд → premium price).
