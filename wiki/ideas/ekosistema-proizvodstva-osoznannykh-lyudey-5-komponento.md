# Экосистема производства осознанных людей — 5 компонентов

## Одна строка

Видение масштабного проекта из пяти компонентов: (1) идеология и система ценностей, (2) онлайн-продукт по развитию мышления, (3) оффлайн-шко.

## Содержание (из Notion)

Видение масштабного проекта из пяти компонентов: (1) идеология и система ценностей, (2) онлайн-продукт по развитию мышления, (3) оффлайн-школа с «армейской» подготовкой и пространством для жизни, (4) программа разработки стратегии жизни с помощью продвинутых специалистов, (5) инвестфонд в проекты и в людей. Все компоненты работают на одну цель — производство осознанных людей.

## Действие

Детализировать каждый компонент экосистемы отдельно
