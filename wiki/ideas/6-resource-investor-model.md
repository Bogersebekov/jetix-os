# 6-Resource Investor Model

## §1 Core claim

Investor recruiting должен покрывать **6 resource categories** (NOT money-only). 5 categories explicit:
1. **Lawyers / law** — «дальше уже разберутся юристы пусть и башен там эти законы»
2. **Financiers / capital** — «инвесторы пусть там инвестируют короче финансисты что-то думают»
3. **Crypto / security** — «криптовалютчики как там это обезопасить»
4. **AI / mechanism** — «искусственный интеллект как это вот снова таки все работать»
5. **Philosophy / ethics** — «философ еще какой-то по еботу короче пусть там все как-то что-то думают»
6. **(TBD)** — implicit 6th category; voice анкер «по всем шести ресурсам» without enumeration

Each category investor provides their resource as «бесконечный хавчик + вера в себя + обстановка + инструменты» → 1000x amplification thesis.
