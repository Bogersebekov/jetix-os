# Шкала влияния 0-100% на день

## Одна строка

Измеримая ежедневная метрика контроля над днём: 0% хаос → 100% полный контроль.

## Шкала

| % | Что значит |
|---|------------|
| 0 | Хаос, реакция на внешнее |
| 10 | Сделана одна осознанная задача |
| 30 | Утро + рабочий блок управляются |
| 50 | План на день написан, частично выполнен |
| 70 | План + ритуалы (утро, спорт, рефлексия) |
| 90 | План + ритуалы + проактивность (создаёшь, не догоняешь) |
| 100 | Идеальный день — встречается редко, эталон |

## Почему это работает

- **Измеримо** — субъективная метрика, но воспроизводимая.
- **Скейлится:** влияние на день → неделю → жизнь → других → синергетический эффект.
- **Диагностика, а не оценка** — не «плохо/хорошо», а «где сейчас, что подкрутить».

## Применимость в Jetix

- Ежедневная самопроверка для агентов и самого Руслана.
- Часть life-coach-agent: автоматическая подсветка падений.
- Возможный data point для wiki/summaries (тренды по дням/неделям).
