# «Инженерная вера» — вера + инструменты + план

## Одна строка

Если составить план и есть инструменты для его внедрения — план может сбыться. Это **вера через инструменты**, не слепой оптимизм.

## Обоснование

[[concepts/engineering-faith]] — ключевой принцип Jetix. Альтернатива:

- **Слепой оптимизм** — «будет хорошо, потому что я хочу». Без плана. Без инструментов.
- **Циничный пессимизм** — «ничего не сработает, не пробуй».
- **Инженерная вера** — «у меня есть план + инструменты, поэтому есть основания верить; буду корректировать по обратной связи».

## Три компонента

1. **Вера** — убеждение, что план осуществим.
2. **Инструменты** — конкретные возможности реализации (люди, технологии, каналы).
3. **План** — последовательность, чекпойнты, метрики.

Отсутствие любого компонента — не инженерная вера, а нечто другое (энтузиазм / ремесло / фантазия).

## Применимость в Jetix

- Решения: проверять, есть ли у нас все три компонента.
- Презентация клиентам: не «обещаем результат», а «вот план, вот инструменты, вот механика проверки».
- Найм: «инженерная вера» — качество, которое фильтруем ([[ideas/ideal-member-portrait]]).
