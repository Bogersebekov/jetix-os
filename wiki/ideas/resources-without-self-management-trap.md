# Ресурсы без управления собой = ловушка

## Одна строка

Если есть ресурсы, но не можешь управлять собой (телом, вниманием, отношениями) — ресурсы бесполезны. Сам себя в клетку сажаешь.

## Обоснование

Деньги без здоровья = недолго.  
Связи без концентрации = пустое.  
Власть без ясности = разрушение.

Управление собой — **prerequisite** для использования ресурсов. Без него — ловушка.

## Идея для бизнеса

Помогать **людям с деньгами**, которые не могут управлять собой. Это конкретный ICP-сегмент:

- Топ-менеджеры с burnout.
- Founder'ы с PMF, но кризисом фокуса.
- Эксперты, заработавшие, но потерявшие meaning.

## Применимость в Jetix позиционировании

> Jetix помогает людям с ресурсами **использовать их на максимум** — без саморазрушения.
