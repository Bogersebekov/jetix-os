# Jetix-методология как «привычка мыть руки» — Semmelweis frame

## Одна строка

Один человек однажды начал мыть руки перед операциями → смертность упала в разы → через 50 лет это глобальный стандарт. Jetix-методология управления информацией имеет такой же потенциал — логична, полезна, самораспространяется по ценности.

## Mechanics

1. **Trigger:** новая стандартная практика, не институциональный мандат
2. **Adoption curve:** rejection → adoption by individuals → standardization (50-year arc для рук)
3. **Self-distribution:** работает потому что ценность *видима* (меньше смертность / меньше шума в голове / больше пропускной способности на единицу времени)
4. **Application к Jetix:** методология управления входящей информацией = новый слой гигиены поверх digital-life
