# Чек-листы с разной частотой проверки по темам

## Одна строка

У каждой жизненной темы своя скорость изменений и свой ритм проверки: питание — раз в месяц, спорт — раз в две недели, стратегические вопрос.

## Содержание (из Notion)

У каждой жизненной темы своя скорость изменений и свой ритм проверки: питание — раз в месяц, спорт — раз в две недели, стратегические вопросы — реже. Для каждой темы нужен чек-лист текущего уровня. Проверяешь: отклоняюсь или нет. Прокачался — поднял планку, держишься нового чек-листа. Элементарная система, которая не перегружает голову.

## Действие

Создать чек-листы по ключевым темам и назначить частоту ревью для каждого.
