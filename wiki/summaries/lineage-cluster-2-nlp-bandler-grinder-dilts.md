# Lineage Cluster 2 — NLP Canon (1975-2000)

## TL;DR

NLP corpus развивался от Structure of Magic (Bandler-Grinder 1975, transformational grammar applied) → Milton-model trance (1981) → Reframing (1982) → submodalities (Andreas 1989) → Robbins applied (1986-1991) → Sleight of Mouth (Dilts 1999) → Encyclopedia (2000).

## Ключевые выводы

- **Bandler-Grinder (1975-1982)** — основной 4-book canon: meta-model / representational systems / Milton-model / reframing
- **Andreas (1989, 1996)** — submodalities + well-formed outcomes
- **Robbins (1986, 1991)** — applied NLP (Unlimited Power, Awaken the Giant)
- **Dilts (1999, 2000)** — Sleight-of-Mouth catalog + Encyclopedia системного NLP
- **Bolton (1979)** — adjacent (People Skills, active listening)
- **O'Connor-Seymour (1990)** — practitioner introduction
