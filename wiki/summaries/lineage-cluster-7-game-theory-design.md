# Lineage Cluster 7 — Game Theory + Game Design

## TL;DR

Game theory / design canon: Schelling Strategy of Conflict (1960 coordination) → Axelrod Evolution of Cooperation (1984) → Csikszentmihalyi Flow (1990) → Salen-Zimmerman Rules of Play (2003) → Koster Theory of Fun (2004) / Rouse Game Design (2004) → Castronova Synthetic Worlds (2005) → Rogers Level Up (2010) → Lehdonvirta-Castronova Virtual Economies (2014) → Eyal Hooked (2014) → Varoufakis Technofeudalism (2023). + Schell Art of Game Design Lenses.

- **Axelrod 1984** — tit-for-tat в iterated PD
- **Csikszentmihalyi 1990** — flow state 8 dimensions
- **Salen-Zimmerman 2003 / Rouse 2004 / Schell** — game design schemas + lenses + interviews method
- **Koster 2004** — fun as pattern / mastery dynamics

- **Rogers 2010** — Level Up GDD
