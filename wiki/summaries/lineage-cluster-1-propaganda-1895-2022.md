# Lineage Cluster 1 — Propaganda / PR / Crowd Psychology

## TL;DR

Propaganda canon развивался от crowd psychology (Le Bon 1895) → group identification (Freud 1921) → pseudo-environment (Lippmann 1922) → engineering of consent (Bernays 1923-1928) → ambient propaganda (Ellul 1965) → 5-filters institutional (Chomsky-Herman 1988) → operational stickiness (Heath 2007, Berger 2013) → network-state cohort (Srinivasan 2022).

## Ключевые выводы

- **Le Bon (1895)** — crowd как entity с prestige leader; substrate всему последующему
- **Freud (1921)** — ego-ideal identification mechanism
- **Lippmann (1922)** — stereotypes + pseudo-environment expert class
- **Bernays (1923/1928)** — operationalization Freud-Le Bon в PR-campaign protocol; "engineering of consent"
- **Ellul (1965)** — propaganda typology (integration vs agitation; sociological vs political); ambient propaganda diagnostic
- **Chomsky-Herman (1988)** — institutional analysis 5 filters
- **Heath (2007), Berger (2013)** — operational stickiness (SUCCES, STEPPS) — neutral tools
- **Srinivasan (2022)** — Network State 7-step recipe — cohort substrate
