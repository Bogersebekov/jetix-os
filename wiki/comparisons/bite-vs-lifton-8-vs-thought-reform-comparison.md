# BITE vs Lifton 8 vs Thought Reform — Cult Diagnostic Comparison

## TL;DR

Lifton (1961) 8-criteria substrate → Hassan (1988) BITE = операционализация в 4 dimensions. Оба defensive-use only.

## Сравнительная таблица

| Framework | Author | Year | Dimensions | Use mode |
|---|---|---|---|---|
| **8 Criteria of Ideological Totalism** | Lifton | 1961 | 8 (milieu / mystical / purity / confession / sacred / language / doctrine / dispensing) | Original diagnostic substrate |
| **BITE Model** | Hassan | 1988 | 4 (Behavior / Information / Thought / Emotion) | Strategic Interactive Approach — operational protocol |
| **Thought Reform programs** | Various | 1950-now | varies | Programs targeted by both diagnostics |

## Mapping Lifton 8 → BITE 4

- Milieu control / sacred science / loading the language → **Information control**
- Mystical manipulation / sacred science → **Thought control**
- Demand for purity / confession / dispensing of existence → **Emotion control**
- Doctrine over person → **Behavior control** (via prescribed action)

## Применимость

- **Defensive audit only** — обе frameworks для protection
- **R12 STRICT** — NEVER use to design cohort recruitment that triggers any dimension
- **Pair с consent-boundary-protocol** mandatory
