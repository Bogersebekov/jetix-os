# Jetix as Hackathon Platform

## Суть в одной строке

Jetix = платформа для проведения качественных глубоких марафонов where participants работают над сложнейшими задачами через FPF protocol + AI capability + mentor pairing + tools/compute access; multi-rhythm cycles (day/month/year).

## Определение

**Hackathon platform** = primary operational pattern Jetix (per text_008 + text_009 18.05 evening Ruslan dictation). Не consulting business, не SaaS — platform для events where:
- Participants connect через FPF protocol-mediated coordination ("за 2 часа в лица в проекты").
- Stack baseline: Cloud Cowork compute + FPF 72K строк vendored + Workshop discipline + mentors + best AI наработки + tools + protocols.
- Dual-sided value: «решить свои проблемы» + «куча возможностей».
- Multi-rhythm: day (sprint mode, low-medium complexity) / month (sponsor cycle) / year (Master Workshop apprenticeship).

## Применимость

- **Phase 1 commercial path:** primary growth vehicle (vs L1-deep-research baseline).
- **First event mode:** bloggers + sponsor projects (Thread 11) — accessible / low-friction.
- **L1 key-person unlock:** Karpathy + Musk + Anthropic + Buterin priority (Thread 12).
- **Master Workshop activation:** «не ступеньки ниже» discipline (Thread 14).

## Operational pattern

1. **Initiate** — sponsor commits + project surfaced + strategy file shared.
2. **Match** — participants (прошаренные + bloggers + general cohorts) + mentors + tools auto-assigned via FPF.
3. **Solve** — multi-day/month marathon с continuous mentor pairing (TPS pattern).
4. **Reward** — QF distribution per Tang+Weyl mechanism (Ethereum architecture acked 18.05).
5. **Recurse** — winners feedback к platform improvement (gratitude loop per Thread 7).
