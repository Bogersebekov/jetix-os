# OMG Essence — 7 Alphas Kernel (2015)

## Суть в одной строке

OMG Essence v1.1 — 7 универсальных alphas (Opportunity / Stakeholders / Requirements / Software System / Team / Work / Way-of-Working) + activity spaces + competency levels.

## Определение

Essence Kernel and Language for Software Engineering Methods (OMG, 2015) — стандарт для описания SE methods через universal kernel:

**7 Alphas (essential things to work with):**
1. Opportunity
2. Stakeholders
3. Requirements
4. Software System
5. Team
6. Work
7. Way-of-Working

**State cards** — каждая alpha имеет 5-6 states с checklist criteria (e.g. Requirements: Conceived → Bounded → Coherent → Acceptable → Addressed → Fulfilled).

**Activity spaces** — типы активностей (independently of specific method).

**Competency levels** — Assists / Applies / Masters / Adapts / Innovates.

## Применимость

- **Universal method description language** — любой specific метод can be described через Essence
- **Foundation для Jetix Part 4 Role Taxonomy alpha-binding** (per Foundation Part 4 §H)
- **Levenchuk explicit inheritance + extends** (СМ т.1/т.2 substrate)
