# Equal-Conditions Hackathon Protocol

## Суть в одной строке

Все participants начинают hackathon с **одинаковыми базовыми условиями** (money / resources / accommodation / food / tool access / team formation) — Rawlsian veil-of-ignorance fairness; устраняет donaters / lucky people / exclusives как distortion factor.

## Что equalize'ится

| Element | Equalized base | Source |
|---|---|---|
| Stipend / база $$ | Equal per-participant amount | audio_687 |
| Resources / tools | Same toolkit access | audio_687 |
| Accommodation / место жизни | Same housing standard | audio_687 |
| Food | «Любая еда» — equal access | audio_687 |
| Team formation | 5-person teams | audio_687 |
| External access | «Доступ до любого человека» — same | audio_687 |

## Что НЕ equalize'ится

- Pre-existing skills / experience
- Personal motivation / drive
- Pre-existing network (но external-access equalized via hackathon protocol)
- Cognitive style / creativity
- Brain extension (cheat-sheets allowed, see O-41) — measures reasoning + tool-use, not raw memory

## Operating cost estimate

Per engineering cell 6.1:
- ~€3K/person/month (€1K housing + €500 food + €500 base $$ + €1K tools/access)
- 100-person × 1-month = **€300K** per event
- 10 events/year = **€3M** annual ops cost (~€3 per user-year if 1M target)

**Investability**: ops cost trivial vs $1B target ($3M = 0.3%).

## Rationale

1. **Anti-donaters**: removes resource asymmetry distortion («не выебывайся»)
2. **Anti-lucky**: removes random-advantage distortion
3. **Anti-exclusives**: removes pre-existing access distortion
4. **Pure skill measurement**: «доказывайте свои скиллы блять свою силу»
5. **Welcome ethic**: «welcome, all you need»

## Philosophical framing (per philosophy cell 6.3)

**Rawlsian veil-of-ignorance reading**: legitimate fairness mechanism — remove arbitrary advantages, expose true ability.

**Paternalism reading risk**: "we equalize because participants can't decide" — frame to AVOID.

Default frame = Rawlsian; communicate accordingly.
