# Hook Model Canvas (Eyal 2014)

## Суть в одной строке

Eyal (2014) Hook Model = Trigger / Action / Variable Reward / Investment — 4-step habit-formation funnel; R12 paired-frame REQUIRED.

## Определение

Hooked (Eyal 2014) — каноническая habit-formation модель:

1. **Trigger** — external (notification) или internal (emotion) cue
2. **Action** — simplest behavior in anticipation of reward
3. **Variable Reward** — unpredictable reward (tribe / hunt / self)
4. **Investment** — user invests time / data / social capital → increases future trigger sensitivity

Variable reward — ключевой engagement multiplier (Skinner box psychology).

## R12 paired-frame STRICT

- **Eyal's manipulation matrix chapter (book itself)** — must pair с Hook model deployment
- **Defensive question:** does user net benefit от engagement? Or extraction?
- **Anti-pattern detection:** dark patterns, infinite scroll, autoplay, FOMO triggers
