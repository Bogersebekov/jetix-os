## §1 Что это

**Принцип:** Jetix selection mechanic = **competence + state + alignment based**, не demographic / status / popularity based. Selection ≠ exclusion; selection = mutual-fit assessment + voluntary opt-in / opt-out.

**Dual frame:**
- **Positive profile** — характеристики того кто **fits naturally** в Jetix cohort
- **Anti-target profile** — характеристики того кто **не fits** (без negative judgment; mismatch, не deficit)

**Key invariant:** все criteria = state / motivation / alignment based. Demographic agnostic. «Школьники / 40-летние мужики / трейдеры / блогеры» — на любом уровне, любой age, любой background.

## §3 Anti-target profile components (substrate verbatim + HR-soften per §6)

### §3.1 Substrate dimensions

| # | Dimension | Substrate verbatim | Soften phrasing for external |
|---|---|---|---|
| 1 | **Surface intellect без desire** | «умные внешне-интеллектуальные» (без «желания развиваться») | «individuals oriented toward intellectual status display rather than continuous growth» |
| 2 | **Mega-popularity без depth** | «мега-популярные» (с «нет времени развиваться») | «attention-rich individuals with limited depth-engagement bandwidth» |
| 3 | **No time / no desire** | «нет желания / нет времени» | «competing priorities precluding deep engagement» |

### §3.2 Anti-target ≠ negative judgment

**Critical clarification:** Anti-target profile **does NOT imply moral deficit / less worth / lower status**. It implies **mismatch with Jetix specifically** (не fit для этой specific community). These people могут excel в других contexts.

**Why this matters:**
- R12 anti-extraction = no extracting value beyond agreed share. Anti-target assessment = pre-engagement filter, не post-engagement extraction
- Voluntary opt-in clause MUST be available для всех (including anti-target candidates); они могут choose to opt-in despite the mismatch flag
- Fork-and-leave preserved — anti-target who joins and later realises mismatch leaves without penalty

## §7 Risks

### §7.1 Demographic creep

**Risk:** «школьники» mention drift к specific demographic targeting (age-based filtering). Demographic-agnostic invariant erodes.

**Mitigation:** §2.2 explicit + outreach copy review checks за demographic-specific language.

### §7.2 Cult-formation risk

**Risk:** «голод + ответственность + дисциплина» language tonally adjacent к high-demand groups / cult patterns.

**Mitigation:** R12 fork-and-leave preserved + voluntary opt-in mandatory + transparent anti-target profile prevents «only-yes-people-included» group-think pattern. Charter v0 enforces.

### §7.3 Selection inflation

**Risk:** «top-tier only» framing creates artificial scarcity / status-extraction.

**Mitigation:** Demographic-agnostic invariant (any background can fit) + voluntary opt-in (no gatekeeper inflation) + fork-and-leave (no captivity).
