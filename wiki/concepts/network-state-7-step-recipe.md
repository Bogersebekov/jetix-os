# Network State 7-Step Recipe (Srinivasan 2022)

## Суть в одной строке

7-шаговый recipe построения network state (Srinivasan 2022) — substrate inheritance для Jetix-Clan.

## Определение

Srinivasan The Network State (2022) — 7-step recipe:

1. **Founder** — индивид с конкретной legitimacy
2. **Moral innovation** — новый моральный consensus, отсутствующий в host society
3. **Strategy** — путь к accumulating people / capital / territory
4. **Community** — online substrate для cohort
5. **Coordination** — economic/social/political mechanics
6. **State-substrate** — physical footprint (со-распределение населения / cloud-first city)
7. **Recognition** — diplomatic legitimacy
