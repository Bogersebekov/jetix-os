# BITE Model — Cult Recruitment 4-Dimensional Diagnostic

## Суть в одной строке

Hassan (1988) 4-мерная диагностика cultic-mechanic: контроль Behavior / Information / Thought / Emotion.

## Определение

Strategic Interactive Approach (Hassan 1988, на базе Lifton 1961 8-criteria) — BITE = диагностическая рамка для defensive детектирования cult-mechanics в любой organisation/movement:

- **Behavior control:** распорядок дня / питание / одежда / разрешения на действия
- **Information control:** ограничение доступа к external information / контроль внутренней коммуникации
- **Thought control:** загруженный язык / thought-stopping / dichotomous thinking
- **Emotion control:** манипуляция guilt / fear / love-bombing / phobia indoctrination

## Применимость — STRICT DEFENSIVE ONLY

- **Defensive audit** любой organisation/movement (своей или внешней) на cult-creep
- **Cross-reference Lifton 8-criteria** (1961) для полноты
- **R12 paired-frame STRICT:** NEVER use to design cohort recruitment that triggers any BITE dimension
- **Auto-pair influence-ethics-expert** mandatory для любого использования
