# Faucet-Sink Virtual Economy

## Суть в одной строке

Lehdonvirta-Castronova Virtual Economies (2014) — баланс через faucets (currency generation) и sinks (currency removal); canonical virtual-economy/tokenomics reference.

## Определение

Virtual Economies (Lehdonvirta-Castronova 2014) — economic design для virtual worlds + token systems:

- **Faucets:** mechanisms создающие currency / items (quests, rewards, drops, periodic distribution)
- **Sinks:** mechanisms удаляющие currency / items (consumables, taxes, fees, equipment decay)
- **Balance equation:** faucet rate ≈ sink rate → стабильная economy
- **Imbalance consequences:** faucet > sink → inflation; sink > faucet → contraction
- **External market interaction** — RMT (real-money trading) requires sink management

Substrate Castronova Synthetic Worlds (2005) — economic anthropology virtual worlds.

## Применимость к Jetix-Clan tokenomics

- **Token system design** — explicit faucet / sink accounting
- **Pair с R12 enforcement** — faucet/sink imbalance ≠ extraction mechanism
- **Anti-pattern detection** — Varoufakis rentier-economy при extreme imbalance
