# CHC (Cattell-Horn-Carroll) — empirical intelligence taxonomy

## Суть в одной строке

CHC = most-empirically-grounded modern intelligence taxonomy (~80 years lineage, 461 datasets reanalyzed by Carroll 1993, used in WAIS/WJ-IV/V cognitive batteries).

## Three-stratum structure (Carroll 1993)

- **Stratum III:** g — general intelligence
- **Stratum II:** ~10 broad abilities
- **Stratum I:** 70+ narrow abilities

## 10 broad abilities (Schneider & McGrew 2018)

| Code | Name | Definition |
|---|---|---|
| **Gf** | Fluid reasoning | Novel-problem-solving via controlled mental operations |
| **Gc** | Comprehension-knowledge | Depth + breadth of acquired knowledge |
| **Gv** | Visual processing | Generate/analyze/manipulate visual information |
| **Ga** | Auditory processing | Discriminate/manipulate auditory stimuli |
| **Gs** | Processing speed | Automatic-task speed + fluency |
| **Gwm** | Working memory (was Gsm) | Encode/maintain/manipulate immediate info |
| **Glr** | Long-term storage + retrieval | Store + fluently retrieve through association |
| **Gq** | Quantitative knowledge | Math reasoning |
| **Grw** | Reading + writing | Literacy fluency |
| **Gkn** | Domain-specific knowledge | Specialized expertise |

## Cross-link к 12-component intellect framework

Per [[jetix-curriculum-module-map]] Phase 2 audit:

- **Strong coverage** (5 of 10 CHC broads): Gf (C9), Gwm (C4/C11/C12), Gkn (C5), Gc (partial), Gq (partial C9+C10)
- **WEAK coverage** (5 of 10 CHC broads): Gv (only C8 partial), Ga (absent), Gs (absent), Glr (partial C7), Grw (absent)

**Implication:** 12-component is **engineering-cognitive subset** of CHC. Phase 5 surfaces Gwm + Glr as critical missing-component candidates (MC-1 + MC-5).

## Usage in cognitive batteries

- WAIS-IV/V (Wechsler Adult Intelligence Scale)
- WJ-IV (Woodcock-Johnson — direct CHC primary)
- KABC-II (Kaufman)
- DAS-II
- Stanford-Binet 5

## Cross-cultural validation

Cross-cultural replication of broad factor structure (Wang & Lin 2016 China; Reynolds et al. 2013 European-American). F4 empirical.

## Critique

- Gardner (1983/1999): «misses musical/BK/intrapersonal»
- Sternberg (1996): «lacks practical-intelligence dimension»
- Critical pedagogy: «reproduces Western academic-IQ bias»
- Schneider-McGrew rebuttal: «Gkn accommodates non-Western expertise»
