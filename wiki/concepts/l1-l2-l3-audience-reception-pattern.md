# L1/L2/L3 audience reception pattern

> Concept (Tier C — Ruslan ack required). Pattern of distinct audience-tier reception filters; applies к Jetix pitch substrate. Source = K-2 research Phase 3-5.

## §1 Three distinct reception tiers

| Tier | Audience | Dominant filters | Mandatory signals |
|---|---|---|---|
| L1 | ML/AI engineers + researchers | Engineering depth + open-source + concrete artifact + Karpathy-lineage | Working code, FPF spec, ROY swarm, benchmark |
| L2 | AI founders + tech founders | PMF + defensibility + GTM + 18-mo milestones + non-crypto | Cohort traction, community retention, revenue model, advisory cluster |
| L3 | Tier-1 VCs + strategic + LPs | Thesis fit + TAM + moat + LP-risk + vision narrative + constitutional governance | Vision narrative depth (K-1 lineage), Foundation Arch LOCKED, DD packet, governance docs |

## §2 Reception filters per tier (top-7 each)

(Detail в research/* per-tier reception simulation files.)

**L1:** benchmark / Plurality precedent / specification / anti-scale / moving-goalposts / open-source / agent+loop / consulting-skepticism / political-collective / geopolitical

**L2:** team / GTM / 18-mo / OpenAI-Anthropic threat / where's-product / DAO-Web3 / unit-economics / why-not-consulting / AGI-bold / cultural

**L3:** TAM / defensibility / team / capital-efficiency / exit / AI-ethics / regulatory / substrate-framing-LP / Web3 / founder-solo

## §3 Cross-tier wording sensitivity

- «Collective» = political baggage в US audiences → «collaborative effective protocol» neutralization
- DAO/Web3 vocabulary = friction L1/L2/L3 — explicit non-crypto positioning mandatory
- «AGI» bold claim = friction L2/L3 — honest-redefinition framing required

- L1-variant: engineering-credibility-forward
- L2-variant: cohort-business-forward
