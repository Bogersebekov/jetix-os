# Milton Model (Bandler-Grinder 1981)

## Суть в одной строке

Bandler-Grinder Erickson-inverse-of-meta-model: 12+ hypnotic patterns (embedded commands / pacing-leading / nested loops) — covert-hypnosis territory.

## Определение

Trance-formations (Bandler-Grinder 1981) — каталог hypnotic language patterns, моделированных с Milton H. Erickson. В противоположность Meta-Model (precision), Milton Model = artful vagueness:

- **Embedded commands** — команды spaced в обычной речи
- **Pacing & leading** — соединение с current experience → ведение к suggestion
- **Nested loops** — рекурсивные nested stories для disorientation
- **Presuppositions, complex equivalents, cause-effect** — implicit assertions
