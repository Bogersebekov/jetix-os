# Hedonic Treadmill

## Суть в одной строке
Humans return к baseline happiness after positive/negative events — chasing happiness via external rewards yields diminishing returns.

## Определение
Brickman & Campbell 1971 — adaptation pattern: lottery winners return to baseline within 1-2 years; major loss similarly fades. Implication: external rewards (money, status) yield short-term happiness, then adapt. Sustainable happiness requires meaning + connection + competence growth (matches SDT).

## Ключевые свойства
- Adaptation to baseline
- Universal across cultures
- Returns from money diminish above $75K-ish (Kahneman 2010)
- Meaning + connection sustainable
- Implication для reward design

## Применимость к Realm
- Money rewards yield diminishing returns
- Meaningful work / connection / growth more sustainable
- Anti-pattern: pure financial motivation
