# Variable Rewards

## Суть в одной строке
Rewards delivered на unpredictable schedule (vs fixed) generate strongest behavior repetition — slot machines, social media notifications, loot boxes use this.

## Определение
Skinner identified variable-ratio reinforcement schedule maximally persistent. Eyal Hook Model uses variable rewards (Reward of the Tribe, Hunt, Self). **Powerful but ethically loaded** — gambling, social media compulsion, loot box mechanics all leverage this. Realm: use cautiously, никак addictive design.

## Ключевые свойства
- Unpredictable timing/amount
- Maximally persistent (slow extinction)
- 3 reward types (tribe / hunt / self)
- Dopamine prediction-error mechanism
- Anti-pattern when manipulative

## Применимость к Realm
- LIMITED use — fixed rewards primary (transparent)
- Variable bonus elements OK (peer kudos timing variable)
- NO variable monetary rewards (gambling adjacent)
