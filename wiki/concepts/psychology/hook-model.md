# Hook Model

## Суть в одной строке
4-stage habit-formation loop: Trigger → Action → Variable Reward → Investment — repeated cycles build habit + investment locks in.

## Определение
Eyal Hooked (2014). **Trigger**: external (notification) → internal (boredom, loneliness). **Action**: simple behavior in anticipation of reward. **Variable Reward**: unpredictable payoff (3 types: tribe/hunt/self). **Investment**: user puts in time/data/social capital → future triggers more potent. Cycle repeats → habit. **Ethically loaded** — Eyal himself wrote Indistractable (2019) as counter.

## Ключевые свойства
- 4-stage cycle
- Internal trigger (post-onboarding)
- Variable reward (gambling-adjacent psychology)
- Investment locks-in
- Habit formation

## Применимость к Realm
- Trigger: scheduled rituals (Monday all-hands, weekly retrospective)
- Action: simple low-friction (check quest board)
- Variable reward: NOT primary — use fixed transparent
- Investment: visible accumulation (knowledge, reputation)
