# Competence (SDT)

## Суть в одной строке
Need для feeling effective при interactions со environment — visible mastery, progress, agency.

## Определение
SDT competence = sense of being capable, mastering challenges, making progress. Threatened by uncalibrated challenges (too easy = boring, too hard = frustrating). Visible progress essential. Feedback timely + meaningful.

## Ключевые свойства
- Effectiveness feeling
- Mastery progression
- Calibrated challenge
- Timely feedback
- Skill visibility

## Применимость к Realm
- E2 visible class progression
- Calibrated quest difficulty (challenge-skill balance — Csikszentmihalyi)
- Timely feedback (peer review, mentor session)
- Visible skill metrics
