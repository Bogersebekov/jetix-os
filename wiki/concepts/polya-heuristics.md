# Polya Heuristics

## Суть в одной строке

Polya How to Solve It (1945) — 4-step plan (understand / plan / execute / look back) + heuristic dictionary (80+ entries) для universal problem-solving.

## Определение

George Polya How to Solve It (1945):

**4-step plan:**
1. **Understand the problem** — what is unknown / what is data / what is condition
2. **Devise a plan** — connect to similar problems / auxiliary problems / generalize / specialize
3. **Execute the plan** — verify each step
4. **Look back** — verify result / alternative routes / generalize lesson

**Heuristic dictionary** — 80+ entries: analogy / auxiliary problem / working-backward / generalization / specialization / decomposition / recombination / induction / etc.

## Применимость

- **Universal problem-solving toolkit** — applicable beyond mathematics
- **Substrate для Lakatos Proofs & Refutations** (1976) — direct extension
- **Methodology-engineering substrate** — heuristic dictionary как explicit pool для method-cards
