# NLP Meta-Model (Bandler-Grinder 1975)

## Суть в одной строке

Bandler-Grinder Meta-Model — 12-14 лингвистических паттернов (deletions / generalizations / distortions) для therapeutic questioning.

## Определение

Structure of Magic vol.1 (Bandler-Grinder 1975) — применение Chomsky transformational grammar к терапевтической практике (Perls / Satir моделирование). Meta-Model = каталог questions, выявляющих:

- **Deletions** — что опущено в surface structure (unspecified noun/verb, comparative deletion)
- **Generalizations** — universal quantifiers, modal operators (must / can't), mind-reading
- **Distortions** — cause-effect / equivalence / presuppositions

Цель: восстановление deep structure из surface speech для повышения precision.

## Применимость

- **Therapeutic questioning** (точечные вопросы для clarification)
- **Audit собственного и чужого discourse** для выявления imprecisions
- **R12 paired-frame required:** defensive-frame allowed; covert offensive use = Default-Deny
