# Rentier Economy Diagnostic (Varoufakis 2023)

## Суть в одной строке

Varoufakis Technofeudalism (2023) — cloud capital / rentier economy / data-as-fief / algorithmic vassalage; canonical platform-critique diagnostic.

## Определение

Technofeudalism: What Killed Capitalism (Varoufakis 2023) — argumentation, что современный platform capitalism — не capitalism, а neo-feudalism:

- **Cloud capital** — platforms (Amazon, Google, Apple) контролируют access к "digital cloud commons"
- **Rentier economy** — primary income — rent (commission, access fees), не value creation
- **Data-as-fief** — user data + attention = land equivalent в feudal economy
- **Algorithmic vassalage** — workers + small businesses зависят от algorithm access
- **Cloudalists** — new ruling class (cloud capital owners)

## Применимость

- **Anti-pattern detection** для Jetix substrate design — REJECT rentier patterns
- **R12 paired-frame:** any platform-design proposal должен pass rentier-diagnostic test
- **Pair с 5-filters propaganda model** — platform-extension fifth filter
- **Cross-ref Castronova / Lehdonvirta** — virtual economy design без rentier mechanics
