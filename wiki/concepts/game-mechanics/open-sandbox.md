# Open Sandbox

## Суть в одной строке
No prescribed goals — player provides own objectives. World as creative substrate (Minecraft, Terraria, Garry's Mod).

## Определение
Open sandbox = anti-quest design. Game provides tools (blocks, crafting, physics), не scenarios. Player creates art, machines, narratives. **Maximal autonomy** (SDT-aligned). Risk: no onboarding hand-holding loses players who need direction. Counter: «creative mode» (free) + «survival mode» (constrained).

## Ключевые свойства
- No prescribed goals
- Rich toolkit (crafting, building)
- Emergent gameplay (player-designed)
- Optional structured modes
- Community-shared creations

## Применимость к Realm
- E4 contrast: alongside structured quests, free-form clan projects
- «Garage» pattern: free experimentation space
- Optional structure (newcomers use quest), advanced create own
