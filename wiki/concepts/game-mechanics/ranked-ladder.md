# Ranked Ladder

## Суть в одной строке
Hierarchical tier system (Iron / Bronze / Silver / Gold / Platinum / Diamond / Master / Grandmaster / Challenger) с MMR underneath — visible status signaling.

## Определение
LoL/Dota canonical ladder: 9 tiers, divisions within each, hidden MMR underneath. Players placed each season, climb through wins. **Visible status** — tier displayed prominently. Seasonal reset (rank goes down 1-2 tiers, must re-prove). MMR persists.

## Ключевые свойства
- 9 tiers + sub-divisions
- Hidden MMR (true skill)
- Visible tier (status signal)
- Seasonal reset
- Tier rewards (skins, badges, frames)

## Применимость к Realm
- E1 anchor: visible tier per class
- Seasonal recalibration (E6 connection)
- Hidden true-skill + visible tier (smoothing volatility)
