# Crime Skill Progression (Torn)

## Суть в одной строке
Skill trees per crime category (vandalism, theft, larceny, fraud, robbery, ...) — sub-class specialization, не main-class lock-in.

## Определение
В Torn 12+ crime categories каждая со своими навыками. Players развивают skills через repetition (high-level players unlock high-payout crimes). Skill в одной не блокирует other. Pattern — sub-specialization внутри main class.

## Ключевые свойства
- 12+ orthogonal skill categories
- Repetition-based gain (do crimes → gain skill)
- High-skill unlocks high-payout crimes
- Cross-skill bonuses (some criminal-trait synergies)
- NO respec needed (накопительно forever)

## Применимость к Realm
- E1 + E2: sub-skills внутри Hunter/Architect/etc.
- Доменно-специфичная expertise (e.g. Architect: AI agents / data viz / system design — orthogonal)
- Sub-skill development без класс-changes
