# Season Pass Cycle (3-Month)

## Суть в одной строке
~10-12 week cycle с consistent rhythm: launch hype → mid-season live event → final stretch → next season hype.

## Определение
Fortnite established 3-month season как dominant cadence. Structure: Week 1-2 (hype + new map elements), Week 3-8 (steady-state content drip), Week 9 (live event preparation), Week 10-11 (live event + finale), Week 12 (pre-new-season teasing). Predictable rhythm allows planning, generates retention spikes.

## Ключевые свойства
- 10-12 week duration
- Live event mid-season (concert, finale)
- Pre-new-season teasing
- New theme per season
- Predictable cadence

## Применимость к Realm
- E6 anchor: 3-month rhythm
- Mid-season live event (online showcase, demo day)
- Pre-season teasing (next theme announcement)
