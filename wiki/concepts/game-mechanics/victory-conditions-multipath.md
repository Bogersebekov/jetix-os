# Victory Conditions Multi-Path

## Суть в одной строке
Multiple parallel «win» definitions (Domination / Science / Culture / Religion / Diplomatic / Economic) — players choose path matching playstyle.

## Определение
Civilization series 5-7 distinct victory conditions, all equally validated «winning». Player can pivot mid-game. **Anti-monoculture pattern** — prevents one optimal strategy. Direct mapping to Realm — multiple ways to be successful clan / player (Hunter dominates sales / Architect builds reusable platforms / Creator builds audience — all valid mastery).

## Ключевые свойства
- 5-7 parallel victory definitions
- All equally «winning»
- Pivot allowed mid-game
- Visible progress on each
- Score = aggregate (combined approach also valid)

## Применимость к Realm
- E2 class anchor: 6 class paths all valid «mastery»
- E4 quest: multi-path quest completion (any class can solve по разному)
- Avoid monoculture optimization
