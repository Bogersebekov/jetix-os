# Mobile-First Short Match

## Суть в одной строке
10-min average match, designed для mobile context (commute, lunch break, between meetings) — bite-size session pattern.

## Определение
Honor of Kings (王者荣耀) optimized MOBA для mobile sessions — 10-min avg vs Dota 35-min. Match windows: morning commute, lunch, evening transit. **Pattern**: design length к context, not «as long as possible». Casual entry, high retention through fits-life-rhythm.

## Ключевые свойства
- 10-min average
- Touchscreen optimized
- Push-notification re-engagement
- Battery-conscious (60% session under 50% battery)
- Auto-pause / disconnect tolerance

## Применимость к Realm
- E4: micro-quests (5-15 min) для micro-windows in day
- Mobile UX considerations
- Re-engagement notifications (calibrated, not spammy)
