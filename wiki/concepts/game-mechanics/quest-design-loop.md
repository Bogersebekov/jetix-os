# Quest Design Loop (WoW)

## Суть в одной строке
Quest design = action verb + scope + reward + narrative — каждый quest 5-15 min, leveraging «5 W+H» framework (who/what/where/why/how + reward).

## Определение
WoW quest design canonical pattern. Loop: NPC giver → объяснение task (1-3 sentence narrative) → travel to objective → execute (kill 10 / collect 5 / interact с object) → return → reward (XP + gold + item + reputation). Designer iteration tools include map of all quests in zone (heat map of player time-spent).

## Ключевые свойства
- 5-15 min duration target
- Single objective verb (kill/collect/escort/interact)
- Narrative context (1-3 sentences)
- Visible reward preview
- Quest chains (sequel quests in same area)
- Daily quests (repeatable) for end-game retention

## Применимость к Realm
- E4 quest template direct lift
- Quest parameters: action verb, scope, reward, narrative, difficulty
- Quest chains для multi-step projects
