# Collection Mechanic

## Суть в одной строке
«Catch'em all» — set completion psychology (Pokedex, WoW pets, Hearthstone deck) — completionist drive за long-tail engagement.

## Определение
Collection mechanic explicit «catalog of items» with completion %. Players intrinsically motivated to fill blanks. Pokedex (1010+ pokemon), WoW Pet Battles (mountains, transmogs), Hearthstone (cards), Magic the Gathering (cards). **Psychological pattern**: closure incomplete sets compels acquisition. Strongest with finite known total.

## Ключевые свойства
- Finite known total (e.g. 1010 pokemon)
- Visible completion % per category
- Rare items (1% drop rate, etc.)
- Trading (helps complete)
- Showcase (cosmetic display)

## Применимость к Realm
- Knowledge artifacts catalog (mastered skills, completed playbooks)
- Visible completion % per class
- «Showcase» on profile
