# Faction Wars (Torn)

## Суть в одной строке
Two-faction competitive war: атаки/защита, points accumulation, prize pool — direct mapping для соревновательных тендеров между клами на real client deals.

## Определение
В Torn Faction Wars — declared conflict между двумя factions; participants атакуют opposite-faction members, набирают points, winner получает prize pool. Часовая длительность 24-72 часа. Создаёт high-intensity coordinated team play, retains members при downtime.

## Ключевые свойства
- Declared by mutual consent (нет gankg случайного)
- Time-bounded (24-72h)
- Prize pool (winner takes most, loser smaller cut)
- Coordinated team strategy required
- Visible scoreboard (real-time)

## Применимость к Realm
- Соревновательные тендеры: 2-4 кланов pitch одному клиенту → клиент выбирает (или blind judge)
- Time-bounded sprints (1-2 weeks pitch window)
- Prize pool = client contract value + reputation gain
