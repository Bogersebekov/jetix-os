# Talent Tree Progression

## Суть в одной строке
Branching skill tree where points spent unlock new abilities and modify existing — players customize class specialization through 30-50 trade-off choices.

## Определение
Talent tree pattern: bottom tier (no prereqs), middle tiers (require prereq points), top tier (capstone). 1 point per level. Customization through trade-offs (cannot max all). WoW Dragonflight (2022) returned classic tree design after Cataclysm/Legion linearization. Diablo II/IV — pure trees.

## Ключевые свойства
- Branching structure (cap row 1 → unlock row 2)
- Trade-offs (cannot max everything)
- Capstone (powerful pinnacle ability)
- Respec available (gold/token cost)
- Visualized graph

## Применимость к Realm
- E2 class progression = talent tree (Hunter sub-specializations: solo prospector / cold-mailer / closer)
- Trade-offs force commitment
- Respec at cost (knowledge token spend)
