# Achievement System (WoW)

## Суть в одной строке
1000+ named achievements (e.g. «Glory of the Raider», «Loremaster») с visible badges + total Achievement Points — long-tail completion psychology.

## Определение
WoW Achievement system introduced Wrath of the Lich King (2008). 1000+ achievements organized by category (PvE, PvP, Exploration, Reputation). Each имеет visible criteria, point value, optional reward (mount, title, pet). Achievement Points aggregate on profile — visible signaling of player «completionist» status. Direct mapping to badges, but with depth.

## Ключевые свойства
- Named (each имеет лор-name)
- Tiered (criteria 1/5/10/25/100 progressions)
- Points (aggregate score)
- Rare achievements (small % completion)
- Visible on profile
- Optional rewards (cosmetic + functional)

## Применимость к Realm
- E1 anchor: visible achievement portfolio per player
- Reputation accumulator separate from monetary
- Long-tail engagement (always more to «100%»)
