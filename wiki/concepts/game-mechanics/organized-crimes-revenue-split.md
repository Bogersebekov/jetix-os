# Organized Crimes 80-95% Revenue Split (Torn)

## Суть в одной строке
Структурированная коллективная активность (8 типов OC, 4-8 участников) с reward split: 80-95% участникам / 5-20% Faction pool — direct lift для Realm Organized Projects.

## Определение
В Torn Organized Crimes = scripted multi-player operations с заранее известными ролями (driver, hacker, muscle, etc.), required skills, time-to-cool-down. Revenue распределяется по фиксированной формуле, не по дискреции лидера. Это формализует team-share договорённости — нет ad-hoc торга на каждом проекте.

## Ключевые свойства
- 8 типов OC (different role mix, payout, risk)
- Fixed roles (никто не может «sub in» на роль без skill)
- Fixed payout split (80-95% workers / 5-20% faction pool)
- Cooldown между OCs одного типа
- Skill check per role (high скилл → лучший loot)

## Применимость к Realm
- E3 + E5 anchor для Organized Projects
- Revenue split (90% executor team / 10% Jetix HQ) — direct template
- Role pre-definition forsts «правильное» формирование команды по специализациям
