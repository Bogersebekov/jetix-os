# Competitive Ranking (CS2 Premier)

## Суть в одной строке
CS2 Premier rating — visible numeric rating (5000-25000+), regional leaderboards — pure numeric (no tier hide) status signaling.

## Определение
CS2 Premier matchmaking uses visible numeric rating (no Bronze/Silver/Gold tier obscurity). Players see exact rating, see leaderboard position globally + regionally. **More transparent** than tier system but psychologically harsher (drop visible).

## Ключевые свойства
- Visible numeric rating
- Regional + global leaderboards
- Direct competitive (no group queue dilution)
- Periodic season resets
- Skill match по rating distance

## Применимость к Realm
- Alternative to tier system: visible numeric metric
- Regional + global leaderboard
- Pro: transparent. Con: «drop psychology» — visible loss painful
