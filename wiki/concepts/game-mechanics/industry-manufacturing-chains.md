# Industry / Manufacturing Chains (EVE)

## Суть в одной строке
Multi-stage production: mine ore → refine into minerals → manufacture components → assemble ship — each stage может быть отдельным player specialization.

## Определение
EVE production chains 4-6 stages для capital ships. Создаёт vertical division of labor: миннеры, refinеры, manufacturers, traders — каждый специалист может выбрать свою стадию. Inputs/outputs market-bought, не «mine all yourself». **Direct mapping для Realm:** complex client project разбит на 4-stage production (research → design → execute → polish), каждая стадия — другая class/clan.

## Ключевые свойства
- 4-6 stages per complex product
- Each stage specialization (skill, infrastructure)
- Inputs/outputs traded at market
- Profit margin per stage (slim, requires volume)
- Vertical integration option (single player whole chain — inefficient)

## Применимость к Realm
- Quest decomposition (E4) into 3-5 stages
- Specialization-based assignment per stage
- Marketplace coupling inputs/outputs
