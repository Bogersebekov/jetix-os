# Skill Ceiling Progression

## Суть в одной строке
Very high skill ceiling (mechanical + strategic) — player progression spans 1000s of hours; mastery as long-tail retention engine.

## Определение
Dota 2 mechanic complexity (130 heroes, item combos, last-hitting, lane management, map awareness) creates years-long progression curve. Top players still improving после 10K+ hours. Contrast: simple games hit ceiling quickly, retention drops. **High skill ceiling = retention engine; pattern для Realm = perpetual mastery space**.

## Ключевые свойства
- Steep learning curve
- Continuous improvement possibility
- Visible (replays, MMR)
- Cross-skill transferable (mechanics, strategy, comms)
- Esports legitimization

## Применимость к Realm
- E2 mastery: each class has multi-year progression
- Mastery dimensions multiple (technical / strategic / interpersonal)
- Visible MMR-like metric per dimension
