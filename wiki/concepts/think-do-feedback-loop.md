# Think-Do feedback loop — цикл мозга

## Суть в одной строке

Мозг переключается между «обдумывать» и «выполнять»; прогресс определяется качеством и скоростью цикла между ними.

## Определение

У мозга два базовых режима:

1. **Thinking** — моделирование, планирование, рефлексия, воображение.
2. **Doing** — действие в мире, физическое исполнение, взаимодействие.

Нет третьего. Всё остальное — комбинации этих двух. Feedback — это thinking после doing: «что получилось, что нужно изменить».

## Ключевые свойства

- **Быстрый цикл > медленный** (при прочих равных).
- **Внешний мир — источник истины**, мысли без действия не проверены.
- **Отсутствие feedback** = бесконечный thinking без прогресса.

## Применимость

- Планирование дня: короткие циклы plan → do → review.
- Product-разработка: MVP → test → iterate.
- Обучение: read → practice → reflect.
- AI: inference → tool call → observation → next action (agent loops).
