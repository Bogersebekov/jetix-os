# Learning–Knowledge–Understanding Trichotomy (O-70)

## Суть в одной строке

**Учеба = поглощение информации / Знание = информация / Понимание = знание метода.**

Trichotomy surface'нута audio_694 — расщепляет popular «knowledge ↔ understanding» dichotomy на 3 distinct phases с distinct epistemic status.

---

## Per-component

### Учеба (Learning) — active info-absorption process

**Process verb.** Active engagement: reading / listening / observing / practicing. Includes deliberate practice (Ericsson 1993) — focused, repetition, feedback-rich.

NOT identical к «студент сидит в классе»; учеба = любой info-absorption process, включая autodidactic self-study + apprenticeship.

[Ryle terminology cross-walk: «coming to know»; Polanyi: indwelling]

### Знание (Knowledge) — accumulated info

**State noun.** Результат учебы — accumulated info: facts / propositions / declarative content.

- Ryle (1949 «The Concept of Mind») — «knowing-that» = declarative knowledge.
- Anderson/Krathwohl revised Bloom's Taxonomy (2001) — Remember / Understand levels.

NOT sufficient для action; can recall facts но не know how to apply.

### Понимание (Understanding) — knowledge of method

**Procedural state.** Знание **метода** по которому accumulated info была generated + can be applied + transformed.

- Ryle: «knowing-how» = procedural knowledge.
- Polanyi (1958 «Personal Knowledge» / 1966 «Tacit Dimension») — tacit dimension не reducible к explicit propositions; «we know more than we can tell».
- Schön (1983) reflective practitioner — understanding embedded в practice.

**Понимание subsumes знание, не наоборот.** Можно знать факты без понимания, но нельзя понимать без знания.

## Pedagogical implication

Education Layer curriculum should teach **all 3**, not just knowledge transfer:

1. **Учеба-module:** how to learn (meta-cognitive skills, deliberate practice protocols)
2. **Знание-module:** content + facts + declarative scaffolding
3. **Понимание-module:** methods + procedural traces + how-to-generate-knowledge

Classical universities teach (2). Apprenticeship-style transmission teaches (3) tacitly. Jetix Education Layer = surface all 3 explicit.
