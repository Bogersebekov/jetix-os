# Jetix as road / protocol infrastructure

## Суть в одной строке

Jetix = «дорога» (рельсы / протокол) поверх которой communities, страны, и meta-workshops могут двигаться coherently без re-инвентинга coordination каждый раз.

## Определение

**Road / Protocol Infrastructure** (NC-2) = re-framing O-05 Methodology через метафору infrastructure-layer. Ruslan voiced 16.05 в audio_670:

> «Может, какой-то протокол создать, что вот как раз эта вся залупа вошла на эти рельсы и дальше уже могла развиваться. Могли развиваться там страны, бл*дь, сообщества и так далее. ... Но в целом, мне кажется, будет логичным просто вот создавать, начинать эти сообщества адекватные с Jetix ... Jetix будет как такая **дорога** для этих сообществ.» [src: audio_670 §end]

Это re-typing candidate для O-05 (Methodology / pattern language). Existing engineering type = U.MethodDescription (A.3.2). Road metaphor может предполагать U.Mechanism (A.6.1) — infrastructure substrate, не recipe.

**D-08 (open decision):** O-05 re-typing — A.3.2 vs A.6.1 FPF primitive. Ruslan decides. NC-2 surfaces вопрос; не resolves.

## Ключевые свойства

- **Macro framing** — операции на уровне сообществ / стран, не individual / org
- **Pre-installation enabling** — обеспечивает basic coordination substrate; participants затем выбирают свой use case (как road carries multiple traffic types)
- **Composes c H1-H7** — H6 Gamified Platform (Realm) = visibility layer; H7 People-NS = first community using road; NC-2 = generalisation pattern
- **Не replacement, infrastructure** — road не отменяет cars; protocol не отменяет communities — enables их movement
- **Connection к Clan Charter** — first Clan = первая community «вышедшая на рельсы» (per audio_670 «начинать сообщества адекватные с Jetix»)

## Применимость

**Когда применимо:** Описание Jetix L1 audience через метафору infrastructure (а не product / service / methodology). Особенно для аудитории привыкшей к infrastructure-thinking (например — государства, протоколы, ШСМ residents знакомые с holonic frames).

**Когда НЕ применимо:** Pre-money / quick-money commercial frame для Mittelstand-style клиентов (TRM consulting). NC-2 = Phase D vision frame, не Phase 1 sales frame.

**Risk frame** (per audio_670 affect-mode):
- «Намордник на государства» / «разодрать в клочья» = affect-mode rhetoric — НЕ literal claim. NC-2 strips affect; preserves coordination-substrate insight.
- Macro framing risk: «Jetix-as-road для стран» — F1 hyperbole без mechanism design. Phase D remit; не L1 outreach сейчас.
