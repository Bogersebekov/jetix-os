# DL Curriculum 3 Parts (Goodfellow 2016)

## Суть в одной строке

Goodfellow-Bengio-Courville Deep Learning (2016) textbook — структура 3 частей: math foundations + modern practical + DL research.

## Определение

Канонический DL textbook (2016), 3 части:

1. **Part I — Applied Math and Machine Learning Basics:** linear algebra / probability / numerical computation / ML basics
2. **Part II — Modern Practical Deep Networks:** feedforward / regularization / optimization / CNN / RNN / practical methodology
3. **Part III — Deep Learning Research:** linear factor models / autoencoders / representation learning / structured probabilistic / Monte Carlo / partition function / inference / deep generative models

## Применимость

- **Canonical DL curriculum reference** для onboarding ML engineers
- **Cross-ref Huyen 2022** для production ML (Part 2-3 overlap)
- **SOTA-tracking baseline** — paradigm shifts отслеживаются against this canon (transformer revolution post-2016, LLM emergence — paradigm shift signals per [[concepts/kuhn-paradigm-shifts]])
