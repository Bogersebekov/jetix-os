# ММК — Московский Методологический Кружок

## Суть в одной строке

Shchedrovitsky vol-17 + ММК канон — мыследеятельность (thought-activity) схема + ОД-игра (организационно-деятельностная игра) + рефлексия.

## Определение

Московский Методологический Кружок (ММК) — школа Г.П. Щедровицкого, развивавшаяся с 1950-х. Канонические элементы:

- **Мыследеятельность** — единство мысли (thought) + коммуникации + действия (схема 3 слоя)
- **ОД-игра (Организационно-Деятельностная игра)** — формат коллективной мыследеятельности на 5-10 дней, проектирующий новое
- **Рефлексия** — выход в "метапозицию" анализа собственной деятельности
- **Чистое мышление, коммуникация, мыследействование** — три слоя substrate
- **Заимствования из Маркса, Гегеля, Фихте** — диалектическая база
