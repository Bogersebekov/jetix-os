# Consent Boundary Protocol

## Суть в одной строке

Consent-boundary audit protocol applicable к cohort design / cohort recruitment / engagement-loop / training-material / sales conversation.

## Определение

Operationalization Cialdini ethical-distinction (Influence 1984) + Hassan defensive-use (1988) + Askell HHH consent dimension:

**Protocol elements:**
1. **Informed:** subject понимает, что приём применяется
2. **Voluntary:** subject может отказаться без penalty
3. **Reversible:** subject может выйти из engagement в любой момент
4. **Specific:** consent for THIS application, не blanket
5. **Renewable:** consent может быть withdrawn → engagement halts

Defaults:
- **Sales conversation:** disclosure про используемые techniques (Cialdini ethical principle)
- **Cohort recruitment:** Hassan BITE diagnostic — никакой dimension не violated
- **Engagement-loop design:** Eyal manipulation matrix → user-facing benefits должны overweight hooks
- **Training-material:** explicit declaration consent ↔ trust

- **Mandatory pair** для любого engagement / cohort / training design
- **R12 enforcement substrate** — operationalizes consent dimension R12 paired-frame
