# Cloud Empires

## Суть в одной строке
Lehdonvirta 2022: platforms (Amazon, eBay, Uber) constitute «empires» governing rules для merchants/workers — parallels с Varoufakis technofeudalism.

## Определение
Книга Cloud Empires (Lehdonvirta, MIT Press 2022) examines how platforms operate как quasi-states. Set rules, enforce them, collect tribute (fees). Critical view distinct от technofeudalism (Varoufakis) — focuses на governance, not extraction. Realm-relevant: how does Realm-as-platform govern itself?

## Ключевые свойства
- Platform as governance entity
- Rule-setting (TOS = constitution)
- Enforcement (suspension, demonetization)
- Tribute (platform fees)
- Comparison to historical empires

## Применимость к Realm
- Realm governance design (constitution-equivalent)
- Transparent rule-setting (open process)
- Appeal mechanisms (no kangaroo bans)
- Distinct from extractive technofeudalism
