# Monthly Economic Report (EVE MER)

## Суть в одной строке
CCP Games публикует quarterly экономический отчёт EVE (~50 страниц) с money supply, regional GDP, mining output, ship destruction — пример полной transparency game economy.

## Определение
EVE MER initiated в 2007 by Eyjólfur Guðmundsson (CCP Lead Economist). Включает: ISK supply / sinks-faucets / regional GDPs / commodity prices / mining output / industry production / ship destruction stats. Это **single most rigorous game economy public reporting** in gaming history.

## Ключевые свойства
- Quarterly/monthly cadence
- 30-50 страниц exhibits
- Macro stats (money supply, velocity)
- Regional breakdowns
- Production/destruction balance
- Methodology disclosure

## Применимость к Realm
- Realm Quarterly Economic Report = transparency archetype
- Quarterly: revenue distribution, knowledge token flows, clan economy size
- Methodology open-published
