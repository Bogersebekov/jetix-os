# Virtual GDP

## Суть в одной строке
Aggregate production output виртуальной экономики (Castronova: EverQuest 2001 GDP > Bulgaria), measured via hourly wage × population × hours played.

## Определение
Methodology: average hourly «wage» (price of items earned per hour) × active player base × hours active = annual «GDP». Castronova's calculation validates virtual economies как «real»: EQ 2001 ~$2,266/player annually, total ~$135M GDP.

## Ключевые свойства
- Measurable wage rate (player time → game currency)
- Active player base (denominator)
- Hours played (intensity)
- Real-money conversion possible
- Comparable across virtual worlds

## Применимость к Realm
- Realm Quarterly Economic Report: Realm GDP estimate
- Member-hour input → output mapping
- Comparable benchmarks across cohorts
