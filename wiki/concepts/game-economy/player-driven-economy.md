# Player-Driven Economy

## Суть в одной строке
Экономика где production, distribution, pricing — full player-driven, system provides только rules and sinks/faucets (EVE Online archetype).

## Определение
Player-driven economy = laissez-faire game economy. NPC vendors absent or limited. Players mine raw materials, manufacture goods, set prices, run logistics, build infrastructure. Game designer roles — central bank (issuing currency через minted activities), regulator (sinks: taxes, ship destruction; faucets: bounties, ISK rewards). EVE — extreme example; WoW Auction House — partial.

## Ключевые свойства
- Player-set prices (buy/sell orders, contracts)
- Player-built infrastructure (citadels, stations)
- Real macro-effects (inflation, bubbles)
- Designer = central banker (sinks/faucets)
- Real economist analysis legit (Guðmundsson MER)

## Применимость к Realm
- E5 anchor: clan-internal economy player-driven
- Jetix HQ = central bank (issues Knowledge tokens, taxes via marketplace fees)
- Real macro-monitoring (Realm MER quarterly)
