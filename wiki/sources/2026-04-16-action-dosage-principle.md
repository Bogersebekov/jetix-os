# Notion idea: Дозировка действий

## TL;DR

Распределение по времени ≠ сумма по времени; дозировка подбирается под цель.

## Summary

Принцип: одинаковый объём действий, разное распределение → принципиально разный эффект. Работает для тренировок, outreach, контента, обучения.

## Ключевые цитаты

> 3 тренировки за день vs 3 за неделю — одинаковый объём, принципиально разный эффект.

> Дозировка подбирается исходя из цели и что полезно для системы.
