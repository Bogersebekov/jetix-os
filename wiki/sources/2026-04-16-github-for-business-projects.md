# Notion idea: GitHub для бизнес-проектов

## TL;DR

Jetix-сообщество по модели GitHub: платформа для бизнес-проектов с курированным допуском и AI-усилителем.

## Summary

Инсайт автора: над бизнесами, стартапами, открытиями можно работать так же, как над кодом — версии, прозрачность, совместность. Предлагается собрать людей на Linux-подобной платформе, дать им GitHub-подобный инструмент для бизнес-проектов, добавить Claude как AI-усилитель и wiki-слой для накопления знаний. Ключевое ограничение — курация: попадают только отобранные. Предполагаемые эффекты: выше эффективность работы, коллективные знания, качественная среда, AI-множитель каждого участника.

## Ключевые цитаты

> Над проектами (бизнесы, стартапы, открытия, решение проблем) можно работать по тем же принципам, что и над кодом — совместно, с контролем версий, прозрачно.

> Допускать к проектам только отобранных, адекватных людей (курация).

> Похоже на: open-source сообщество, но для бизнеса. Не каждый может контрибьютить — только те, кого одобрили.
