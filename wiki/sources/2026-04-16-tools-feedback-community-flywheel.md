# Notion idea: Модель обмена — tools → feedback → community

## TL;DR

Низкий порог входа в Jetix: 2 часа бесплатной помощи с AI-инструментами → feedback → инвайт в сообщество → upsell.

## Summary

Маховик привлечения: не продаём, а помогаем. В обмен получаем живые данные (feedback), из которых формируется оффер. Конверсия — не в продажу сразу, а в членство сообщества.

## Ключевые цитаты

> Бесплатные 2 часа помощи с AI-инструментами → получить обратную связь → пригласить в сообщество → допродажа.

> Низкий порог входа.
