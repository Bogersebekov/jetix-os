### 2. Andrej Karpathy — "Intro to LLMs" + "LLM OS" talks ⭐⭐⭐

**Architectural grounding для multi-agent Jetix.**

Karpathy формулирует:
- LLM = CPU (inference engine)
- Context window = RAM (working memory)
- Tools = I/O peripherals
- File system = storage
- Vector embeddings = associative memory

**Это буквально архитектура** которую 11-12 Jetix agents реализуют.

**Откуда смотреть:**
- YouTube: "[1hr Talk] Intro to Large Language Models" — Karpathy 2023/2024
- YouTube: "LLM OS" — Karpathy talk (вероятно с State of GPT или с аналогичного venue)
- Twitter/X: @karpathy thread threads на эти темы

**Jetix connections:**
- D2 §7 AI as new electricity → LLM as CPU analogy
- D4 Foundation Layer (agent contracts / protocols / memory) ← LLM OS pattern
- D20 USB-C protocol layer — Karpathy thinks на этом abstraction level

**Что взять:**
- Mental model для architect Stage 6
- Пониманиe где Jetix parallels existing thought, где diverges
