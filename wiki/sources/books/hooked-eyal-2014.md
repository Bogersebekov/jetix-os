# Hooked

## TL;DR
Eyal 2014 product-design playbook for habit-forming products. Hook Model (Trigger → Action → Variable Reward → Investment). T1 (controversial — author wrote anti-version 2019).

## Summary
4-stage Hook Model. Case studies: Pinterest, Twitter, Instagram, etc. Ethical caveats included но subject to criticism (used by «addictive tech» industry). Eyal counter-positioned with Indistractable (2019). **Useful for product design, BUT use with ethical care.**
