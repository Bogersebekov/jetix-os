# Fortnite

## Кто / что это
Battle Royale Epic Games с 247M MAU, ставшая archetype для 3-month battle-pass seasons + IP crossovers (Marvel, Star Wars, etc.).

## Контекст
- Роль: archetype battle pass + seasonal cycle gamification
- Mining focus: Battle pass mechanic, season structure (Chapter/Season), crossovers, V-Bucks economy

## Факты
- 247M MAU (2024 Q3 Epic disclosures)
- 3-month battle pass cycles (with mid-season "live events")
- V-Bucks: 1000 V-Bucks ≈ $7.99; battle pass costs 950 V-Bucks
- Live concerts (Marshmello 10.7M concurrent, Travis Scott 27M concurrent)
- Cross-IP: Marvel, DC, Star Wars, etc. — IP licensing model
