# Minecraft

## Кто / что это
Open sandbox с 200M MAU, archetype для emergent gameplay (нет заданной цели) + Realms server economy.

## Контекст
- Роль: archetype для open-sandbox tools, redstone мини-программирование
- Mining focus: emergent gameplay, server economy (Realms hosting), educational version

## Факты
- 200M MAU (2024 Microsoft disclosures)
- Realms (managed servers) — subscription $7.99/month
- Education Edition (Microsoft Education)
- Modding ecosystem — миллионы UGC модов (Forge, Fabric)
- Acquired by Microsoft 2014 for $2.5B
