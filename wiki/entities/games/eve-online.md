# EVE Online — SPECIAL FOCUS (Economy anchor)

## Кто / что это
Sandbox space MMO CCP Games с player-run экономикой; **первая игра нанявшая реального econ PhD** (Eyjólfur Guðmundsson 2007-2014). Realm E5 (Resources) economy depth anchor.

## Контекст
- Роль: ARCHETYPE player-run economy + monthly-economic-report transparency
- Mining focus: ~12 concepts (DEEP); player-driven economy primary source
- Lehdonvirta-Castronova Virtual Economies (2014) — many EVE case studies

## Факты
- ~300K active subscribers (declining from 500K peak)
- Single shard (single server universe) — true global economy
- ISK currency, no Pay-to-Win (Plex bridges $ ↔ ISK in-game)
- Monthly Economic Report (MER) — actual macroecon disclosure pattern
- Corporation/Alliance system — nested groups
- Full-loot PvP — death has economic consequence
- Player-built (literally) markets, contracts, freight services
