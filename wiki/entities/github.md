# GitHub

## Кто / что это

Платформа для совместной разработки ПО на основе git: репозитории, pull requests, code review, issues, actions. Принадлежит Microsoft (с 2018).

## Контекст в Jetix

- **Метафорическая основа** для [[ideas/github-for-business-projects]]: те же принципы (версии, ветки, review, merge) применимы к бизнес-проектам.
- **Инструмент работы**: Jetix OS живёт в git-репозитории на GitHub (`Bogersebekov/jetix-os`).
- **Прецедент массовой коллаборации** — доказательство, что распределённая работа над сложным артефактом масштабируется.

## Факты

- Год запуска: 2008.
- Active developers: 100M+ (2024).
- Используется как social-layer для open-source (звёзды, форки, профили).

## Что заимствует Jetix

| GitHub | Jetix |
|--------|-------|
| Pull request → review → merge | Предложение → обсуждение → принятие в общую базу |
| Star / fork как сигналы | Reputation в сообществе |
| Issues как threads | Open questions в wiki |
| Actions для CI | Skills + agents для автоматизации |
