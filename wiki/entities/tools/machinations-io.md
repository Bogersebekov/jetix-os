# Machinations.io

## Кто / что это
Web-based tool для diagramming game economies (resource pools, sources, drains, converters) с built-in Monte Carlo simulation для balance testing.

## Контекст
- Роль: industry-standard tool для economy design + balance testing
- Mining focus: methodology (визуальный язык + simulation engine)

## Факты
- Founded ~2010 (Joris Dormans + team)
- Visual language: pools, sources, drains, converters, gates
- Built-in Monte Carlo simulator
- Used by major studios (EA, Ubisoft, others)
- ~60% pre-covered в Strategic Insight §6.4 (light pass only)
