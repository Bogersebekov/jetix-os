# Linux

## Кто / что это

Семейство операционных систем с открытым исходным кодом на основе ядра Linux (создано Линусом Торвальдсом, 1991). Стандарт серверного и embedded мира; основа Android, ChromeOS и многих desktop-дистрибутивов.

## Контекст в Jetix

**Философская основа** для модели Jetix. Linux показал, что:

- Открытый код + тысячи независимых контрибьюторов могут обогнать закрытые корпоративные продукты.
- Прозрачность кода создаёт доверие, особенно для security-чувствительных применений.
- Open-source даёт пользователям **цифровой суверенитет** ([[concepts/digital-sovereignty]]) — независимость от вендора.
- Платформа без gatekeeping, но с meritocratic фильтром (commiters), масштабируется.

См. [[ideas/jetix-open-source-philosophy]] — три принципа Jetix, выведенные из Linux-модели.

## Факты

- Создан: 1991, Линус Торвальдс.
- Лицензия: GPL v2.
- Доминирует на серверах: 96% top-1M веб-сайтов.
- Базис для: Android, Chromebooks, Tesla, AWS Linux, Steam Deck.

## Что заимствует Jetix

| Linux | Jetix |
|-------|-------|
| Открытое ядро | Открытые методологии, фреймворки, документация |
