# System Design — Управление проектами, философия работы

## Зачем эта тема

Hub-узел навигации по всему материалу, который описывает **как должна работать
целевая система Jetix OS** — на человеческом языке (не на тех. уровне).
Используется staging-writer'ами Шага 2 для сборки `design/SYSTEM-DESIGN-INPUTS.md`,
и затем Ruslan'ом для диктовки `SYSTEM-DESIGN-HUMAN.md`.

Объединяет 65 страниц wiki/, помеченных `topics: [system-design]`: видение, ICP,
ритуалы, философия, потоки информации, жизненные циклы. Структура hub'а
повторяет **6 частей** SYSTEM-DESIGN-HUMAN.md.
