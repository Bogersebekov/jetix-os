# L6 Business & Trajectory — где живёт revenue, capital accumulation и phase-gate логика Jetix

## 1. Миссия этого слоя

L6 — это бизнес-слой Jetix: где живут ICP, outreach, pricing, partnerships, и phase-gate прогрессия от €0 (текущее состояние, 2026-04-24) до $1T trajectory. Если L5 отвечает на вопрос «что мы продаём», то L6 отвечает на вопросы «кому», «как подходим», «за сколько», «с кем в партнёрстве», и «что разблокирует следующий уровень».

*[src:decisions/JETIX-PLAN.md#§1]* Центральный механизм — D15 revenue-gated spend: каждая фаза unlock'ает следующий уровень расходов и инфраструктуры только после достижения revenue checkpoint. Это не запрет на heavy-spend — это gate. Phase 1 Q2 2026 €50K — единственная committed absolute date в плане. Все остальные milestone — ranges и triggers.

*[src:decisions/JETIX-VISION.md#§1]* На бизнес-уровне Jetix — методология, воплощённая как consulting → productized service → platform → holding. Ни один из этих frames не исключает другие; они фазово-зависимы. Для клиента сейчас: компания / агентство (Phase 1). Для партнёра завтра: сеть / платформа (Phase 2+). Для рынка в целом: USB-C standard (Phase 3+), де-факто инфраструктура, как Windows прошивка — на каждом AI-augmented бизнесе.

## 2. ICP — кто покупает

### 2.1 Полная картина: 11 архетипов (D7, Stage 3)

*[src:decisions/JETIX-VISION.md#§7.1]* Jetix обращается к объединению 11 архетипов: Предприниматели / бизнесмены — Ресёрчеры — Инженеры — Инвесторы — Политики — Продавцы — Менеджеры / коммуникаторы — Философы — Разработчики идей — Разработчики жизни — Блогеры (Stage 3 addition, 21-04-2026). Общая нить: intelligent, driven, skin-in-game, активно развивающие себя и среду.

Блогеры — primary Phase 1 customer для English-speaking infobiz target (D10): специалисты с 5K+ подписчиков, реальной глубокой экспертизой в нише, существующими или потенциальными платными продуктами. Продюсерский центр (D11) — их прямой value-prop: research → script → footage → edit → repurpose pipeline под AI-enhanced operations.

### 2.2 ICP Qualitative Filter (D22) — 5 критериев поверх архетипов

*[src:decisions/JETIX-VISION.md#§7.2]* Любой архетип проходит через D22 quality-gate:

1. **Startupper mindset** — builder's instinct, инициатор, не исполнитель.
2. **Предпринимательский азарт** — skilled-player drive, азарт от собственной компетенции.
3. **Stable** — надёжный, дисциплинированный, не volatile.
4. **Adequate** — common sense, anti-bullshit detector, понимает реальность.
5. **Upward-direction** — вертикальная ось: вверх (развитие) vs вниз (деградация). Это primary filter.

## 4. Pricing — как берём деньги

*[src:decisions/JETIX-PLAN.md#§1, §3]* Pricing эволюция по D18 (productization path):

| Phase | Механика | Entry point | Scale mechanism |
|---|---|---|---|
| Phase 0 | €150/час consulting (solo) | Hourly session | Ручное масштабирование |
| Phase 1 | 4-pack: сессия + 10 шаблонов + chat + конкретная помощь; Продюсерский центр monthly retainer | €150/час [audio_452] | Templates / frameworks / packages |
| Phase 2+ | Secure Network subscription + Matchmaker fees + revenue-share | Membership tier | Platform subscription + network-effects |
| Phase 3+ | Methodology licensing + roy-replication fees + token economy | USB-C standard tier | Ecosystem multiplier |

*[src:decisions/STRATEGIC-INSIGHT-JETIX-AI-BIOS-MOMENT-2026-04-24.md#§5]* Path A/B/C tiered по deployment model:

- **Path A (Managed / Jetix-hosted):** managed VPS per client, EU data center, Jetix manages infra. Про: easier sale, higher margin, consistent quality. Contra: higher liability, scale burden.
- **Path B (Client-hosted / Jetix as methodology provider):** Jetix ships methodology + agent configs + templates + setup scripts. Client hosts on own infra. Про: maximum data sovereignty, lowest Jetix liability. Contra: harder sale, harder support.
- **Path C (Hybrid / Client owns data, Jetix hosts agents):** client KB/data на client infra; Jetix agent-swarm queries через secure tunnel. Про: best of both. Contra: networking complexity.

Early recommendation per §5: Path C для Enterprise, Path B для self-sufficient technical clients, Path A для low-touch SMB. Один product line с 3 tier pricing.

*[src:decisions/JETIX-PLAN.md#§3.1.1]* Revenue-mix arithmetic для €50K gate: ~€30K из productized contracts (Path-B) + ~€35K из hourly consulting (233 часа × €150/hr). Hourly — co-equal revenue line, не overflow.
