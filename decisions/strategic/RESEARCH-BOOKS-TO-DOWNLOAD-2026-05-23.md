## 📥 How to download

### Easiest sources:

1. **Library Genesis (libgen)** — `libgen.is` / `libgen.rs` — массовая бесплатная digital library; есть почти всё (PDF/EPUB)
2. **Z-Library** (`z-lib.io` или mirrors) — backup library
3. **Anna's Archive** (`annas-archive.org`) — meta-search через libgen + Z-Library
4. **Левенчук** — `system-school.ru` (paid Aisystant subscription для full ШСМ corpus) + `ailev.livejournal.com` blog (free)
5. **archive.org** — public domain (Bernays Propaganda 1928 = public domain; Le Bon 1895 = PD; Freud 1921 = PD; etc.)
6. **Google Books** — preview snippets

### Format preference

- **PDF** > EPUB > DJVU > MOBI (для server CC processing — PDF / EPUB best)
- **Russian** versions для русских книг (Левенчук + Щедровицкий + переводы)
- **English** versions для англоязычных оригиналов
- Если есть оба — оригинал предпочтительнее (lossless)
