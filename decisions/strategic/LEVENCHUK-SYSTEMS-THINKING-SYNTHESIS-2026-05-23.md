## §2 Per-book deep synthesis

### §2.1 Системное мышление 2024 (Т1+Т2)

**Thesis:** Системное мышление = transdiscipline для attention management в complex collective projects. Shared ontological vocabulary describing systems across all reality levels (physical/biological/social/engineering) с same primitives. **NOT recipe, а scaffolding для thinking.**

**3-generation framework:**
- 1st gen 1950-1970: Bertalanffy GST + Wiener cybernetics + Koestler holons
- 2nd gen 1980-2010: Checkland SSM + ISO 15288/42010 + SEMAT Essence (Jacobson)
- **3rd gen 2010-present:** Friston Active Inference + Wolfram physics + open-ended evolution + mereotopology + continuous everything

**Том 1 structure (foundations):** Three generations decomposition + Holons part-whole nesting + Emergent properties + System-in-environment ontology + Создания граф directed graph + Mereotopology grounding.

**Том 2 structure (applied):** Constructor systems + Техно-эволюция OEE NQD + Continuous everything paradigm + Mereotopological ontology + 4D extensionalism + Active Inference + practical applications.

**Unique contributions:**
1. 3rd-gen formalisation single coherent statement
2. Граф создания vs hierarchical decomposition — ecosystem recognition
3. Past-participle convention для machine-readable state descriptions
4. Physics-based ontology (отказ purely conceptual modelling)
5. Continuous everything paradigm (Method/Work distinction)

**Cross-cite Master Quals:** L3 Стратег требует системное мышление; L5+ practitioners apply systematically; L6 Мастер proposing improvements к этой материи.

**Cross-cite Jetix substrate:** Foundation Architecture 3rd-gen validation; alpha state machines для wiki/experiments/; creation-graph для Hypothesis Architecture dependency mapping.

### §2.2 Методология 2025

**Thesis:** Methodology = transdiscipline describing способы работы создателей/агентов в графах создания успешных систем (per ailev/1725757). **Largest книга в corpus (~872 pp).**

**3 method-selection approaches:**
1. **Прикладная** — domain knowledge + meta-model concrete situation
2. **Фундаментальная** — system thinking + cultural-general knowledge для unknown domains
3. **Распределённые представления** — pattern detection через ML/RL (emerging AI methodology)

**Structure:** Method as transdiscipline / Modeling engineering processes / Method engineering (Essence integration) / 250+ practices catalogue / Methods vs strategies vs practices / Method-as-artefact vs method-as-discipline / AI methodology section / Cross-disciplinary integration (semantics/ontology/rhetoric/ethics).

**Unique contributions:**
1. Most detailed Russian methodology textbook ever published
2. Method↔Practice↔Discipline distinction operationalised
3. AI methodology section recognises distributed-representations methodology emergence
4. Cross-disciplinary integration
