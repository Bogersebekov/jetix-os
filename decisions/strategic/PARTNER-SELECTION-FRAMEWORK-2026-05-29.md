# 🎯 Partner Selection Framework — первые партнёры для фундамента

> **Что это.** Каркас выбора первых партнёров: (1) **кто именно нужен** для построения фундамента — по ролям, четко; (2) **кто НЕ нужен** сейчас; (3) **рамочка первых конкретных людей** для быстрого просмотра. Потом — per-person deep work.
>
> **Pitch-логика (Ruslan voice 29.05):** «У нас ~50 потенциальных ебейших людей, которых я нашёл — реально могут помочь этому развиться. Если их объединить → может получиться что-то мега. Уже есть 2-3 человека готовых работать — чуваки, вы будете или нет?»
>
> **Цель фундамента:** технический + финансовый + юридический + стратегический базис, чтобы дальше двигаться быстро. R12 floor: co-founder/advisor tier, НЕ employee extraction; fork-and-leave preserved.

## §1 Кто нужен — роли для построения фундамента (R1 surface)

> Сгруппировано по тому что фундаменту нужно. Per founder-role-research H1-H5 + фундамент tech/fin/legal. Роли abstract (IP-1); конкретные люди — §3.

| Роль | Что закрывает | Почему нужен | Приоритет |
|---|---|---|---|
| **R-1 Технический co-founder / lead engineer** | Stack / infra / multi-agent / pipeline / AI-ML (Build кластер B1-B3) | Ruslan = strategist, не хочет coding burden; tech = главный time-sink сейчас | ⭐⭐⭐ |
| **R-2 Методологический партнёр** | Co-author метода / системное мышление / FPF (A2 support) | Ruslan solo на methodology; нужен peer-level методолог | ⭐⭐⭐ |
| **R-3 Финансово-юридический advisor** | Финансы / юр-структура / fundraising / корпоративка (D2) | Фундамент tech+fin+legal; Ruslan не expert в юр/фин | ⭐⭐⭐ |
| **R-4 Стратегический партнёр / советник** | Стратегическое управление / построение «стратегической компании из стратегических людей» (O-254) | Ruslan хочет peer для стратегии; ускоряет фундамент | ⭐⭐ |
| **R-5 Operations / delegation lead** | Ops / data / делегирование рутины (D1+D3) | O-253 «всё что можно отдать — отдать»; рутина не founder | ⭐⭐ |
| **R-6 Content / community amplifier** | Сообщество / контент / events (C2+C3) | Founder-as-Exhibit нужен amplifier (Ruslan-face core, остальное delegate) | ⭐ |

**R12 для всех:** co-founder / advisor tier (share, не extraction); fork-and-leave preserved; Mondragón 5:1 internal; cash-not-equity для advisors на старте где уместно.

## §2 Кто НЕ нужен (сейчас) — четко

> Чтобы не распыляться + не тащить wrong-fit людей в фундамент.

| НЕ нужен | Почему |
|---|---|
| **Чистые исполнители рутины** | Рутина → AI / delegation позже (O-253); не фундамент |
| **Sales / продажники** | НЕ продаём сейчас (цель — фундамент + партнёры, не выручка) |
| **Инвесторы с extraction-условиями** | R12 violation (anti-extraction floor) — деньги да, доение нет |
| **«Звёзды» без alignment с ценностями** | R12 + триада alignment > raw status; wrong-fit ломает floor |
| **Люди которым нужен готовый продукт** | Сейчас building-stage; нужны со-строители, не потребители |
| **Те кто хочет lock-in / контроль** | Противоречит fork-and-leave + mesh-не-star архитектуре |
