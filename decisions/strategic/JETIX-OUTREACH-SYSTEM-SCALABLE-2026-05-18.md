## §3 10-person video team → 100 trained → personalized (pattern formalized)

### §3.1 Stage 1 — 10-person video team (target 60 days)

**Composition:**
- Video producers (2) — production + editing.
- Outreach copywriters (2) — script + personalization.
- Video on-screen talent (2) — recording.
- Researchers (2) — target identification + warm-link discovery.
- Tools / CRM operators (2) — pipeline execution.

**Output:** baseline outreach video library (50-100 templates по target classes).

### §3.2 Stage 2 — 100 trained operators (target Q4 2026)

**Process:**
- 10-team trains 100 cohort members.
- TPS mentor-pairing pattern (per research/deepening-2026-05-18/14).
- Per-trainee specialization (target class subject-matter expertise).
- Quality predicate: trainee can produce personalized outreach within ≤4h per target.

### §3.3 Stage 3 — Personalized pattern (target Q1 2027)

**Process:**
- Each operator covers ~30 targets/month.
- Personalization layer: target research → script tailoring → personalized video → followup pipeline.
- CRM integration (crm/ system spec); status pipeline tracking.

**Mathematical scale:** 100 operators × 30 targets/month × 12 months = 36,000 personalized engagements/year. Subset → cohort entry.

- Modern sales pipeline platforms: cadence sequencing + personalization layer + CRM integration.

- Cohort-based recruitment + personalized follow-up.
- **Lesson:** cohort entry-point + personalized retention works at YC scale (~10K/year applicants).

### §6.3 Public intellectual outreach (Karpathy / Musk patterns)
- Twitter + blog + cohort engagement; personalized response on selective basis.
- **Lesson:** key-person engagement = highly selective; personalized signal required for response.
