## Уточнение Overview layering (Ruslan voice 29.05)

Overview подаётся **слоями**:

1. **Базовый слой — методология работы с информацией.** Всё = информация + методы её обработки. Интеллект (человек / система) **пытается развиваться** и использует **лучшие методы для собственного развития**.
2. **Накладывается 3P** (продукты / процессы / проекты) — это и есть информация + методы работы с ней в применении.
3. **Дальше суть Jetix:** мастерская + сеть + возможности.
4. **Что на текущий момент нужно** (фундамент / помощь / партнёры).

То есть 3P НЕ сам по себе — он накладывается на базовую методологию «информация + методы обработки → саморазвитие интеллекта».
