## 2. One-paragraph description

Jetix — это **методология плюс operating system**, а не продукт и не агентство. Основной актив — как AI применяется (метод), не что при этом делается (услуга). Адресовано 10 архетипам профессионалов (предпринимателей, ресёрчеров, инженеров, инвесторов, политиков, продавцов, менеджеров, философов, разработчиков идей, разработчиков жизни) — всех, кто направляется вверх (развитие себя и среды), не вниз (деградация, тупёж). Jetix достигает cascading-leverage на пяти уровнях одновременно: жизнь → проекты → знания → мышление → технологии. Отличается тем, что работает против attention-economy (substance, не reach) и строит закрытое ядро при открытой поверхности (методология защищена, результаты и фреймы показываются). Структурно — solo-сейчас с team-ready scaffolding, Phase 1 консалтинг-first (€50K Q2 2026), Phase 2+ Secure Network, Phase 3+ холдинг на траектории $100M → $100B → $1T.

## 4. Core Identity — Jetix = Methodology + Operating System

**Центральный тезис:** Jetix не продукт. Не компания в обычном смысле. Jetix — **методология** (как делать), которая воплощается в **operating system** (где и чем делать). Это два уровня единого identity, не два отдельных объекта.

Методология — способ превращения AI-рычага в cascading-leverage. Operating system — concrete stack инструментов (wiki, agents, pipelines, контракты, dashboards, протоколы handoff), через которые методология исполняется. Analogy: Apple не софт без харда, хард без софта — тоже хуйня; у Jetix «хард» — люди + agents + протоколы [src:audio_502].

### Decision 8 — Layered Identity (LOCKED)

Jetix — один объект с разными лицами, зависимо от observer и фазы. Это не противоречие, это **грамматика** — одна ментальная ткань, натянутая на разные социальные контексты.

| Для кого | Frame | Когда доминирует |
|---|---|---|
| Для Ruslan (self-inside) | Methodology + Operating System | всегда (base) |
| Для клиентов | Company / Agency (consulting) | Phase 1 |
| Для партнёров | Network / Platform (Secure Network) | Phase 2+ |
| Для community members | Club (Kingsman inside) | всегда |
| Для external world | Corporation («организация имеет вес») | Phase 2+ |
| Для Phase 3+ observer | USB-C universal connector + Holding | Phase 3+ |
| Для Phase 4 horizon | Civilizational infrastructure (200y) | Phase 3+ → 200y |
| Для команды сейчас | Solo-with-team-ready founder | Phase 1 |

### Grammar insight — inside/outside patterns как core design pattern

Decisions 1 (Gentleman inside / Predator outside), 3 (Open inside / Closed outside), 8 (Layered identity), 9 (Opportunity inside / Pain outside), 12 (Deep inside / Hooks outside), 13 (Closed core / Open surface) — **одна ментальная грамматика**, не набор случайных решений.

Для **inside** (member, partner, team, ядро методологии): gentleman, open, deep, cooperative, civil, substantive.
Для **outside** (world, competitors, TOF audience, external observers): predator, closed, hooks, defensive, aggressive, teasing.

Мембрана между inside и outside — quality-gate (invite / subscription / self-selection through framing / skin-in-game) + ICP-filter (Decision 22). Кто проходит мембрану — получает gentleman mode. Кто не проходит — видит только predator face + open surface.

Это не двуличие. Это **context-adaptive identity**: идентичность-как-грамматика, не идентичность-как-монолит. Identity у Jetix — функция контекста, как у ножа: для хлеба — кухонный, в драке — оружие, один объект.

## 5. What Jetix Does — Phase-by-Phase Offerings

### Current (Phase 1, до €50K Q2 2026)

- **Consulting** (€150/час baseline, но scale через productization per D18) — AI-implementation для smart entrepreneurs + English-speaking infobiz.
- **Продюсерский центр** (Decision 11 core Phase 1) — AI-produced content-production для English-speaking bloggers/educators/consultants: research → script → footage → edit → repurpose pipeline, monthly retainer, не часовой рейт.
- **Offer 4-pack** (Note 1): сессия / 10 шаблонов / сообщество попутно / услуги конкретные. Entry-point → lead-magnet → optional-upsell → core-billable ladder.
- **Простой community-chat** (Decision 16 Phase 1) — Telegram/Discord группа, invite-based, 5-20 знакомых face-to-face, без formal anti-free-riding механик.

### Emerging (Phase 2+, post €200K validation)

- **Secure Network** (Decision 5 evolution) — quality-gated платформа для умных, adequate, sильных, заряженных. Не LinkedIn для «рабов, ищущих других рабов» [Note 2], а место глубокой кооперации.
- **Thematic sub-networks** — политики общаются между собой, бизнесмены между собой, философы между собой, и так далее. Parent platform + sub-channels per archetype/topic [Note 2].
- **Tool-sharing + складчина layer** — Perplexity-max, Claude Code-на-максималках, expensive research tools — pool через membership. Плюс expertise-sharing: последние наработки в AI, экспертиза всех экспертов вместе взятых [Note 2].
- **Partnership-Matchmaker mechanic** (Decision 21) — Jetix матчит complex tasks с specialists, AI-agents смазывают coordination, качественно/быстро/адекватно фиксируются контракты и touch-points. Network effects: накапливаются контакты, specialists постепенно transition внутрь Jetix.
- **Token economy exploration** (Decision 23, trigger $100K self-earned) — internal utility Phase 2 (specialist compensation, складчина tokens), limited public Phase 3+ (alternative to IPO). Design safe + adequate + thoughtful, explicitly не crypto-pump style.

### Long-term (Phase 3+, €1M+)

- **USB-C-level universal connector** (Decision 20) — Jetix existing на уровне USB-C в electronics. Универсальный протокол подключения AI-агентов, бизнесов, specialists. Не монополия (USB-C работает с любым vendor), а standards-level interoperability. Развитие протоколов, законов, инструментов верификации AI-to-AI контрактов.
- **Holding structure** (Decision 19) — компания AND холдинг одновременно. *«наши агенты друг с другом коммуницируют, мы даём друг другу какие-то наработки, и мы в безопасности, если что, у одного какая-то хуйня, быстро даются бабки — эффект ROI в крупных масштабах»* [Note 5].
- **Roy-replication methodology** (Decision 21) — Jetix сперва строит свой рой из 10 людей, зарабатывающих $10M-$100M; затем распаковывает методологию как инструмент/систему; затем другие рои emerge в других вертикалях/нишах/странах. Jetix meta-coordinates ecosystem of swarms.
- **Open-source research direction** (Decision 24) — как люди коммуницируют, как компании кооперируют, как должна работать будущая экономика. Открыто для исследователей, но Jetix использует первым (first-mover data + thought-leadership). Core методология остаётся closed (per D13).
- **Civilizational infrastructure** — bet на 200-year horizon, $1T market-cap trajectory (Decision 19). *«Jetix настолько будет масштабным, мощным… что это будет сопоставимо, сука, с странами какими-то»* [src:audio_496].

### Scale frame

Jetix — **enterprise-fast corporation**, не solo, не boutique, не slow-corporate. *«нет у нас тут большая компания с мощными дядями ребятами у нас enterprise но только enterprise который довольно быстрый но при этом прочные адекватные»* [Note 3].

Target trajectory: €50K Phase 1 → €200K validation → $100K self-earned trigger расширения → $100M revenue → $100B revenue/market cap → **$1T market cap world-record attempt**. XEI as precedent to beat (2 года до $100B market cap).

## 6. What Jetix Does NOT

Чёткие анти-рамки важны не меньше рамок. Вот чего Jetix **НЕ** делает, явно и без оговорок:

- **НЕ mass-market attention economy.** Anti-TikTok / Instagram / YouTube / LinkedIn стойка [Community-Addendum §3.2]. *«это тоже не linkedin где блять быдло одно которое ищет каких-то рабов»* [Note 2]. Attention сохраняется, не экстрагируется.
- **НЕ hour-billing-only consulting trap.** D18 productization path — часы = entry, масштаб через шаблоны/фреймворки/пакеты, не через увеличение часов.
- **НЕ enterprise C-Level focus в Phase 1.** Hard Rule: Phase 1 ICP — smart entrepreneurs + English-speaking infobiz, не Fortune-500 закупщики (они активируются Phase 3+).
- **НЕ equity-based investor pitches в Phase 1.** Skin-in-game (self-funded $20-30K earned) + D15 revenue-gated unlocks. Инвесторы приходят в Phase 2+ с validation.
- **НЕ pure open-source platform.** D3 closed outside / D13 closed core. Open — только для inside + research-layer Phase 2+ (D24).
- **НЕ pure think-tank / philosophical retreat.** Revenue + action первичны (D14 research = revenue-instrumental Phase 1). Философия и research-direction — Phase 2+ только.
- **НЕ SaaS / продукт-tech-startup narrative.** D8 methodology, не product. Jetix не строит «ещё один инструмент» — строит способ применения инструментов.
- **НЕ research ради «великих дел» в Phase 1.** D14 revenue-instrumental only. 200-year vision informs arhitecture decisions, но Phase 1 execution — только то, что serves near-term revenue.
- **НЕ One-Person-Company / One-Million-Company aesthetics.** *«я не конкурирую с этим… One Million Company ещё что-то, нет. Или One Person Company. нет у нас тут большая компания»* [Note 3]. Enterprise tier, не solopreneur.
- **НЕ pyramid / scam / speculative hype.** *«Это все должно быть не какая-то там ебаная пирамида или еще что-то. Нет. А реально адекватные инструменты, наработки, технологий»* [Note 5]. $1T bet грунтуется в engineering-faith, не в affirmations.

## 7. Target Audience — Union of 10 Archetypes + ICP Qualitative Filter

### 7.1 — 10 архетипов (Decision 7 union)

Jetix не продаёт кому попало. Только к 10 архетипам, merged из двух списков Ruslan'а:

1. **Предприниматели / бизнесмены** (merged: digest audio_495 ≈ Note 2) — hungry, action-driven, ищут AI-leverage для scale.
2. **Ресёрчеры** — depth-of-analysis seekers, ищут structured thinking + AI-enhancement + Phase 2+ open-source research direction participation (D24).
3. **Инженеры** — system builders, ищут productivity multiplier + architectural insight + USB-C protocol development (Phase 2+).
4. **Инвесторы** — pattern seekers, ищут smart bets + resource-allocation frameworks + Phase 2+ token economy + Phase 3+ $1T trajectory participation.
5. **Политики** — influence operators, ищут strategic communication leverage + Phase 3+ institutional weight platform.
6. **Продавцы** — conversion specialists, ищут AI-powered sales infrastructure + methodology-as-scalable-asset.
7. **Менеджеры / коммуникаторы** — orchestrators, ищут team/process effectiveness + governance frameworks + Phase 2+ roy-coordination tools.
8. **Философы** — meaning seekers, ищут operational wisdom infrastructure + Phase 2+ open-source research-direction contribution channel.
9. **Разработчики идей** — concept generators, ищут leverage для validation + execution через AI-augmented prototyping.
10. **Разработчики жизни** — life-designers, ищут holistic life-system tooling + engineering-approach к собственной жизни.

**Common thread across all 10:** intelligent, driven, skin-in-game, НЕ passive consumers. *«не на широкие массы а конкретно на профессионалов на вот заинтересованных людей в своем развитии в развитии своего сообщества»* [Note 2].

### 7.2 — ICP Qualitative Filter (Decision 22)

Ruslan explicitly directed: *«это реально на бумагу прям вынести прям несколько страниц как минимум»* [Note 1]. ICP filter — это не optional bullet list, это **core membership qualification layer** поверх 10 архетипов.

**5 качественных критериев (layered on top of 10 archetypes):**

1. **Startupper mindset** — builder's instinct, proactive, creates rather than consumes. Не наёмник-исполнитель, а инициатор. Человек, который видит задачу и сам запускается к её решению, без внешнего push.
2. **Предпринимательский азарт** — entrepreneurial game-drive, enjoys skill-based risk. Не gambler (у которого исход зависит от случая), а skilled player — азарт от игры, где ставка — собственная компетенция. Игра в серьёзное дело как форма энергии.
3. **Stable** — reliable, disciplined, не volatile. Может выдержать длительную траекторию без срывов, mood-swings, burnout-spirals. Без этого качества все остальные обесцениваются — unstable человек с highest startupper mindset просто сгорает за 6 месяцев.
4. **Adequate** — common sense, no delusion. Understands реальность + как она работает. Не строит замков на облаках, не верит в «секретные техники успеха», не отрицает physics / economics / human nature. Adequate — это anti-bullshit detector, направленный на собственные самообманы в первую очередь.
5. **Upward-direction** — actively develops self + среду. Vertical axis: вверх (развитие) vs вниз (degradation, тупёж, «towards death»).

### Direction-of-life axis (NEW dimension beyond 10 архетипов)

Ruslan explicit distinction:
> *«прикол в том что темы это не по темам а по направлению жизни короче вот в чем штука что темы они как-то расходятся… вправо влево получается по горизонтали а по вертикали это вверх вниз нахуй это направление вверх направление к жизни или направление вниз там к смерти блять тупежу»* [Note 2]

- **Horizontal axis (topics):** бас / фотография / финансы / бизнес / research / и так далее. Темы расходятся вправо-влево. Может быть either up OR down direction — сам topic не предопределяет вектор жизни.
- **Vertical axis (direction-of-life):** up (развитие себя + среды) OR down (degradation, тупёж, «towards death»). Это primary axis для Jetix community membership.

**Jetix community = ВСЕ 10 archetypes × UPWARD-DIRECTION ONLY.** Topic — secondary, direction — primary.

### NOT target:

- Мотивационный circle (Tony Robbins passive audience).
- Философский retreat (passive reflection без action).
- Mass-market consumer base (*«не oriflame»* [Note 2]).
- Superficial networking (LinkedIn-style transactional collisions).
- Attention-economy consumers.

### IS target:

> *«нормальная для чуваки собрались там инструменты у них нормальная запросу на жизнь тоже нормальная ну как бы амбиции адекватные там этот инструмент и все при себе нахуй работают отлично»* [Note 2]

Adequate professionals/entrepreneurs cooperating среди своих, инструменты в руках, амбиции адекватные, direction вверх, работают отлично. Не glamorous, не academic, не corporate — **craftsman + professional blend.**

### Self-selection через framing

*«Самая большая авантюра века»* invitation (Decision 20 aligned) — self-filtering механика. Только человек, self-identifying как авантюрист-с-навыками, отвечает на такой призыв. Passive consumer проходит мимо — им не интересно. Вызов громкий, но adressant узкий. Это predator-outside tool: провокативный, но работает как мембрана, не как ловушка.
