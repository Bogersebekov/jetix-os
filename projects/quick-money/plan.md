# План — Запуск бизнеса (v2 — 04.04.2026)

> **Принцип:** Сырой материал собираем сами (AI + источники) → калибруем с экспертами → фиксируем финальные документы → воплощаем

## Этапы

### Фаза 1: Очистка сырья + вопросы для экспертов 🟢 В РАБОТЕ
**Суть:** Сырьё уже есть (заметки, идеи, записи, анализ Gemini). Задача — очистить, выбрать основные идеи по каждой секции, собрать лучшие варианты. Не утверждать, а подготовить качественный материал для калибровки с экспертами. Параллельно по ходу фиксируем вопросы для экспертов.
**Результат:** По каждой секции — очищенные варианты + список вопросов для бренд-стратега и B2B Sales эксперта.
- [x] 1.1 Миссия — утверждена ✅
- [ ] 1.2 Видение (3-5 лет) — варианты готовы, очистка и выбор
- [ ] 1.3 Большая идея
- [ ] 1.4 Ценности и принципы
- [ ] 1.5 Позиционирование и мастер-оффер
- [x] 1.6 Обещание бренда — сырьё собрано ✅
- [x] 1.7 Тон коммуникации — сырьё собрано ✅
- [ ] 2.1-2.3 Личное позиционирование Руслана (черновик)
- [ ] Список вопросов для бренд-стратега
- [ ] Список вопросов для B2B Sales эксперта

### Фаза 2: Найти и провести консультации ⭕ ОЖИДАЕТ
**Что:** Найти экспертов (LinkedIn, ADPList, Intro.co, стартап-комьюнити Берлина). Провести 1-2 сессии по 60 мин с каждым. Записать.
**Бюджет:** 100-300€ за сессию (итого до ~€1200 на 4 сессии)
**Результат:** Записи консультаций, экспертная обратная связь, конкретные рекомендации.
**После консультаций:** Обработать транскрипты → извлечь фреймворки, рекомендации, принципы → создать AI-роли в Claude Projects (виртуальный C-Suite с экспертной калибровкой)

### Фаза 3: Финализация бренд-стратегии ⭕ ОЖИДАЕТ
**Что:** На основе очищенного сырья (Фаза 1) + экспертной калибровки (Фаза 2) — зафиксировать официальную бренд-стратегию.
**Результат:** Финальный Company Hub + брендбук v1:
- Company Hub с утверждёнными формулировками (миссия, видение, ценности, позиционирование, тон)
- Личное позиционирование Руслана
- Визуальный стиль (цвета, вайб)

### Фаза 4: ICP + Оффер + Стратегия продаж (сырьё) ⭕ ОЖИДАЕТ
**Что:** Только после бренд-стратегии — детально описать ICP, оффер, цепочки касаний, методы продвижения. Собрать сырой материал самостоятельно.
**Результат:** Черновые варианты оффера, плана продаж, цепочек касаний, конкретных примеров компаний-клиентов.

### Фаза 5: Калибровка продаж с экспертом ⭕ ОЖИДАЕТ
**Что:** Обсудить сырой план продаж с B2B Sales экспертом (или наставником). Какие методы привлечения работают, как касаться клиентов, что реально работает в консалтинговых продажах.
**Результат:** Финальная стратегия продаж, GTM-план, конкретные шаги на первые 2-4 недели.

### Фаза 6: Реализация и первые продажи ⭕ ОЖИДАЕТ
**Что:** Воплощать: outreach, первые контакты, discovery-звонки, бесплатные аудиты, первый pilot.
**Результат:** Первый клиент, первые деньги, первый кейс.
**Рычаги ускорения:** Trojan Horse (бесплатный аудит), CEO на работе, предприниматели с Буки, LinkedIn outreach, AI-сообщества.

## Воронка продуктов Jetix

```
Training (€500-1500, lead magnet) → Assessment (€2-8K) → Pilot (€15-50K) → Retainer (€2-5K/мес)
```

## Зависимости
- Фаза 2 зависит от завершения Фазы 1 (нужны вопросы для экспертов)
- Фаза 3 зависит от Фаз 1+2
- Фаза 4 зависит от Фазы 3 (бренд-стратегия определяет позиционирование для продаж)
- Фаза 5 зависит от Фазы 4
- Фаза 6 зависит от Фазы 5

## Риски
- Затянуть на фазе сырья и не выйти на контакт с рынком
- Бюджет на консультации (до €1200) — нужно контролировать ROI
- Совмещение с наймом — ограниченное время
