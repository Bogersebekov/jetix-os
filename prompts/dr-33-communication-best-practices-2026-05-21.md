## §4 Phase 3 — FPF-vs-natural language analysis (1h)

**Output:** `research/communication-best-practices-2026-05-21/03-fpf-vs-natural.md`

### Content (~1500w)

- **FPF strengths:**
  - Explicit F-G-R triple → epistemic clarity
  - F-grade ladder → claims gradated by confidence
  - Group-scope → audience disambiguation
  - Reliability tracking → calibrated belief

- **FPF weaknesses:**
  - Cognitive overhead для recipient (need to learn syntax)
  - Slower initial parsing
  - Risk of «academic» perception in non-technical contexts
  - Curse of knowledge — author over-estimates recipient FPF literacy

- **Natural language strengths:**
  - Low cognitive overhead
  - Story-friendly
  - Emotional resonance accessible
  - System 1 directly engageable

- **Natural language weaknesses:**
  - Ambiguity / equivocation
  - Confidence inflation (no F-grade explicit)
  - Group-scope unclear
  - Hard to track epistemic state

- **Hybrid approach (recommended):**
  - Natural language surface
  - FPF annotations selective (footnotes / brackets)
  - F-grade explicit только для contested claims
  - Group-scope explicit для tier-specific claims

- **Universal language thesis testing:**
  - Hypothesis: FPF allows 30-60 min conveyance of colossal idea
  - Falsifiability conditions
  - Test design (audience × time-budget × comprehension)

### Content (~800-1000w)

**Channel 1: Written one-pager / brief**
- Pros: scannable / shareable / async / control over framing
- Cons: requires read commitment / passive
- Best for: L1 engineer / L3 institutional / archival

**Channel 2: Video (recorded / live)**
- Pros: emotional resonance / personality conveyance / time-flexible
- Cons: production overhead / less searchable
- Best for: L2 amplifier / humanitarian / RU community

**Channel 3: Podcast appearance**
- Pros: long-form / conversational / audience pre-built
- Cons: dependent on host / public scrutiny
- Best for: L1 engineer (Lex Fridman tier) / L2 amplifier

**Channel 4: 1-on-1 conversation**
- Pros: highest fidelity / iterative / R12 paired-frame easy
- Cons: doesn't scale / single recipient
- Best for: first-cohort partner discovery / Дмитрий / Левенчук pitch

**Channel 5: Async DM (Telegram / email)**
- Pros: low pressure / async / preserves context
- Cons: easy to ignore / risk of missing
- Best for: warm-touch initial / weekly cadence / follow-ups

**Channel 6: Slide deck / pitch deck**
- Pros: visual / structured / partial automation
- Cons: requires pitch context / linear consumption
- Best for: L3 institutional / formal meetings

**Channel 7: Workshop / talk (live presentation)**
- Pros: high engagement / Q&A / community building
- Cons: prep overhead / one-shot
