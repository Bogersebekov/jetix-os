### NEW directions (3 — добавляются к 11 v2):

**12. 🏛️ Концепция Мастерской** — что такое «мастерская» как пространство:
- Все мастера собираются
- Разные инструменты доступны (можно улучшать, ставить новый «станок»)
- Исследовательский центр (создать, изобрести)
- Можно встретить любого человека / попасть в мега-клуб / познакомиться с интересными людьми
- Побыть в роли инвестора / бизнес-партнёра / консультанта
- Сделать что-то интересное, нарабатывать мастерство
- Тренировочный зал + спортивный зал
- Место для медитации, йоги
- Всё вместе — что только можно
- Сначала **виртуально/онлайн**, потом быстро **оффлайн физические места**

**13. 🎯 Концепция Мастерства** — что такое «мастерство»:
- Мастерство = накопление методов = накопление информации + правильное использование в нужный момент
- Концепция информации накладывается (всё = информация + методы переработки)
- Информацию быстро сохранять и перерабатывать
- Мастерство = **выбирать в нужный момент нужный навык**
- Вечно тренируешь, вечно прокачиваешь
- AI делает рутину, люди занимаются прокачкой мастерства
- Решение сложных задач, соревнования, создание сообществ/городов/стран

**14. 🌍 Сеть Мастерских (Online → Offline)** — как мастерская становится распределённой сетью:
- Разные точки мира (multi-location)
- Multi-language (основной английский, локальные тоже)
- Сотрудничество между точками
- Передвижение людей между точками
- Открытие больших проектов
- Роли: консультанты / бизнес-партнёры / инвесторы / мастера
- Одна сеть с лучшими наработками
- Resources pooling: знакомства / склады / помещения / хаты / машины / гаражи / бизнес-идеи / деньги / информация — всё в кучу
- Активности: походы, исследования в разные ниши/направления/города, поездки на учёбу/соревнования, школа, совместный отдых, путешествия
- Direction: жизнь / развитие / голод / тренированность / дисциплина (поддерживается, не enforced)
