## §2 Per-direction FULL document portfolio template

> Per каждое из 14 directions — **полный портфель minimum 8-12 artefacts**. Per artefact: spec (НЕ sample content).

### Portfolio template (применяется к каждой direction)

#### A. Главный документ (executive summary / one-pager)
- 1-pager (≤500 words) для quick share
- Long-form article (≤5K words) для deep read

#### B. Видео (объяснение на пальцах)
- 5-15 мин YouTube
- Hook 30 сек + structure 3-5 scenes + closing
- Audience target
- Substrate sources

#### C. Курс (если applicable)
- Modules / lessons skeleton (≤10 modules, ≤5 lessons each)
- Per module: outcome + materials + exercise + assessment
- Format: Notion-based + video + reading list
- Cohort delivery format

#### D. Книга (если applicable — long-arc reading)
- Chapters skeleton (≤20 chapters)
- Per chapter: thesis + 3-5 sub-points + worked examples + references
- Format: Markdown source → PDF / EPUB output
- Audience: deep dive

#### E. Инструкции / SOP (operational how-to)
- Step-by-step procedure
- Decision trees
- Anti-patterns
- Troubleshooting
- Format: Markdown + Notion DB

#### F. Шаблоны / Templates
- Notion DB schemas (если data-tracking)
- Markdown templates (если documents-creation)
- Spreadsheet templates (если financial / planning)
- Format-agnostic templates lists

#### G. Презентация (slides — для face-to-face / talks)
- 10-20 slides skeleton
- Per slide: headline + 3 bullet points + visual concept
- Audience: live presentation / conferences / partner pitch
- Format: PDF / Notion / Canva / Figma

#### H. FAQ (10-30 anticipated questions)
- Real questions из discovery calls
- Honest answers (no marketing fluff)
- Cross-link к глубокому материалу
- Format: Markdown / Notion page

#### I. Worked examples / Case studies (vivid scenarios)
- 3-5 concrete examples per direction
- Per example: context + что сделали + outcome + lessons
- RUSLAN-LAYER examples (Maxim / Дмитрий / Прапион / Левенчук — per IP-1)
- Format: long-form prose

#### J. R12 paired-frame check (per direction)
- 8 paired-frame questions explicitly applied
- AUTO-FIRE influence-ethics + recruitment-dynamics + nlp + propaganda + gamification per applicable
- Anti-patterns spec

#### K. AI tooling integration (per direction)
- Что AI делает в этой direction (preparation / synthesis / routine)
- Что человек делает (mastery moments / decisions / transitions)
- Готовые prompts / scripts / workflows
- Per O-182 stratification + Preparation Stage Extended Meta-Method

#### L. Partner-extension hook (как партнёры добавляют свои artefacts)
- Fork protocol (как взять direction + adapt под свой business / cohort / personal use)
- Extension points (где можно add без ломать core)
- Contribution guidelines (что accept в community library / что нет)
