## §3 Phase 2 — «Чему обучаем» plain language

**Source:** Outreach Content Phase 1 (7+3 universal principles) + RUSLAN NOTES O-176..O-185 + Method V2 §J meta-method.

**KEY message:** мы НЕ учим конкретному tool / framework / app. Мы учим **способу думать + работать** который потом применяется к любому tool.

### §3.A Что такое «cross-skill» (на пальцах)

«Cross-skill» = не конкретный навык (как блять Notion использовать или как воронку строить).

**А способ выбирать + комбинировать методы для конкретной задачи.**

Аналог: разница между:
- «Учить какой-то один молоток» (skill) vs
- «Учить как выбирать правильный инструмент для каждой задачи + комбинировать» (cross-skill / approach)

### §3.B 7 принципов нашего подхода (plain prose)

Каждый принцип = 1 абзац human language + 1 пример из жизни:

1. **Всё = информация + методы переработки** — пример: твой день = поток информации (письма, разговоры, задачи) + методы обработки (фильтрация, приоритизация, делегирование). Качество жизни = качество методов.

2. **Метод выбора методов = leverage** — вместо учить 100 методов отдельно, учить **как выбирать** между ними. Это даёт x10-100 leverage потому что новые методы integrate'ятся быстро.

3. **AI = усилитель внешний, а не замена** — AI делает рутину (вычисления, поиск, обобщение); ты освобождаешь время для frontier (сложные решения, новое, контекстное).

4. **Сначала вопрос, потом функция** — НЕ «я учу sales / SEO / etc.» (function-first), а «у меня вопрос X / гипотеза Y / задача Z — какие методы применить» (question-first).

5. **Адекватный интеллект выбирает методы которые его же развивают** — НЕ shortcuts чтобы быстро срубить и шиllить. ДА — методы которые после применения **тебя самого** делают сильнее.

6. **Системное мышление + ответственность + инженерный подход = baseline прошивка** — это то что человек ставит себе сам перед чем-либо. Без этого learning не работает.

7. **Вклад на frontier** — после освобождения времени (через AI rutина) — вклад в науку / технологии / clan / community.

### §3.C Что человек получает в результате

После обучения человек умеет:
- Видеть свой день как переработку информации
- Выбирать метод под контекст (не applying один любимый)
- Использовать AI стратегически (где rutина / где frontier)
- Спрашивать «зачем» прежде «как»
- Audit'ить свой intellectual leverage честно
- Применять системное мышление к любой задаче

## §4 Phase 3 — «Какую информацию + порядок» plain language

**Source:** Outreach Content Phase 2 (Bloom Stage 1-7) + Phase 5 (narrative sequence + 18 artifacts).

### §4.A 7 ступенек обучения (plain language)

Каждая ступень = 1 абзац human language:

1. **Сначала узнаешь** (5-15 минут) — basic thesis «всё = информация + методы». Видео / 1-pager.

2. **Понимаешь** (1-3 часа) — explain to your own context. Прочитал Method V2 deep / посмотрел Navigation Guide.

3. **Применяешь** (5-10 часов / 1 неделя) — apply 1-2 principles daily. Question-first для каждой задачи / AI stratification для своего workflow.

4. **Анализируешь** (30-60 мин call + 2-5 часов prep) — diagnose свои current methodology gaps. Discovery call с нами.

5. **Оцениваешь** (1-4 weeks) — audit твой intellectual leverage. Trial — попробовать Notion template + Claude Code subscription / Workshop intro.

6. **Создаёшь** (3-12 месяцев) — co-create new methodology applications. Cohort partnership / L4-L5-L6.

7. **Передаёшь** (ongoing) — teach others (Feynman test). Authoring / mentorship / mass advocate.

### §4.B Что мы даём на каждой ступени

| Ступень | Что человек получает |
|---|---|
| 1 Узнаёшь | Видео + 1-pager + welcome message |
| 2 Понимаешь | Navigation Guide + Method V2 distilled + FAQ |
| 3 Применяешь | Practice exercises + pre-task pause + self-audit checklist |
| 4 Анализируешь | Discovery call script + pre-call questionnaire + gap inventory |
| 5 Оцениваешь | Notion template + Claude Code setup + trial measurement |
| 6 Создаёшь | Charter draft + onboarding kit + cohort coordination |
| 7 Передаёшь | Advocate materials + co-creation channels + R12 ethics doc |

### §4.C Можно остановиться на любой ступени без штрафа

(Это R12 на человеческом — "fork-and-leave preserved everywhere")

Никаких обязательств / lock-in. Хочешь — берёшь только Ступень 1-2 (бесплатно). Хочешь — идёшь до 7. **Твой выбор каждый раз.**

## §5 Phase 4 — «Прошивка» + этапы + варианты plain language

**Source:** Phase 3 above + RUSLAN NOTES (O-176 «прошивка») + multi-variant programs (Task B Phase 6 7 variants).

### §5.A Что такое «прошивка» (на пальцах)

«Прошивка» = базовые установки которые человек ставит себе **сам** перед изучением конкретных tools / методов:

1. **Системное мышление** — видеть систему / связи / контекст
2. **Ответственность** — за свой intellect / свою жизнь / свои методы выбора
3. **Инженерный подход** — discipline / testability / iteration
4. **Question-first** — спрашивать «зачем» прежде «как»
5. **Honesty** — audit себя без иллюзий

Без этой «прошивки» learning не работает — человек просто collect'ит факты / функции без applying.

### §5.B Варианты программ (per time investment)

7 вариантов:

| Вариант | Время | Кому подходит |
|---|---|---|
| **2-часовой обзор** | 2 часа | Stranger → Stage 1 — посмотрел видео + прочитал 1-pager. Если зацепило → дальше. |
| **1-дневный intensive** | 8 часов | Stage 2-3 — workshop format. За день получает baseline + первое практическое applying. |
| **1-недельная программа** | 30-40 часов | Stage 3-4 — async + 2 live sessions. Full Method V2 + practice exercises. |
| **1-месячная программа** | 80-120 часов | Stage 4-5 — гибрид онлайн + live + cohort. Method V2 + first hypothesis test + R12 framework. |
| **3-месячная резиденция** | 200-300 часов | Stage 5-6 — analogue МИМ-style residency. Master-track adapt. |
| **1-годовая mastery track** | 800-1200 часов | Stage 6-7 — multi-residency + projects + Charter. |
| **Multi-year continuous** | ongoing | Stage 7+ advocate — co-creation + cohort + trajectory mastery/reformer. |
