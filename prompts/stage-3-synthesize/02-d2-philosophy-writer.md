### Section 3: Engineering Faith — Core Fundamental Principle (1.5 pages)

Самый foundational принцип Jetix. **Engineering faith = belief backed by plan + tools.**

- **Formula:** faith = confidence × concrete_plan × toolset_availability
- Противоположно wishful thinking (no plan, no tools)
- Противоположно FUD / cynicism (no plan because "impossible")
- Противоположно pure hype (plan faked, tools overblown)
- **Falsifiable:** если один из трёх компонентов ослабляет — вера становится ненадёжной

Quote anchor: wiki atom-0030, atom-0031, atom-0032, atom-0033.

Этот принцип applicable в каждом decision Jetix — спрашиваем "это engineering faith или wishful thinking?"

### Section 5: Cascading Leverage (1.5 pages)

**Ruslan's central structural metaphor** — life → projects → knowledge → thinking → technologies, each amplifying the next.

```
Life (жизнь)
  → управляется через Projects (проекты)
    → усиливаются Knowledge (знаниями)
      → усиливаются Thinking (мышлением)
        → ускоряются Technologies (технологиями)
```

Каждый уровень multiplies previous. Это НЕ linear productivity, это compounding.

**5 рычагов масштабирования дохода** (wiki atom-0083): трафик / контент / автоматизация / команда / продукт. Каждый требует системы за ним (wiki atom-0084).
