## Multi-pass writing approach

### Pass 1 — Skeleton (60-90 minutes)

1. Read all mandatory inputs (skim, не deep)
2. Build section skeleton — headers + 1-line description каждой section
3. Insert anchor quotes (2-4 per section from top 65 quotes)
4. Rough prose — 1-2 paragraphs per section

**Use subagents** (Task tool, parallel):
- Subagent A: "Extract top 10 quotes per dimension (base / company / community / philosophy) from ATOM-REGISTRY + top 65. Return JSONL."
- Subagent B: "Filter ATOM-REGISTRY for atoms type=concept, dimension=base. Return markdown list."
- Subagent C: "Filter ATOM-REGISTRY for atoms type=value OR principle, dimension=philosophy. Return markdown list."

Это ускоряет Pass 1 с 90 до 60 min.

### Pass 2 — Flesh (90-120 minutes)

1. Expand each section с narrative flow
2. Insert per-archetype angles (Section 8) detailed
3. Add nuances + edge cases
4. Cross-check каждую секцию against 18 locked decisions
5. Preserve Ruslan voice в quotes (мат в цитатах сохраняется verbatim)

**Use subagents**:
- Subagent D: "Draft Per-archetype angles Section 8 — 6 angles of 5-8 sentences each."
- Subagent E: "Draft Positioning vs Competitors Section 10 (7 competitor comparisons)."
- Subagent F: "Draft 200-Year Vision Section 14 using all voice-memos + wiki quotes mentioning civilizational framing."

### Pass 3 — Coherence (30-60 minutes)

1. Full re-read own document top-to-bottom
2. Read sibling drafts (JETIX-PHILOSOPHY.md + JETIX-PLAN.md) if they finished Pass 2 already (check by `ls decisions/JETIX-*`)
3. Flag any cross-document contradictions в stdout for Ruslan review
4. Voice consistency check — Russian paragraphs, English section titles / structural elements OK
5. Final polish

**If D2/D3 not yet finished Pass 2** — coherence check только own document. Note в stdout: "Coherence с D2/D3 incomplete, will flag if они finish позже."

## Воice preservation rules

**Direct quotes verbatim:**
- Русский Ruslan voice — копировать точно, включая мат
- НЕ "улучшать" формулировки
- НЕ переводить русский на английский
- Attribution: `[src:audio_XXX]` или `[src:wiki/sources/file.md]` или similar

**Narrative prose:**
- Русский primary для substantive content
- English для technical/structural elements (headers, labels, diagrams)
- Tone: прямой, без воды, ориентированный на действие (matches CLAUDE.md persona)
- НЕ academic prose — operator prose

**Frame quotes:**
- Context: указывай когда Ruslan это сказал если relevant (e.g., "после pivot'а 17 апреля" / "в вечернем planning memo")
- Integration: quotes supportят narrative, не replace it
