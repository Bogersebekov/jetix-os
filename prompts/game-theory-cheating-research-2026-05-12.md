### §3 PHASE 1 — Game Theory Foundations (deep, 1500+ слов)

Classical foundations:
- **Prisoner's Dilemma** (Tucker 1950) — setup, payoffs, Nash equilibrium, why defection dominant single-round
- **Iterated PD** (Axelrod 1980-84) — Axelrod tournament, why cooperation emerges
- **Tit-for-tat strategy** (Rapoport) — simplest winner mechanics
- **Forgiving TFT, Generous TFT, Pavlov (Win-Stay-Lose-Shift)** — refinements
- **Folk theorem** — any cooperation possible in infinite games
- **Cooperation collapse conditions** — high discount rate, noise, single rounds

Other key paradigms:
- Stag Hunt (coordination game) — Rousseau original
- Battle of Sexes (coordination with conflict)
- Chicken / Hawk-Dove — escalation games
- Public Goods Game — free-rider problem
- Ultimatum Game — fairness intuitions
- Trust Game — Berg-Dickhaut-McCabe
- Tragedy of the Commons (Hardin 1968 + Ostrom counter 1990) — when commons work
- Free-rider problem — when it kills cooperation

Advanced theory:
- **Evolutionary game theory** (Maynard Smith ESS)
- **Behavioral game theory** (Camerer) — humans ≠ rational
- **Mechanism design** (Hurwicz / Maskin / Myerson Nobel 2007) — designing rules
- **Matching markets** (Roth Nobel 2012) — stable allocations
- **Auction theory** (Vickrey-Clarke-Groves)
- **Cooperative game theory** (Shapley value, core)
- **Signaling games** (Spence) — costly signals
- **Bayesian games** (Harsanyi)
- **Repeated games + reputation**
- **Indirect reciprocity** (Nowak) — image scoring
