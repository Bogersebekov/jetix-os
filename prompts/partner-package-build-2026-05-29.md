## §2 Overview layering (Ruslan voice 29.05 — КАК подавать P-1)

Overview подаётся **слоями** (это структура P-1, и общая логика всего пакета):

1. **Базовый слой — методология работы с информацией.** Всё = информация + методы её обработки. Интеллект (человек / система) **пытается развиваться** и использует **лучшие методы для собственного развития**. Это фундамент — всё остальное растёт отсюда.
2. **Накладывается 3P** (продукты / процессы / проекты) — это и есть информация + методы работы с ней **в применении**. Жизнь человека = его главный продукт; управление = люди + системы + процессы + технологии; «ты управляющий своей жизнью, но и другие хотят ею управлять».
3. **Дальше суть Jetix:** мега-мастерская + сеть кооперативных кланов + возможности.
4. **Что на текущий момент нужно:** фундамент (tech / fin / legal) + помощь + партнёры.

> ⚠️ 3P НЕ сам по себе — он **накладывается** на базовую методологию «информация + методы обработки → саморазвитие интеллекта». Это критично: сначала базовый слой, потом 3P как его применение.
