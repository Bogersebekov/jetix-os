# AI-Powered Лидген: стек, возможности, план ресёрча

## Зачем
Лидген — сервис с коротким циклом продажи и подписочной моделью. Jetix находит и квалифицирует B2B-клиентов для партнёров с помощью AI-стека, который заменяет команду из 3-5 SDR. Входная точка в экосистему Jetix: клиент приходит за лидами → остаётся на AI-консалтинг.

## AI-стек для лидгена (~€320-460/мес)

### 1. Мозг (координация)
- **Notion** (~€10/мес) — CRM + база + pipeline
- **Claude Pro** ($20/мес) — анализ, скоринг, MCP-интеграция
- **Gemini Advanced** ($20/мес) — большой контекст, синтез данных

### 2. Sourcing (поиск)
- **LinkedIn Sales Navigator** (~€80/мес) — поиск decision-makers
- **Apollo.io** (~€50-100/мес) — база 250M+ контактов, email enrichment

### 3. Research
- **Perplexity Pro** ($20/мес) — ресёрч компаний с источниками за 1-2 мин
- **Claude Deep Research** (входит в Pro) — аналитический отчёт за 10-15 мин

### 4. Квалификация
- **Claude/GPT API** (~€30-50/мес) — автоматический скоринг 500 лидов/час по 10+ критериям

### 5. Автоматизация
- **n8n / Make** (€20-40/мес) — связка всех инструментов в единый поток

### 6. Outreach
- **Instantly / Lemlist** (~€50-80/мес) — 200-500 персонализированных писем/день
- **Expandi** (~€100/мес) — LinkedIn-автоматизация

### 7. Контент
- **Claude + Canva Pro** (~€15/мес) — 10-20 постов/день с адаптацией под нишу

## Сравнение: AI-стек vs команда SDR

| Параметр | Ручная работа (3 SDR) | AI-стек Jetix (1 оператор) |
|----------|----------------------|----------------------------|
| Стоимость/мес | €9,000-15,000 | €320-460 + 1 оператор |
| Касаний/день | 80-150 | 300-500+ |
| Квалификация 1 лида | 15-30 минут | 1-3 минуты |
| Персонализация | Шаблонная, устаёт к обеду | Глубокая, стабильная, 24/7 |
| Масштабирование | Линейное | Экспоненциальное |
| Ошибки | 30-40% лидов теряется | <5% |
| Время до результата | 2-3 месяца | 1-2 недели |

## Уникальные возможности AI-стека
1. **Гиперперсонализация в масштабе** — каждое письмо учитывает 10+ параметров
2. **Intent-based targeting** — компании, которые ПРЯМО СЕЙЧАС ищут решение
3. **Multi-channel orchestration** — email + LinkedIn + content синхронизированы
4. **Predictive scoring** — AI видит паттерны конверсии
5. **Real-time enrichment** — данные обновляются автоматически

## Модели ценообразования

| Модель | Структура | Для кого | Плюсы | Минусы |
|--------|-----------|---------|-------|--------|
| A. Ретейнер | €2,000-5,000/мес | Стабильный поток | Предсказуемый доход | Долгий цикл продажи |
| B. Pay per result | €150-500/лид, €300-2,000/встреча | Скептики | Легко продать | Непредсказуемый доход |
| C. Setup + Performance | €3-5K setup + €1-2K/мес + 5-10% | Амбициозные | Максимальный upside | Сложно отслеживать |

## Целевые ниши (ранжированы)
1. **IT-аутсорсинг / dev-агентства** — чек €50-500K, ретейнер €3-5K/мес
2. **SaaS seed/Series A** — нет SDR, бюджет ограничен
3. **Консалтинговые фирмы** — высокие чеки, длинный цикл
4. **Рекрутинговые агентства** — всегда нужны клиенты
5. **Маркетинговые агентства** — низкие чеки, высокая конкуренция
6. **Mittelstand** — upsell на AI-консалтинг (длинный цикл)

## План ресёрча (7 этапов)

### Этап 0: Экономика лидов 🟢 ТЕКУЩИЙ

- IT-аутсорсинг SQL CPL = $1,680–$2,800

## Инсайты

- **Конкуренция не в инструментах, а в глубине:** все используют одни инструменты. Массовый аутрич умирает. Выигрывает тот, кто глубоко понимает нишу и приходит с готовым результатом. 95% агентств продают обещание, Jetix продаёт факт
- **«Троянский лид» = Rainmaker для бедных:** модель Rainmaker ($15K+/интро) требует 10+ лет нетворка. Jetix компенсирует это AI-стеком. Скрипт: «Мы вам уже одного нашли и хотим отдать бесплатно. А если у вас есть час — мы можем найти ещё больше за приятный прайс»
